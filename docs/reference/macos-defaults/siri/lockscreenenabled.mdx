# Siri on Lock Screen

Allow Siri to be used when your Mac is locked.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `LockscreenEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether Siri can be activated and used while your Mac is on the lock screen. When enabled, anyone with physical access to your Mac can invoke Siri without unlocking, potentially accessing information or performing actions. When disabled, the Mac must be unlocked before Siri can be used, providing better security in shared or public environments.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enabled** (default) — Siri available on lock screen |
| `false` | **Disabled** — Must unlock Mac to use Siri |

## Our Configuration

**Set to**: `false` — Siri disabled on lock screen for security.

### Why Disable Siri on Lock Screen?

1. **Security** — Prevents unauthorized information access
2. **Privacy** — Calendar, messages, contacts protected
3. **Physical access** — Passersby can't query your Mac
4. **Work environment** — Protects sensitive data
5. **Best practice** — Defense in depth security

### Why Keep Enabled?

- Hands-free convenience at home
- Quick timers and basic queries
- Single-user environment
- Physical security already strong
- Prefer convenience over security

## Code Examples

```bash
# Read current value
defaults read com.apple.Siri LockscreenEnabled

# Disable Siri on lock screen (more secure)
defaults write com.apple.Siri LockscreenEnabled -bool false

# Enable Siri on lock screen (default)
defaults write com.apple.Siri LockscreenEnabled -bool true

# Delete (reset to default)
defaults delete com.apple.Siri LockscreenEnabled
```

Changes may require logout to take effect.

## What Siri CAN'T Do When Locked

Even with lock screen Siri enabled:
- Cannot unlock the Mac
- Cannot view photos or files
- Cannot open apps
- Cannot change system settings
- Cannot access encrypted content

But can still access calendar, messages, contacts, and perform Siri searches.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Siri still works when locked | Verify setting and logout/login |
| "Hey Siri" not responding | Check if Siri enabled overall |
| Setting not persisting | Check MDM/profile restrictions |
| Siri icon still on lock screen | Restart Mac after changing |
| Can't find setting in UI | Look under Siri & Spotlight |

## Compatibility

- **macOS**: 10.12+ (Sierra, when Siri was added to Mac)
- **Note**: May require logout for changes to take effect
- **Note**: Independent of "Hey Siri" setting
- **Note**: Managed Macs may have this set by policy
