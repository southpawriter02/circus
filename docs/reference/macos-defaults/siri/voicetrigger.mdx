# Siri Voice Trigger

Enable "Hey Siri" voice activation.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `VoiceTriggerUserEnabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether you can activate Siri by saying "Hey Siri" (or just "Siri" on newer Macs). When enabled, your Mac constantly listens for the wake phrase and activates Siri hands-free. This allows voice-only interaction without pressing any buttons or using keyboard shortcuts.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Disabled** (default) — Must use keyboard/click to activate Siri |
| `true` | **Enabled** — "Hey Siri" voice activation active |

## Our Configuration

**Set to**: `false` — Voice trigger disabled for privacy.

### Why Enable Hey Siri?

1. **Hands-free** — Activate Siri without touching Mac
2. **Accessibility** — Essential for motor impairments
3. **Convenience** — Quick queries while working
4. **HomeKit** — Control smart home with voice
5. **Multitasking** — Set timers, reminders while cooking

### Why Keep Disabled?

- Privacy concerns about always-on microphone
- Don't use Siri regularly
- Prefer keyboard/click activation
- Shared or public environment
- Battery/power conservation

## Code Examples

```bash
# Read current value
defaults read com.apple.Siri VoiceTriggerUserEnabled

# Enable Hey Siri
defaults write com.apple.Siri VoiceTriggerUserEnabled -bool true

# Disable Hey Siri (default)
defaults write com.apple.Siri VoiceTriggerUserEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.Siri VoiceTriggerUserEnabled
```

Changes may require logout to take effect.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Hey Siri not responding | Check if enabled, verify Mac support |
| Activates accidentally | Retrain Siri voice recognition |
| Only works when lid open | Normal for some models |
| Works intermittently | Check microphone access, restart |
| Doesn't recognize voice | Disable/re-enable to retrain |

## Compatibility

- **macOS**: 10.12+ (Sierra)
- **Hardware**: T2 chip or Apple Silicon for always-on
- **Note**: May require logout for changes to take effect
- **Note**: Older Macs need to be awake for Hey Siri
- **Note**: Requires Siri to be enabled first
