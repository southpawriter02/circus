# Siri Keyboard Shortcut

Configure the keyboard shortcut used to activate Siri.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `HotkeyTag` |
| **Type** | Integer |
| **Default Value** | `0` |
| **Our Setting** | `0` |

## Description

Controls which keyboard shortcut activates Siri on macOS. By default, holding the microphone key (on supported keyboards) or pressing a configured key combination invokes Siri. This setting allows you to customize the keyboard trigger or disable it entirely. The setting works alongside voice activation ("Hey Siri") if that feature is enabled.

## Values

| Value | Shortcut | Description |
|-------|----------|-------------|
| `0` | **Hold Dictation Key** | Hold Fn key or mic key (default) |
| `1` | **Hold Command Space** | Hold âŒ˜ + Space |
| `2` | **Hold Option Space** | Hold âŒ¥ + Space |
| `3` | **Press Fn Space** | Press Fn + Space |
| `4` | **Disabled** | No keyboard shortcut |

## Our Configuration

**Set to**: `0` â€” Hold Dictation Key (default).

### Why Use Default (0)?

1. **No conflicts** â€” Dedicated key doesn't override other shortcuts
2. **Intuitive** â€” Microphone key naturally associated with voice
3. **Apple default** â€” Consistent with expected macOS behavior
4. **Works everywhere** â€” Supported on all Mac keyboards

### Alternative Options

**Hold Command Space (1)**:
- Familiar to Spotlight users
- Quick access near primary typing position
- Risk of accidental activation

**Hold Option Space (2)**:
- Fewer software conflicts
- Less discoverable
- Good for power users

**Press Fn Space (3)**:
- Immediate activation (no holding)
- Clearer separation from other shortcuts
- Requires two hands

**Disabled (4)**:
- For users who never use Siri
- Use "Hey Siri" voice only
- Maximum keyboard availability

```bash
defaults write com.apple.Siri HotkeyTag -int 1

# Delete (reset to default)
defaults delete com.apple.Siri HotkeyTag
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Shortcut not working | Verify Siri is enabled in System Settings |
| Conflicts with Spotlight | Use Option Space or Fn Space instead |
| Accidental activation | Switch to Fn Space (requires press, not hold) |
| Shortcut changed unexpectedly | Check third-party keyboard software |
| Setting not persisting | Log out and log back in |
| No microphone key | Use alternative shortcut option |

## Compatibility

- **macOS**: 10.12+ (Sierra) when Siri was introduced
- **Note**: Microphone key available on MacBook Pro 2016+ and Magic Keyboard
- **Note**: Older keyboards use Fn key as dictation key
- **Note**: May require logout for changes to apply
- **Note**: Works alongside "Hey Siri" voice activation
