# Siri Status Menu Visible

Show Siri icon in the menu bar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `StatusMenuVisible` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether the Siri icon appears in the macOS menu bar. When visible, clicking the Siri icon activates Siri immediately. Hiding the icon declutters the menu bar while still allowing Siri access via keyboard shortcuts, the Dock, or "Hey Siri" voice activation.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Visible** (default) — Siri icon shown in menu bar |
| `false` | **Hidden** — Siri icon removed from menu bar |

## Our Configuration

**Set to**: `false` — Siri icon hidden from menu bar.

### Why Hide Siri Icon?

1. **Cleaner menu bar** — More space for other items
2. **Minimalism** — Less visual clutter
3. **Use keyboard** — Prefer shortcut activation
4. **Use voice** — Hey Siri doesn't need icon
5. **Rarely use** — Don't need quick access

### Why Keep Visible?

- Quick one-click Siri access
- Visual reminder Siri is available
- Prefer mouse over keyboard
- Don't use Hey Siri
- Training new users

## Code Examples

```bash
# Read current value
defaults read com.apple.Siri StatusMenuVisible

# Hide Siri from menu bar
defaults write com.apple.Siri StatusMenuVisible -bool false

# Show Siri in menu bar (default)
defaults write com.apple.Siri StatusMenuVisible -bool true

# Delete (reset to default)
defaults delete com.apple.Siri StatusMenuVisible

# Restart menu bar (if needed)
killall SystemUIServer
```

Changes take effect immediately or after a brief delay.

## Related Settings

```bash
# Menu bar visibility (this setting)
defaults write com.apple.Siri StatusMenuVisible -bool false

# Master Siri toggle
defaults write com.apple.assistant.support "Assistant Enabled" -bool true

# Keyboard shortcut for Siri
defaults write com.apple.Siri HotkeyTag -int 0

# Hey Siri voice activation
defaults write com.apple.Siri VoiceTriggerUserEnabled -bool false
```

If you hide the menu bar icon, add Siri to Dock for quick access:

```bash

# Drag Siri from Applications folder to Dock
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Icon won't hide | Try `killall SystemUIServer` |
| Icon returns after restart | Verify setting persisted |
| Can't find Siri anymore | Use keyboard shortcut or Dock |
| Setting doesn't stick | Check for MDM restrictions |
| Icon in wrong position | Hold ⌘ and drag to reorder |

## Compatibility

- **macOS**: 10.12+ (Sierra)
- **Note**: Takes effect immediately
- **Note**: Siri functionality unaffected by hiding icon
- **Note**: Works independently of other Siri settings
