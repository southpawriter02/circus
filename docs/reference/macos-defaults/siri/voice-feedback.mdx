# Siri Voice Feedback

Control whether Siri speaks responses aloud.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `VoiceFeedback` |
| **Type** | String |
| **Default Value** | `On` |
| **Our Setting** | `Off` |

## Description

Controls whether Siri speaks its responses aloud or displays them silently as text. When voice feedback is off, Siri shows responses on screen without audio, making it suitable for quiet environments or when you prefer to read responses.

## Values

| Value | Behavior |
|-------|----------|
| `On` | **Voice feedback enabled** (default) — Siri speaks responses |
| `Off` | **Silent mode** — Responses displayed as text only |

## Our Configuration

**Set to**: `Off` — Silent Siri responses (text only).

### Why Disable Voice Feedback?

1. **Quiet environments** — Library, shared office, bedroom
2. **Privacy** — Others don't hear your queries
3. **Discretion** — Responses aren't announced
4. **Focus** — No audio interruptions
5. **Preference** — Some prefer reading to listening

### Why Keep Voice Feedback?

- Hands-free operation
- Accessibility needs
- Can't look at screen
- Natural conversation feel
- HomeKit voice control

## Code Examples

```bash
# Read current value
defaults read com.apple.Siri VoiceFeedback

# Disable voice feedback (silent)
defaults write com.apple.Siri VoiceFeedback -string "Off"

# Enable voice feedback (default)
defaults write com.apple.Siri VoiceFeedback -string "On"

# Delete (reset to default)
defaults delete com.apple.Siri VoiceFeedback
```

Changes take effect immediately for new Siri interactions.

## "Hey Siri" Behavior

When using "Hey Siri" (voice activation):
- Siri still listens with microphone
- Response follows Voice Feedback setting
- Voice Feedback Off = Siri responds silently


## Troubleshooting

| Issue | Solution |
|-------|----------|
| Siri still speaking | Restart Siri, verify setting |
| Can't hear Siri | Check Voice Feedback is On |
| Setting not persisting | Verify with `defaults read` |
| Volume issues | Check system volume separately |
| Siri voice different | Separate "Siri Voice" setting |

## Accessibility Considerations

Voice feedback can be important for:
- Vision impairments
- Motor difficulties (can't look at screen)
- Learning disabilities
- Multitasking needs

If using Siri for accessibility, keep Voice Feedback enabled.

## Compatibility

- **macOS**: 10.12+ (Sierra)
- **Note**: Takes effect immediately
- **Note**: Separate from system volume settings
- **Note**: "Hey Siri" activation still uses microphone
