# Siri Suggestions

Control Siri Suggestions in apps and Spotlight.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Siri` |
| **Key** | `SiriSuggestionsEnabled` / `ShowSiriSuggestionsInSpotlight` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether Siri provides proactive suggestions based on your app usage, calendar events, messages, and browsing history. These suggestions appear in Spotlight, Safari, and throughout apps. Disabling prevents Siri from analyzing your activity patterns.

## Values

### SiriSuggestionsEnabled

| Value | Behavior |
|-------|----------|
| `true` | **Suggestions enabled** (default) — Siri learns from usage |
| `false` | **Suggestions disabled** — No activity-based suggestions |

### ShowSiriSuggestionsInSpotlight

| Value | Behavior |
|-------|----------|
| `true` | **Show in Spotlight** (default) — Suggestions appear in search |
| `false` | **Hide from Spotlight** — Only file/app results shown |

## Security & Privacy Implications

### Data Siri Analyzes

| Data Type | Used For |
|-----------|----------|
| Calendar events | Time-based suggestions |
| Emails | Contact and follow-up suggestions |
| Messages | Response suggestions |
| Safari history | Website suggestions |
| App usage | App suggestions |
| Location | Location-based suggestions |
| Contacts | People suggestions |

### Privacy Trade-offs

```
+-------------------------------------------------------------+
|  Privacy vs Convenience                                      |
+-------------------------------------------------------------+
|                                                              |
|  ENABLED:                        DISABLED:                   |
|  +------------------------+      +------------------------+  |
|  | + Smart suggestions    |      | + Maximum privacy      |  |
|  | + Time-saving         |      | + No data analysis     |  |
|  | + Context-aware       |      | + Predictable search   |  |
|  | - Activity tracked    |      | - Manual searching     |  |
|  | - Data processed      |      | - No smart features    |  |
|  | - Patterns learned    |      | - More keystrokes      |  |
|  +------------------------+      +------------------------+  |
|                                                              |
+-------------------------------------------------------------+
```

## Our Configuration

**Set to**: `false` — Disable Siri Suggestions for privacy.

### Why Disable Siri Suggestions?

1. **Privacy** — No behavioral tracking
2. **Data minimization** — Less data processed
3. **Predictability** — Search results are consistent
4. **Security** — Sensitive info not surfaced
5. **Corporate** — Compliance requirements

### Why Keep Enabled?

- Time savings from smart suggestions
- Convenience of proactive reminders
- Better calendar integration
- Useful for personal devices
- Context-aware assistance

## Code Examples

```bash
# Read current values
defaults read com.apple.Siri SiriSuggestionsEnabled
defaults read com.apple.Siri ShowSiriSuggestionsInSpotlight

# Disable Siri Suggestions in apps
defaults write com.apple.Siri SiriSuggestionsEnabled -bool false

# Disable Siri Suggestions in Spotlight
defaults write com.apple.Siri ShowSiriSuggestionsInSpotlight -bool false

# Enable both (default)
defaults write com.apple.Siri SiriSuggestionsEnabled -bool true
defaults write com.apple.Siri ShowSiriSuggestionsInSpotlight -bool true

# Delete (reset to defaults)
defaults delete com.apple.Siri SiriSuggestionsEnabled
defaults delete com.apple.Siri ShowSiriSuggestionsInSpotlight
```

Changes take effect immediately.

## Related Settings

```bash
# Siri Suggestions in apps (this setting)
defaults write com.apple.Siri SiriSuggestionsEnabled -bool false

# Siri Suggestions in Spotlight (this setting)
defaults write com.apple.Siri ShowSiriSuggestionsInSpotlight -bool false

# Master Siri toggle
defaults write com.apple.assistant.support "Assistant Enabled" -bool false

# Type to Siri
defaults write com.apple.Siri TypeToSiriEnabled -bool true

# Voice feedback
defaults write com.apple.Siri VoiceFeedback -string "Off"
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Suggestions still appearing | Check per-app settings in System Settings |
| Spotlight not updating | Rebuild index: `sudo mdutil -E /` |
| Setting not persisting | Verify Siri is enabled first |
| Safari still suggests | Also check Safari > Settings > Search |

## Compatibility

- **Note**: Per-app settings in System Settings
