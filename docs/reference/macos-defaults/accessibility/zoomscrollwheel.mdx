# Accessibility Zoom with Scroll Wheel

Use scroll gesture with modifier keys to zoom the screen.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewScrollWheelToggle` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Enables screen zooming using scroll gestures combined with a modifier key (typically Control). When enabled, holding the modifier key while scrolling magnifies the entire screen, making content larger and easier to see. This accessibility feature is useful for users with visual impairments, designers reviewing fine details, or anyone who occasionally needs to magnify screen content.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enable scroll zoom** â€” Hold modifier + scroll to zoom |
| `false` | **Disable scroll zoom** (default) â€” Standard scrolling only |

## Our Configuration

**Set to**: `true` â€” Enable scroll wheel zoom for quick magnification.

### Why Enable Scroll Zoom?

1. **Accessibility** â€” Magnify content when needed
2. **Design work** â€” Inspect fine details
3. **Presentations** â€” Zoom for audience visibility
4. **Eye strain** â€” Temporarily enlarge small text
5. **Pixel inspection** â€” Check image/icon details

### Why Disable Scroll Zoom?

- Accidental zoom triggers
- Conflict with other shortcuts
- Prefer keyboard-only zoom
- System performance concerns
- Don't need magnification

## Modifier Key Options

The modifier key can be configured:

| Modifier | Setting Value |
|----------|---------------|
| Control (âŒƒ) | `262144` (default) |
| Option (âŒ¥) | `524288` |
| Command (âŒ˜) | `1048576` |
| Control + Option | `786432` |

```bash
# Set modifier to Option key
defaults write com.apple.universalaccess closeViewScrollWheelModifiersInt -int 524288
```

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewScrollWheelToggle

# Enable scroll wheel zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Disable scroll wheel zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool false

# Set modifier key (Control = 262144)
defaults write com.apple.universalaccess closeViewScrollWheelModifiersInt -int 262144

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewScrollWheelToggle
```

## Related Zoom Settings

```bash
# Enable scroll wheel zoom (this setting)
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Set zoom modifier key (Control)
defaults write com.apple.universalaccess closeViewScrollWheelModifiersInt -int 262144

# Zoom style: 0=fullscreen, 1=split-screen, 2=picture-in-picture
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Smooth zooming
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Maximum zoom level (1.0 to 40.0)
defaults write com.apple.universalaccess closeViewMaxZoom -float 5.0

# Minimum zoom level
defaults write com.apple.universalaccess closeViewMinZoom -float 1.0

# Enable keyboard shortcuts for zoom
defaults write com.apple.universalaccess closeViewHotKeysEnabled -bool true
```

## Zoom Styles

| Style | Description | Best For |
|-------|-------------|----------|
| Full screen | Entire screen magnified | General use |
| Split screen | Zoomed view in portion | Reference + detail |
| Picture-in-picture | Small magnified window | Following cursor |

```bash
# Full screen zoom
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Split screen zoom
defaults write com.apple.universalaccess closeViewZoomMode -int 1

# Picture-in-picture zoom
defaults write com.apple.universalaccess closeViewZoomMode -int 2
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Zoom not working | Verify setting enabled in System Settings |
| Wrong modifier key | Check closeViewScrollWheelModifiersInt value |
| Zoom too sensitive | Adjust scroll speed in trackpad/mouse settings |
| Can't zoom out fully | Check closeViewMinZoom is set to 1.0 |
| Performance issues | Disable smooth images or reduce max zoom |

## Performance Considerations

| Setting | Impact |
|---------|--------|
| Smooth images ON | Better quality, more GPU usage |
| Smooth images OFF | Pixelated at high zoom, faster |
| High max zoom | More memory usage |
| Full screen mode | Higher GPU usage than PIP |
