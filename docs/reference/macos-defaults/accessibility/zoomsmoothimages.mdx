# Zoom Smooth Images

Smooth images when zoomed to reduce pixelation.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewSmoothImages` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether images and screen content are smoothed (anti-aliased) when using the accessibility zoom feature. When enabled, zoomed content appears smoother and less pixelated. When disabled, individual pixels are visible, which can be useful for precise pixel-level work or when you prefer sharper edges.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Smooth images** (default) — Anti-aliased, less pixelated zoom |
| `false` | **Pixelated zoom** — Sharp pixels visible, no smoothing |

## Our Configuration

**Set to**: `true` — Smooth images for comfortable viewing.

### Why Enable Smoothing?

1. **Readability** — Text remains clear at high zoom
2. **Comfort** — Less eye strain from jagged edges
3. **Natural** — Content looks more like the original
4. **General use** — Best for accessibility needs
5. **Quality** — Professional appearance when zoomed

### Why Disable Smoothing?

- Pixel-level design work
- Debugging graphics rendering
- Pixel art appreciation
- Technical precision needs
- Prefer seeing actual pixels

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewSmoothImages

# Enable smooth images (default)
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Disable smooth images (pixelated)
defaults write com.apple.universalaccess closeViewSmoothImages -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewSmoothImages
```

Changes may require logout to take effect.

## Related Settings

```bash
# Smooth images (this setting)
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Zoom mode (full screen, split, PiP)
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Enable scroll wheel zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Enable keyboard zoom shortcuts
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true

# Flash screen on zoom toggle
defaults write com.apple.universalaccess closeViewFlashScreen -bool false
```

## Keyboard Shortcut for Toggle

You can toggle image smoothing while zoomed using:

| Shortcut | Action |
|----------|--------|
| Option + Cmd + / | Toggle smooth images on/off |
| Option + Cmd + 8 | Toggle zoom on/off |
| Option + Cmd + = | Zoom in |
| Option + Cmd + - | Zoom out |

## Performance Considerations

| Setting | Performance Impact |
|---------|-------------------|
| Smooth ON | Slightly higher GPU usage |
| Smooth OFF | Lower GPU usage, may be faster |

For most modern Macs, the performance difference is negligible.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Images still pixelated | Verify setting is enabled, re-enable zoom |
| Setting not persisting | May require logout to apply |
| Smoothing looks blurry | This is expected behavior of anti-aliasing |
| No visual difference | Effect is most visible at high zoom levels |
| Text hard to read | Enable smoothing for text readability |
