# Accessibility Maximum Zoom Level

Set the maximum magnification limit for screen zoom.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewFarPoint` |
| **Type** | Integer |
| **Default Value** | `1` |
| **Our Setting** | `1` |

## Description

Controls the maximum zoom level (magnification) when using macOS's accessibility zoom feature. This sets the upper limit of how far you can zoom in on the screen. The value represents the minimum zoom state — confusingly named `FarPoint` because it's the "farthest out" (least zoomed) you can be when zoom is active. The complementary setting `closeViewNearPoint` sets the maximum zoom-in level.

## Values

| Value | Meaning |
|-------|---------|
| `1` | **1x** (default) — Normal size (100%), no zoom |
| `2` | **2x** — Double size (200% magnification) |
| `5` | **5x** — Five times normal |
| `10` | **10x** — Ten times normal |
| `20` | **20x** — Maximum useful zoom |
| `40` | **40x** — Extreme zoom (rarely needed) |

## Our Configuration

**Set to**: `1` — Normal size as the minimum zoom level.

### Why 1x Minimum?

1. **Standard** — Full screen visible at minimum zoom
2. **Expected** — Matches normal screen behavior
3. **Flexible** — Can zoom out completely
4. **Usable** — See entire screen when needed
5. **Default** — Apple's recommended setting

### Why Higher Minimum?

- Always need some magnification
- Prevent accidental zoom-out
- Vision requires constant zoom
- Lock zoom to specific range

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewFarPoint

# Set minimum zoom to 1x (normal, recommended)
defaults write com.apple.universalaccess closeViewFarPoint -int 1

# Set minimum zoom to 2x (always some magnification)
defaults write com.apple.universalaccess closeViewFarPoint -int 2

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewFarPoint
```

## Related Settings

```bash
# Minimum zoom level (this setting)
defaults write com.apple.universalaccess closeViewFarPoint -int 1

# Maximum zoom level
defaults write com.apple.universalaccess closeViewNearPoint -int 20

# Enable keyboard shortcuts for zoom
defaults write com.apple.universalaccess closeViewHotKeysEnabled -bool true

# Enable scroll gesture zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Zoom style (0=full screen, 1=split, 2=pip)
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Smooth zoomed images
defaults write com.apple.universalaccess closeViewSmoothImages -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Zoom not working | Enable zoom shortcuts in Accessibility settings |
| Can't zoom out fully | Check closeViewFarPoint setting |
| Zoom too pixelated | Enable "Smooth images" setting |
| Zoom disorienting | Try split screen or PiP zoom style |
| Settings not applying | May require logout or restart |

## Compatibility

- **macOS**: 10.7+ (Lion)
- **Note**: Part of accessibility features
- **Note**: Works with all zoom styles
