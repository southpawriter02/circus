# Slow Keys

Adjust keyboard to require keys to be held before registering.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `slowKey` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Slow Keys is an accessibility feature that adds a delay between pressing a key and the keystroke being registered. This helps users who may accidentally press keys or have difficulty with precise key presses. The key must be held down for a specified duration before it registers, preventing accidental keystrokes from tremors or imprecise movements.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Disabled** (default) — Keys register immediately |
| `true` | **Enabled** — Keys require hold duration to register |

## Our Configuration

**Set to**: `false` — Disabled for standard keyboard response.

### Why Enable Slow Keys?

1. **Tremor assistance** — Prevents accidental presses
2. **Motor control** — Helps with imprecise movements
3. **Deliberate typing** — Each key intentional
4. **Accessibility** — Accommodates various needs
5. **Error reduction** — Fewer typos from mishits

### Why Keep Disabled?

- Standard typing ability
- Speed typing needed
- No motor control issues
- Delay frustrating
- Gaming or quick input needs

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess slowKey

# Enable Slow Keys
defaults write com.apple.universalaccess slowKey -bool true

# Disable Slow Keys (default)
defaults write com.apple.universalaccess slowKey -bool false

# Set delay duration (in seconds, e.g., 0.5)
defaults write com.apple.universalaccess slowKeyDelay -float 0.5

# Delete (reset to defaults)
defaults delete com.apple.universalaccess slowKey
```

Changes may require logout to take effect.

## Related Settings

```bash
# Enable Slow Keys (this setting)
defaults write com.apple.universalaccess slowKey -bool true

# Slow Keys delay
defaults write com.apple.universalaccess slowKeyDelay -float 0.5

# Click sounds for Slow Keys
defaults write com.apple.universalaccess slowKeyBeepOn -bool true

# Sticky Keys
defaults write com.apple.universalaccess stickyKey -bool false

# Sticky Keys show window
defaults write com.apple.universalaccess stickyKeyShowWindow -bool true
```

## Acceptance Delay Values

| Delay | Description | Best For |
|-------|-------------|----------|
| 0.1-0.2s | Short | Mild assistance |
| 0.3-0.5s | Medium | Moderate tremor |
| 0.6-0.8s | Long | Significant assistance |
| 1.0s+ | Very long | Maximum protection |

## Click Key Sounds

When enabled:
- **Key press start**: Sound when delay begins
- **Key accepted**: Confirmation sound when registered
- Helps provide audio feedback for timing

## Slow Keys vs Other Features

| Feature | Function |
|---------|----------|
| Slow Keys | Requires key hold duration |
| Sticky Keys | Holds modifier keys |
| Key Repeat | Controls held key repetition |
| Touch Typing | Virtual keyboard assistance |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Keys not registering | Hold longer, adjust delay |
| Delay too long | Reduce acceptance delay |
| No feedback | Enable click sounds |
| Modifier keys affected | Slow Keys affects all keys |
| Gaming issues | Disable for gaming |

## Keyboard Shortcuts

Even with Slow Keys enabled, you can quickly toggle:

| Shortcut | Action |
|----------|--------|
| Press Shift 5 times | Toggle Slow Keys (if enabled) |
| System Settings | Full control over settings |

Note: The 5x Shift shortcut must be enabled separately.

## Compatibility

- **macOS**: 10.6+ (Snow Leopard)
- **Note**: Affects all keyboard input
- **Note**: May require logout for changes
- **Note**: Works with external keyboards
