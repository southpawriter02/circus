# Zoom Keyboard Shortcuts

Enable keyboard shortcuts for screen zoom.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewHotkeysEnabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Enables keyboard shortcuts for controlling screen zoom accessibility features. When enabled, you can toggle zoom on/off, zoom in, and zoom out using Option+Command key combinations. This provides quick keyboard-based access to magnification without needing to use the mouse or trackpad gestures.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Disabled** (default) — Zoom keyboard shortcuts inactive |
| `true` | **Enabled** — Zoom keyboard shortcuts active |

## Our Configuration

**Set to**: `true` — Keyboard shortcuts enabled for quick zoom access.

### Why Enable Keyboard Shortcuts?

1. **Quick access** — Toggle zoom instantly
2. **No mouse needed** — Keyboard-only operation
3. **Precise control** — Consistent zoom steps
4. **Accessibility** — Essential for some users
5. **Presentations** — Quick magnification during demos

### Why Keep Disabled?

- Prefer scroll gesture zoom only
- Avoid accidental activation
- Key combinations conflict with other apps
- Don't use zoom feature
- Minimal accessibility needs

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewHotkeysEnabled

# Enable zoom keyboard shortcuts
defaults write com.apple.universalaccess closeViewHotkeysEnabled -bool true

# Disable zoom keyboard shortcuts (default)
defaults write com.apple.universalaccess closeViewHotkeysEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewHotkeysEnabled
```

Changes may require logout to take effect.

## Complete Keyboard Shortcut Reference

| Shortcut | Action | When Available |
|----------|--------|----------------|
| ⌥⌘8 | Toggle zoom on/off | When hotkeys enabled |
| ⌥⌘= | Zoom in | When zoom is active |
| ⌥⌘+ | Zoom in (alternative) | When zoom is active |
| ⌥⌘- | Zoom out | When zoom is active |
| ⌥⌘/ | Toggle smooth images | When zoom is active |
| ⌥⌘\ | Toggle image inversion | Advanced setting |

## Related Settings

```bash
# Keyboard shortcuts (this setting)
defaults write com.apple.universalaccess closeViewHotkeysEnabled -bool true

# Scroll gesture zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Scroll wheel modifier key (Control = 262144)
defaults write com.apple.universalaccess closeViewScrollWheelModifiersInt -int 262144

# Zoom style (0=full, 1=split, 2=PiP)
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Smooth images
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Zoom follows keyboard focus
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Shortcuts not working | Verify setting is enabled, logout/login |
| Conflicts with other apps | Some apps may override these shortcuts |
| Zoom not activating | Check zoom feature is enabled overall |
| Wrong zoom level on toggle | Zoom remembers last level used |
| Can't zoom out further | May have reached minimum zoom level |

## Compatibility

- **macOS**: 10.6+ (Snow Leopard)
