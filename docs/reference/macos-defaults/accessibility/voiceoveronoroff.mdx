# VoiceOver Enable/Disable

Master toggle for the VoiceOver screen reader accessibility feature.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `voiceOverOnOrOff` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether VoiceOver, Apple's built-in screen reader, is enabled. VoiceOver provides spoken descriptions of what's on screen, enabling blind and visually impaired users to use a Mac. When enabled, VoiceOver reads aloud text, describes images, announces UI elements, and provides audio cues for navigation. It completely changes how the Mac is operated, using keyboard commands and gestures instead of visual interaction.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **VoiceOver off** (default) â€” Standard visual interface |
| `true` | **VoiceOver on** â€” Screen reader active with spoken feedback |

## Our Configuration

**Set to**: `false` â€” VoiceOver disabled by default.

### Why Keep Disabled?

1. **Standard use** â€” Most users navigate visually
2. **Learning curve** â€” VoiceOver requires training to use effectively
3. **Different workflow** â€” Completely changes Mac interaction
4. **Toggle available** â€” Command + F5 enables when needed

### Why Enable?

- User is blind or has severe visual impairment
- Testing VoiceOver compatibility of applications
- Learning to use screen readers
- Accessibility compliance testing

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess voiceOverOnOrOff

# Enable VoiceOver
defaults write com.apple.universalaccess voiceOverOnOrOff -bool true

# Disable VoiceOver
defaults write com.apple.universalaccess voiceOverOnOrOff -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess voiceOverOnOrOff

# Note: The keyboard shortcut Command+F5 is the standard way to toggle
# VoiceOver. Using defaults may require logout/restart.
```

## Related Settings

```bash
# VoiceOver on/off (this setting)
defaults write com.apple.universalaccess voiceOverOnOrOff -bool false

# VoiceOver cursor size
defaults write com.apple.universalaccess voiceOverCursorSize -int 1

# Enable audio descriptions for video
defaults write com.apple.universalaccess mediaAccessibilityAudioDescriptions -bool true

# Speak selected text (alternative to VoiceOver)
defaults write com.apple.universalaccess speakSelectedText -bool true

# Speak announcements
defaults write com.apple.universalaccess speakAnnouncements -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't turn off VoiceOver | Press Command+F5; or hold power button for options |
| VoiceOver not speaking | Check volume; check System Settings audio output |
| Wrong voice | Open VoiceOver Utility > Speech to change voice |
| Commands not working | Ensure using Control+Option (VO) modifier |
| Accidentally enabled | Press Command+F5 to toggle off |

## Compatibility

- **macOS**: 10.4+ (Tiger introduced VoiceOver)
- **Note**: VoiceOver has significantly improved in each macOS version
- **Note**: Works with most macOS applications
- **Note**: Supports Braille displays via Bluetooth or USB
- **Note**: Required for Apple's accessibility compliance
