# Accessibility Minimum Zoom Level (Maximum Magnification)

Set the maximum magnification when zoomed in.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewNearPoint` |
| **Type** | Integer |
| **Default Value** | `20` |
| **Our Setting** | `20` |

## Description

Controls the maximum zoom magnification level when using macOS's accessibility zoom feature. This sets how far you can zoom into the screen — the "closest" view to the content. Despite the confusing naming convention (`NearPoint` = maximum zoom, `FarPoint` = minimum zoom), this setting determines how much magnification is available when you're fully zoomed in.

## Values

| Value | Magnification | Use Case |
|-------|---------------|----------|
| `2` | **2x** | Slight magnification for small text |
| `5` | **5x** | General readability assistance |
| `10` | **10x** | Low vision support |
| `15` | **15x** | Detailed inspection |
| `20` | **20x** (default) | Full accessibility range |
| `40` | **40x** | Maximum - pixel-level viewing |

## Our Configuration

**Set to**: `20` — Standard accessibility range for maximum flexibility.

### Why 20x Maximum?

1. **Accessibility** — Supports various vision needs
2. **Flexibility** — Wide range for different situations
3. **Standard** — Apple's default recommendation
4. **Practical** — Beyond 20x rarely useful
5. **Balance** — Enough zoom without extremes

### Why Higher (40x)?

- Pixel-level design work
- Extreme vision impairment
- Quality assurance testing
- Digital art creation

### Why Lower (5-10x)?

- Prevent over-zooming
- Simpler zoom control
- Casual use only
- Faster zoom transitions

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewNearPoint

# Set maximum zoom to 20x (recommended)
defaults write com.apple.universalaccess closeViewNearPoint -int 20

# Set maximum zoom to 10x (moderate)
defaults write com.apple.universalaccess closeViewNearPoint -int 10

# Set maximum zoom to 40x (extreme)
defaults write com.apple.universalaccess closeViewNearPoint -int 40

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewNearPoint
```

## Related Settings

```bash
# Maximum zoom level (this setting)
defaults write com.apple.universalaccess closeViewNearPoint -int 20

# Minimum zoom level
defaults write com.apple.universalaccess closeViewFarPoint -int 1

# Enable keyboard shortcuts for zoom
defaults write com.apple.universalaccess closeViewHotKeysEnabled -bool true

# Enable scroll gesture zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Zoom follows keyboard focus
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true

# Smooth zoomed images
defaults write com.apple.universalaccess closeViewSmoothImages -bool true
```

## Complete Zoom Configuration

```bash
#!/bin/bash
# Complete zoom setup for accessibility

# Enable zoom features
defaults write com.apple.universalaccess closeViewHotKeysEnabled -bool true
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Set zoom range
defaults write com.apple.universalaccess closeViewFarPoint -int 1
defaults write com.apple.universalaccess closeViewNearPoint -int 20

# Use Control for scroll gesture
defaults write com.apple.universalaccess closeViewScrollWheelModifiersInt -int 262144

# Full screen zoom style
defaults write com.apple.universalaccess closeViewZoomMode -int 0

# Smooth images when zoomed
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Follow keyboard focus
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true

echo "Zoom configured. May require logout to take effect."
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't zoom in enough | Increase closeViewNearPoint value |
| Zoom too extreme | Decrease closeViewNearPoint value |
| Zoomed view is pixelated | Enable "Smooth images" |
| Zoom not following cursor | Check zoom style and follow focus settings |
| Settings not saving | Try logging out and back in |
