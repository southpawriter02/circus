# Accessibility Sticky Keys Show Window

Display a visual indicator window showing the state of modifier keys when Sticky Keys is enabled.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `stickyKeyShowWindow` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether a visual indicator window appears on screen when Sticky Keys is enabled. The window shows the current state of modifier keys (Shift, Control, Option, Command), indicating whether each is off, pressed once (sticky), or locked (pressed twice). This visual feedback is essential for users who need Sticky Keys but may not remember which modifiers are currently active.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show window** (default) â€” Visual indicator visible |
| `false` | **Hide window** â€” No visual indicator; rely on audio feedback |

## Our Configuration

**Set to**: `true` â€” Show visual feedback when using Sticky Keys.

### Why Show Window?

1. **Visual feedback** â€” Know which modifiers are active
2. **Prevent mistakes** â€” See state before pressing keys
3. **Essential for accessibility** â€” Core feature of Sticky Keys
4. **Learning aid** â€” Helps learn keyboard shortcuts

### Why Hide Window?

- Prefer audio-only feedback
- Screen space is very limited
- Find window distracting
- Already memorized modifier states

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess stickyKeyShowWindow

# Show Sticky Keys status window (default when Sticky Keys enabled)
defaults write com.apple.universalaccess stickyKeyShowWindow -bool true

# Hide Sticky Keys status window
defaults write com.apple.universalaccess stickyKeyShowWindow -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess stickyKeyShowWindow

# Also need Sticky Keys enabled to see the effect
defaults write com.apple.universalaccess stickyKey -bool true
```

## Related Settings

```bash
# Show status window (this setting)
defaults write com.apple.universalaccess stickyKeyShowWindow -bool true

# Enable Sticky Keys
defaults write com.apple.universalaccess stickyKey -bool true

# Play sound when modifier pressed
defaults write com.apple.universalaccess stickyKeyBeepOnModifier -bool true

# Slow Keys (separate feature)
defaults write com.apple.universalaccess slowKey -bool false

# Slow Keys delay
defaults write com.apple.universalaccess slowKeyDelay -float 0.25
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Window not appearing | Ensure stickyKey is enabled |
| Window in wrong position | Drag to reposition; position is remembered |
| Window too distracting | Disable this setting; use audio feedback |
| Can't see modifier states | Ensure window not hidden behind other apps |
| Setting not taking effect | Toggle Sticky Keys off/on |

## Compatibility

- **macOS**: 10.0+ (all macOS versions)
- **Note**: Only shows when Sticky Keys is enabled
- **Note**: Window position persists across sessions
- **Note**: Works with all applications
- **Note**: Essential accessibility feature
