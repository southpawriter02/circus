# Stereo as Mono

Combine left and right audio channels into mono output.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `stereoAsMono` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Plays stereo audio as mono, combining the left and right channels so the same audio is heard in both ears. This accessibility feature ensures users with hearing loss in one ear don't miss audio content that's panned to one side. It's also useful when using a single earbud or speaker.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Stereo** (default) — Left and right channels separate |
| `true` | **Mono** — Left and right channels combined |

## Our Configuration

**Set to**: `false` — Stereo audio for full spatial experience.

### Why Enable Mono Audio?

1. **Hearing loss** — Single-sided deafness or hearing impairment
2. **Single earbud** — Using one AirPod/earbud
3. **Shared listening** — Two people sharing one pair of earbuds
4. **Accessibility** — Ensure no audio content is missed
5. **Mono speakers** — Single speaker setup

### Why Keep Stereo?

- Normal hearing in both ears
- Full spatial audio experience
- Music production/mixing
- Immersive gaming
- Movie surround effects

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess stereoAsMono

# Enable mono audio (combine L+R)
defaults write com.apple.universalaccess stereoAsMono -bool true

# Disable mono audio (default stereo)
defaults write com.apple.universalaccess stereoAsMono -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess stereoAsMono
```

Changes take effect immediately for most audio.

## Audio Balance vs Mono

| Feature | Audio Balance | Stereo as Mono |
|---------|---------------|----------------|
| Effect | Shifts audio L/R | Combines L+R |
| Use case | Partial hearing loss | Complete loss one side |
| Maintains stereo | Yes | No |
| Spatial audio | Preserved (shifted) | Lost |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Audio still stereo | Check setting is enabled, restart audio app |
| Mono sounds weird | Normal - spatial effects lost |
| Only affects some apps | Some apps handle audio independently |
| Balance not working | Balance has no effect in mono mode |
| Setting not saving | Check for MDM restrictions |

## Compatibility

- **macOS**: 10.14+ (Mojave)
- **Note**: Takes effect immediately for most audio
- **Note**: Some professional audio apps may override
- **Note**: Works with all audio output (speakers, headphones)
- **Note**: Affects system-wide audio
