# Zoom Follows Keyboard Focus

Zoom window automatically follows keyboard focus.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `closeViewZoomFollowsFocus` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the zoom view automatically moves to follow the keyboard focus as you navigate through the interface. When enabled, the zoomed area shifts to keep the currently focused element (text field, button, menu item) visible on screen. This is essential for users who rely on both zoom and keyboard navigation, as it eliminates the need to manually reposition the zoom view while tabbing through interfaces.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Disabled** (default) — Zoom stays in place, manual repositioning needed |
| `true` | **Enabled** — Zoom follows keyboard focus automatically |

## Our Configuration

**Set to**: `true` — Zoom follows keyboard focus for seamless navigation.

### Why Enable Zoom Follows Focus?

1. **Accessibility** — Essential for keyboard-only users who need zoom
2. **Efficiency** — No interruption to reposition zoom view
3. **Workflow** — Seamless form and dialog navigation
4. **Reduced strain** — Less mouse movement needed
5. **VoiceOver companion** — Works well with screen reader

### Why Keep Disabled?

- Prefer manual control of zoom position
- Find automatic movement disorienting
- Use mouse/trackpad for zoom navigation
- Only use zoom occasionally for static content
- Prefer predictable zoom behavior

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess closeViewZoomFollowsFocus

# Enable zoom follows focus
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true

# Disable zoom follows focus (default)
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess closeViewZoomFollowsFocus
```

Changes may require logout to take effect.

## Related Settings

```bash
# Zoom follows focus (this setting)
defaults write com.apple.universalaccess closeViewZoomFollowsFocus -bool true

# Enable keyboard shortcuts for zoom
defaults write com.apple.universalaccess closeViewHotKeysEnabled -bool true

# Enable scroll gesture zoom
defaults write com.apple.universalaccess closeViewScrollWheelToggle -bool true

# Smooth images when zoomed
defaults write com.apple.universalaccess closeViewSmoothImages -bool true

# Zoom style (0=full, 1=split, 2=PiP)
defaults write com.apple.universalaccess closeViewZoomMode -int 0
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Zoom not following focus | Verify setting, logout/login |
| Follows slowly or lags | Reduce zoom level for smoother panning |
| Doesn't follow in some apps | Some apps have non-standard focus |
| Too jumpy/disorienting | Try split screen or PiP zoom mode |
| Focus goes off screen | Enable both scroll gesture and follow focus |

## Compatibility

- **macOS**: 10.6+ (Snow Leopard)
- **Note**: May require logout for changes to take effect
- **Note**: Works best with Full Screen zoom mode
- **Note**: Some older apps may not properly report focus
