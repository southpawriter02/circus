# Hearing Aid Mode

Enable support for Made for iPhone (MFi) hearing aids.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.universalaccess` |
| **Key** | `HearingAidMode` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Enables Made for iPhone (MFi) hearing aid compatibility mode on your Mac. When enabled and compatible hearing aids are paired via Bluetooth, this allows the Mac to route audio directly to hearing aids, control hearing aid settings from the Mac, and optimize audio output for hearing aid users. This provides a seamless audio experience for users with hearing impairments.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Disabled** (default) — Standard audio output |
| `true` | **Enabled** — MFi hearing aid support active |

## Our Configuration

**Set to**: `false` — Disabled unless hearing aids are used.

### Why Enable Hearing Aid Mode?

1. **Direct streaming** — Audio goes straight to hearing aids
2. **Better quality** — Optimized audio processing
3. **Control** — Adjust hearing aids from Mac
4. **Convenience** — Unified device management
5. **Accessibility** — Essential for hearing aid users

### Why Keep Disabled?

- Don't use hearing aids
- Using standard headphones/speakers
- Hearing aids not MFi compatible
- Prefer other audio output
- No Bluetooth hearing devices

## Code Examples

```bash
# Read current value
defaults read com.apple.universalaccess HearingAidMode

# Enable hearing aid mode
defaults write com.apple.universalaccess HearingAidMode -bool true

# Disable hearing aid mode (default)
defaults write com.apple.universalaccess HearingAidMode -bool false

# Delete (reset to default)
defaults delete com.apple.universalaccess HearingAidMode
```

Changes may require logout to take effect.

## Related Settings

```bash
# Hearing aid mode (this setting)
defaults write com.apple.universalaccess HearingAidMode -bool true

# Stereo as mono (for single-sided hearing)
defaults write com.apple.universalaccess stereoAsMono -bool false

# Audio balance (L/R adjustment)
defaults write com.apple.universalaccess audioBalance -float 0.0

# Flash screen for alerts (visual alerts)
defaults write com.apple.universalaccess flashScreen -bool false
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Hearing aids not detected | Put in pairing mode, check Bluetooth |
| Audio not routing | Verify selected as output device |
| Connection drops | Check battery level, distance to Mac |
| Only one aid connects | Re-pair both aids |
| Poor audio quality | Check for interference, update firmware |

## Compatibility

- **macOS**: 10.14+ (Mojave)
- **Hardware**: Mac with Bluetooth 4.0+
- **Hearing Aids**: MFi certified devices
- **Note**: May require logout for changes to take effect
- **Note**: Works best with recent hearing aid models
