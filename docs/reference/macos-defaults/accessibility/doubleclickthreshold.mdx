# Double-Click Speed Threshold

Adjust the time interval recognized as a double-click.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `com.apple.mouse.doubleClickThreshold` |
| **Type** | Float (seconds) |
| **Default Value** | `0.5` |
| **Our Setting** | `0.5` |

## Description

Controls the maximum time interval between two clicks for them to be recognized as a double-click. A shorter threshold requires faster clicking, while a longer threshold allows more time between clicks. This is particularly important for accessibility, allowing users with motor control difficulties to perform double-clicks at a comfortable pace.

## Values

| Value | Speed | Description |
|-------|-------|-------------|
| `0.2` | Fast | Requires quick double-clicks |
| `0.5` | Medium | Default, balanced |
| `0.75` | Slow | More time between clicks |
| `1.0` | Very Slow | Extended time for accessibility |
| `1.5` | Slowest | Maximum time between clicks |

## Our Configuration

**Set to**: `0.5` â€” Standard double-click speed (default).

### Why Use Faster Threshold?

1. **Efficiency** â€” Quicker file access
2. **Precision** â€” Reduces accidental double-clicks
3. **Power users** â€” Accustomed to fast clicking
4. **Reduces latency** â€” Actions execute sooner

### Why Use Slower Threshold?

1. **Accessibility** â€” Motor control difficulties
2. **Arthritis** â€” Painful rapid clicking
3. **Tremors** â€” Parkinson's or essential tremor
4. **Age-related** â€” Slower motor response
5. **Injury recovery** â€” Temporary limitation

## Code Examples

```bash
# Read current value
defaults read NSGlobalDomain com.apple.mouse.doubleClickThreshold

# Set to default (0.5 seconds)
defaults write NSGlobalDomain com.apple.mouse.doubleClickThreshold -float 0.5

# Set to slow (1.0 seconds)
defaults write NSGlobalDomain com.apple.mouse.doubleClickThreshold -float 1.0

# Set to fast (0.25 seconds)
defaults write NSGlobalDomain com.apple.mouse.doubleClickThreshold -float 0.25

# Delete (reset to default)
defaults delete NSGlobalDomain com.apple.mouse.doubleClickThreshold
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Double-click doesn't register | Increase threshold (make slower) |
| Accidental double-clicks | Decrease threshold (make faster) |
| Clicks feel delayed | Threshold may be too long |
| Setting not saving | Log out and back in |
| Different in some apps | Some apps may override system settings |

## Third-Party Mouse Software

Some mice have their own software (Logitech Options, Razer Synapse) that may:
- Override system double-click settings
- Provide their own double-click speed control
- Need separate configuration

Check manufacturer software if system settings don't seem to apply.
