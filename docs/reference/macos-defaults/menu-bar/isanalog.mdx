# Menu Bar Analog Clock

Control whether the menu bar displays an analog or digital clock.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.clock` |
| **Key** | `IsAnalog` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the menu bar clock displays as a traditional analog clock face with hands or as a digital display showing numbers. When set to analog, a small circular clock face appears with hour and minute hands. The analog option provides a classic aesthetic but takes up more horizontal space and doesn't show seconds.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Digital clock** (default) — Shows time as numbers |
| `true` | **Analog clock** — Shows clock face with hands |

## Our Configuration

**Set to**: `false` — Digital clock for precise time display.

### Why Use Digital Clock (false)?

1. **Precision** — Shows exact time with seconds
2. **Date display** — Can show day and date
3. **Quick reading** — Numbers read faster than hands
4. **Seconds option** — Important for timing
5. **Default behavior** — Expected by most users

### Why Use Analog Clock (true)?

- **Aesthetic** — Classic watch appearance
- **Minimalism** — No text, just clock face
- **Distraction-free** — No changing numbers
- **Preference** — Some users prefer analog
- **Unique** — Stands out from default

## Code Examples

```bash
# Read current value
defaults read com.apple.menuextra.clock IsAnalog

# Use digital clock (default, recommended)
defaults write com.apple.menuextra.clock IsAnalog -bool false

# Use analog clock
defaults write com.apple.menuextra.clock IsAnalog -bool true

# Delete (reset to default)
defaults delete com.apple.menuextra.clock IsAnalog

# Restart menu bar to apply changes
killall SystemUIServer
```

## Related Settings

```bash
# Clock style (this setting)
defaults write com.apple.menuextra.clock IsAnalog -bool false

# Show seconds (digital only)
defaults write com.apple.menuextra.clock ShowSeconds -bool true

# Show day of week (digital only)
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool true

# Show date (digital only)
defaults write com.apple.menuextra.clock ShowDate -int 1

# Flash separators (digital only)
defaults write com.apple.menuextra.clock FlashDateSeparators -bool false

# Custom format (digital only)
defaults write com.apple.menuextra.clock DateFormat -string "EEE d MMM  HH:mm:ss"

# Restart menu bar
killall SystemUIServer
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Setting not applying | Run `killall SystemUIServer` |
| Date/seconds disappeared | They're digital-only; switch back |
| Analog too small to read | No size adjustment; use digital |
| Clock options grayed out | Switch to digital to access them |
| Want analog with date | Not supported; use digital |
| Clock disappeared | Check Control Center settings |

## Compatibility

- **macOS**: 10.0+ (feature available since early macOS)
- **Note**: Requires `killall SystemUIServer` to apply
- **Note**: Analog mode disables date/seconds options
- **Note**: Click analog clock to see full date/time
- **Note**: Works in both light and dark mode
