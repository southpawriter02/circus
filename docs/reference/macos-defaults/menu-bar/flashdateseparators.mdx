# Menu Bar Flash Date Separators

Control whether the clock's colon separators flash each second.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.clock` |
| **Key** | `FlashDateSeparators` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the colon (:) separators in the menu bar clock flash on and off each second. When enabled, the colons blink at one-second intervals, similar to how many digital clocks and displays work. This provides a visual indication that the clock is running and updating.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Static separators** (default) — Colons always visible |
| `true` | **Flashing separators** — Colons blink each second |

## Our Configuration

**Set to**: `false` — Static separators for clean appearance.

### Why Use Static Separators (false)?

1. **Distraction-free** — No constant visual movement
2. **Clean interface** — Matches modern macOS aesthetic
3. **Professional** — Stable, consistent display
4. **Screen recording** — No flashing artifacts
5. **Default behavior** — Expected appearance

### Why Use Flashing Separators (true)?

- **Visual feedback** — Confirms clock is running
- **Nostalgia** — Classic digital clock look
- **Attention** — Draws eye to time periodically
- **Preference** — Some users like the effect

## Code Examples

```bash
# Read current value
defaults read com.apple.menuextra.clock FlashDateSeparators

# Disable flashing (static separators - default)
defaults write com.apple.menuextra.clock FlashDateSeparators -bool false

# Enable flashing separators
defaults write com.apple.menuextra.clock FlashDateSeparators -bool true

# Delete (reset to default)
defaults delete com.apple.menuextra.clock FlashDateSeparators

# Restart menu bar to apply changes
killall SystemUIServer
```

## Related Settings

```bash
# Flash time separators (this setting)
defaults write com.apple.menuextra.clock FlashDateSeparators -bool false

# Show seconds in clock
defaults write com.apple.menuextra.clock ShowSeconds -bool true

# Use analog clock (disables this setting's effect)
defaults write com.apple.menuextra.clock IsAnalog -bool false

# Show day of week
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool true

# Custom date format
defaults write com.apple.menuextra.clock DateFormat -string "EEE d MMM  HH:mm:ss"

# Restart menu bar
killall SystemUIServer
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Setting not applying | Run `killall SystemUIServer` |
| Flashing not visible | Ensure IsAnalog is false (digital mode) |
| Want flash without seconds | Works with ShowSeconds false too |
| Flashing too distracting | Set to false (disable) |
| Can't find in System Settings | Look under Control Center > Clock Options |
| Setting reverts | May need to restart after reboot |

## Compatibility

- **macOS**: 10.8+ (Mountain Lion and later)
- **Note**: Only applies to digital clock mode
- **Note**: Requires `killall SystemUIServer` to apply
- **Note**: Has no effect when IsAnalog is true
- **Note**: Works with or without seconds displayed
