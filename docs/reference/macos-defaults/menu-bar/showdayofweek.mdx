# Menu Bar Clock Show Day of Week

Display the day of the week in the menu bar clock.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.clock` |
| **Key** | `ShowDayOfWeek` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the menu bar clock displays the abbreviated day of the week (Mon, Tue, Wed, etc.) alongside the time. This provides quick context about what day it is without opening Calendar or checking a calendar widget. Particularly useful for people who work irregular schedules or need constant day awareness.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show day** (default) â€” Day abbreviation visible |
| `false` | **Hide day** â€” Time only displayed |

## Our Configuration

**Set to**: `true` â€” Show day of week for quick date context.

### Why Show Day of Week?

1. **Context** â€” Know what day it is instantly
2. **Planning** â€” Quick schedule awareness
3. **Work-life** â€” Track weekday vs weekend
4. **Meetings** â€” Verify day for appointments
5. **Default behavior** â€” Standard macOS experience

### Why Hide Day of Week?

- Maximum menu bar space
- Already know what day it is
- Prefer minimal clock display
- Use other calendar tools
- Aesthetic preference

## Menu Bar Space Impact

```
Without day: [ 2:45 PM ]      (8 chars)
With day:    [ Wed 2:45 PM ]  (12 chars)
                     +4 chars wider

With date:   [ Wed Jan 8  2:45 PM ]  (19 chars)
             Much wider, consider 24-hour format
```

## Code Examples

```bash
# Read current value
defaults read com.apple.menuextra.clock ShowDayOfWeek

# Show day of week (default)
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool true

# Hide day of week
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool false

# Delete (reset to default)
defaults delete com.apple.menuextra.clock ShowDayOfWeek

# Restart menu bar to apply
killall SystemUIServer
```

## ShowDate Values

The date display is controlled separately:

| Value | Display |
|-------|---------|
| 0 | Never show date |
| 1 | Always show date |
| 2 | Show date when space allows |

```bash
# Always show date with day
defaults write com.apple.menuextra.clock ShowDate -int 1
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool true
```

## Custom Date Format

For advanced control, use DateFormat:

```bash
# Full day name + time
defaults write com.apple.menuextra.clock DateFormat -string "EEEE h:mm a"
# Result: Wednesday 2:45 PM

# Day + month + day + time
defaults write com.apple.menuextra.clock DateFormat -string "EEE MMM d  h:mm a"
# Result: Wed Jan 8  2:45 PM
```

Format codes:
| Code | Displays | Example |
|------|----------|---------|
| `E` | Day, shortest | W |
| `EE` | Day, short | We |
| `EEE` | Day, abbreviated | Wed |
| `EEEE` | Day, full | Wednesday |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Day not appearing | Run `killall SystemUIServer` |
| Wrong day abbreviation | Check system language settings |
| Clock too wide | Disable date or use 24-hour |
| Setting not persisting | Check for MDM profiles |

## Analog Clock Note

When using analog clock (`IsAnalog -bool true`), the day of week setting has no effect since analog clocks only show time visually.

```bash
# Check if analog clock is enabled
defaults read com.apple.menuextra.clock IsAnalog

# Switch to digital to see day
defaults write com.apple.menuextra.clock IsAnalog -bool false
killall SystemUIServer
```
