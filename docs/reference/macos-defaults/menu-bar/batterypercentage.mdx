# Menu Bar Battery Percentage

Show battery percentage in the menu bar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.battery` |
| **Key** | `ShowPercent` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays the battery percentage next to the battery icon in the menu bar. Provides precise battery level information without needing to click the battery icon.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show percentage** â€” e.g., "78%" |
| `false` | **Icon only** (default) â€” Visual estimate |

## Our Configuration

**Set to**: `true` â€” Always show percentage.

### Why Show Percentage?

1. **Precision** â€” Know exact level
2. **Planning** â€” Gauge time remaining
3. **Awareness** â€” No guessing from icon
4. **Charging** â€” Track progress exactly

## Code Examples

```bash
# Read current value
defaults read com.apple.menuextra.battery ShowPercent

# Show percentage (recommended)
defaults write com.apple.menuextra.battery ShowPercent -bool true

# Hide percentage (icon only)
defaults write com.apple.menuextra.battery ShowPercent -bool false

# Apply changes
killall SystemUIServer
```
