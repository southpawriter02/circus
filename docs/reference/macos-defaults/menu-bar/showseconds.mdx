# Menu Bar Clock Show Seconds

Display seconds in the menu bar clock.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.clock` |
| **Key** | `ShowSeconds` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the menu bar clock displays seconds alongside hours and minutes. When enabled, the clock updates every second, showing the precise time. This is useful for time-sensitive work, synchronization tasks, or simply for users who prefer a more detailed time display.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show seconds** â€” Clock displays HH:MM:SS |
| `false` | **Hide seconds** (default) â€” Clock displays HH:MM |

## Our Configuration

**Set to**: `true` â€” Show seconds for precise time awareness.

### Why Show Seconds?

1. **Precision** â€” Know exact time instantly
2. **Synchronization** â€” Useful for coordinated tasks
3. **Meetings** â€” Join at exact start time
4. **Development** â€” Timing debugging sessions
5. **Personal preference** â€” Complete time display

### Why Hide Seconds?

- Cleaner menu bar
- Less visual movement/distraction
- Saves menu bar space
- Battery (minimal impact)
- Default macOS behavior

## Resource Impact

| Setting | CPU Impact | Visual Updates |
|---------|------------|----------------|
| Seconds OFF | Minimal | Once per minute |
| Seconds ON | Negligible | Once per second |

The impact is minimal on modern Macs.

## Code Examples

```bash
# Read current value
defaults read com.apple.menuextra.clock ShowSeconds

# Show seconds in clock
defaults write com.apple.menuextra.clock ShowSeconds -bool true

# Hide seconds (default)
defaults write com.apple.menuextra.clock ShowSeconds -bool false

# Delete (reset to default)
defaults delete com.apple.menuextra.clock ShowSeconds

# Restart menu bar to apply
killall SystemUIServer
```

## Related Clock Settings

```bash
# Show seconds
defaults write com.apple.menuextra.clock ShowSeconds -bool true

# Show day of week
defaults write com.apple.menuextra.clock ShowDayOfWeek -bool true

# Show AM/PM indicator
defaults write com.apple.menuextra.clock ShowAMPM -bool true

# Flash time separators (colons blink)
defaults write com.apple.menuextra.clock FlashDateSeparators -bool false

# Use analog clock
defaults write com.apple.menuextra.clock IsAnalog -bool false

# Custom date format
defaults write com.apple.menuextra.clock DateFormat -string "EEE d MMM  HH:mm:ss"
```

## Custom Date Format

For advanced customization, use `DateFormat`:

| Code | Displays | Example |
|------|----------|---------|
| `HH` | 24-hour | 14 |
| `hh` | 12-hour | 02 |
| `mm` | Minutes | 45 |
| `ss` | Seconds | 32 |
| `a` | AM/PM | PM |
| `EEE` | Day abbrev | Wed |
| `EEEE` | Day full | Wednesday |
| `d` | Day of month | 8 |
| `MMM` | Month abbrev | Jan |
| `MMMM` | Month full | January |

Example:
```bash
# "Wed 8 Jan  14:45:32"
defaults write com.apple.menuextra.clock DateFormat -string "EEE d MMM  HH:mm:ss"
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Clock not updating | Run `killall SystemUIServer` |
| Seconds not showing | Verify setting, restart SystemUIServer |
| Format looks wrong | Check other clock settings |
| Too wide for menu bar | Reduce other menu bar items |

## Menu Bar Space Consideration

```
Without seconds: [ 2:45 PM ]     (8 chars)
With seconds:    [ 2:45:32 PM ]  (11 chars)
                          +3 chars wider

With 24-hour + seconds: [ 14:45:32 ]  (8 chars)
More compact than 12-hour with seconds
```
