# ShowPercent

Show battery percentage in the menu bar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.menuextra.battery` |
| **Key** | `ShowPercent` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays the battery charge percentage next to the battery icon in the menu bar, providing precise information about remaining charge.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show percentage** — "100%" displayed next to icon |
| `false` | **Icon only** (default) — Battery icon with no number |

## Our Configuration

**Set to**: `true` — Always know exact battery level at a glance.

## Code Examples

```bash
# Show battery percentage
defaults write com.apple.menuextra.battery ShowPercent -bool true

# Hide battery percentage
defaults write com.apple.menuextra.battery ShowPercent -bool false

# May need to restart SystemUIServer
killall SystemUIServer
```

## UI Location

**System Settings** → **Control Center** → **Battery** → **Show Percentage**

Also accessible by clicking the battery icon in the menu bar.

## Note

This setting only applies to MacBooks with batteries. On desktop Macs, the battery menu extra isn't shown.
