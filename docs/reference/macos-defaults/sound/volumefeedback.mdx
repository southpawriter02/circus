# Volume Change Feedback Sound

Control the "pop" sound when adjusting volume.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `com.apple.sound.beep.feedback` |
| **Type** | Integer |
| **Default Value** | `1` |
| **Our Setting** | `0` |

## Description

Controls whether a "pop" sound plays each time you adjust the volume using the keyboard keys or menu bar slider. While helpful for knowing volume levels, it can be annoying in quiet environments.

## Values

| Value | Behavior |
|-------|----------|
| `1` | **Enabled** (default) â€” Pop sound on volume change |
| `0` | **Disabled** â€” Silent volume adjustment |

## The Volume Feedback Sound

When enabled:
```
Press Volume Up â†’ "Pop" sound at new volume level
Press Volume Down â†’ "Pop" sound at new volume level
Press Mute â†’ No sound (muted)
```

## Our Configuration

**Set to**: `0` â€” Disable feedback sound.

### Why Disable?

1. **Quiet environments** â€” Libraries, shared offices
2. **Headphones** â€” Loud pop in ears
3. **Night use** â€” Silent adjustments at night
4. **Clean audio** â€” Recording/streaming
5. **Less noise** â€” Minimalist preference

### Why Enable?

- Audio confirmation of level
- Know when volume actually changed
- Accessibility (audio feedback)
- Habit from other devices

## Code Examples

```bash
# Read current value
defaults read NSGlobalDomain com.apple.sound.beep.feedback

# Disable volume feedback sound (recommended)
defaults write NSGlobalDomain com.apple.sound.beep.feedback -int 0

# Enable volume feedback sound (default)
defaults write NSGlobalDomain com.apple.sound.beep.feedback -int 1

# Delete (reset to default)
defaults delete NSGlobalDomain com.apple.sound.beep.feedback
```

## Shift Key Trick

Hold **Shift** while adjusting volume:
- Temporarily inverts this setting
- If sounds are on: silent adjustment
- If sounds are off: plays feedback

Useful for quick one-off adjustments.

## Alternative: Option Key

Hold **Option+Shift** + Volume keys:
- Finer volume increments (1/4 normal)
- Precise control

## Keyboard Volume Keys

| Key | Action |
|-----|--------|
| F10 | Mute toggle |
| F11 | Volume down |
| F12 | Volume up |

With Shift: Opposite feedback behavior.

## Audio During Meetings

For video calls:
- Disable feedback to avoid interrupting
- Adjust volume with visual indicator only
- Prevents "pop" sounds in meetings

## Troubleshooting

| Issue | Solution |
