# Screenshot Show Thumbnail

Control the floating thumbnail after taking screenshots.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.screencapture` |
| **Key** | `show-thumbnail` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether a floating thumbnail appears in the corner of the screen after taking a screenshot. This thumbnail allows quick annotation, sharing, or deletion before the file is saved. Introduced in macOS Mojave, it mirrors the iOS screenshot experience.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show thumbnail** (default) â€” Floats briefly |
| `false` | **No thumbnail** â€” Save immediately |

## Thumbnail Interactions

What you can do with the floating thumbnail:
| Action | Result |
|--------|--------|
| **Wait** | Auto-saves after ~5 seconds |
| **Click** | Opens Markup for annotation |
| **Swipe right** | Dismiss and save immediately |
| **Drag** | Drop into app (Messages, Mail, etc.) |
| **âŒ˜Delete** | Delete without saving |
| **Right-click** | Context menu with options |

## Our Configuration

**Set to**: `true` â€” Show thumbnail.

### Why Show Thumbnail?

1. **Quick annotation** â€” Mark up immediately
2. **Drag to apps** â€” Share without finding file
3. **Review** â€” Verify screenshot captured correctly
4. **Delete** â€” Remove bad screenshots instantly
5. **Modern workflow** â€” Seamless like iOS

### Why Disable Thumbnail?

- Taking many screenshots quickly
- Automation/scripting
- Prefer immediate save
- Find it distracting

## Code Examples

```bash
# Read current value
defaults read com.apple.screencapture show-thumbnail

# Show thumbnail (default)
defaults write com.apple.screencapture show-thumbnail -bool true

# Disable thumbnail (immediate save)
defaults write com.apple.screencapture show-thumbnail -bool false

# Restart to apply
killall SystemUIServer
```

## Thumbnail Duration

Thumbnail shows for approximately 5 seconds, then:
- Auto-saves to configured location
- Disappears from screen

No way to change duration.

## Quick Save/Dismiss

To skip waiting:
- **Swipe right** on thumbnail
- Or **Control+click** â†’ **Save to Desktop**
- Or **Drag** to destination

## Related Settings

```bash
# Screenshot location
defaults write com.apple.screencapture location -string "~/Pictures/Screenshots"

# Screenshot format
defaults write com.apple.screencapture type -string "png"

# Disable shadow
defaults write com.apple.screencapture disable-shadow -bool true

# Include date in filename
defaults write com.apple.screencapture include-date -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Thumbnail not appearing | Check setting is true |
| Thumbnail too fast | Can't adjust duration |
| Blocks screen content | Move to corner during capture |
