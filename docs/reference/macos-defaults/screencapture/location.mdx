# Screenshot Location

Configure where macOS saves screenshots.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.screencapture` |
| **Key** | `location` |
| **Type** | String |
| **Default Value** | `~/Desktop` |
| **Our Setting** | `~/Pictures/Screenshots` |

## Our Configuration

**Set to**: `~/Pictures/Screenshots` â€” Organized in Pictures folder.

### Why a Custom Location?

1. **Clean Desktop** â€” No screenshot clutter
2. **Organization** â€” All screenshots in one place
3. **Easy cleanup** â€” Empty one folder periodically
4. **Findable** â€” Know where to look
5. **Professional** â€” Tidy visible Desktop

### Why Keep Desktop?

- Immediate visibility
- Drag-and-drop convenience
- Screenshots are temporary anyway
- Simple workflow

## Code Examples

```bash
# Read current location
defaults read com.apple.screencapture location

# Set to Screenshots folder (create folder first!)
mkdir -p ~/Pictures/Screenshots
defaults write com.apple.screencapture location -string "~/Pictures/Screenshots"

# Set to Downloads
defaults write com.apple.screencapture location -string "~/Downloads"

# Set to Desktop (default)
defaults write com.apple.screencapture location -string "~/Desktop"

# Set to iCloud Drive folder
defaults write com.apple.screencapture location -string "~/Library/Mobile Documents/com~apple~CloudDocs/Screenshots"

# Restart SystemUIServer to apply
killall SystemUIServer
```

## Save to Clipboard Instead

For temporary screenshots, skip saving:
```
âŒ˜â‡§âŒƒ3 â€” Full screen to clipboard
âŒ˜â‡§âŒƒ4 â€” Selection to clipboard
```

Or set default to clipboard:
```bash

```

## Related Screenshot Settings

```bash
# Disable shadow in window screenshots
defaults write com.apple.screencapture disable-shadow -bool true

# Change file format (png, jpg, gif, pdf)
defaults write com.apple.screencapture type -string "png"

# Include date in filename
defaults write com.apple.screencapture include-date -bool true

# Show thumbnail preview (floating thumbnail)
defaults write com.apple.screencapture show-thumbnail -bool true
```
