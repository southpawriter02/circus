# Mission Control Gesture

Control whether the trackpad swipe-up gesture opens Mission Control.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.dock` |
| **Key** | `showMissionControlGestureEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether swiping up with three or four fingers on the trackpad opens Mission Control. Mission Control provides a bird's-eye view of all open windows, desktop spaces, and full-screen apps, making it easy to navigate between them. This gesture is enabled by default and is one of the most commonly used trackpad gestures on macOS.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enable gesture** (default) — Swipe up opens Mission Control |
| `false` | **Disable gesture** — Swipe up does nothing |

## Our Configuration

**Set to**: `true` — Enable Mission Control gesture for efficient window management.

### Why Enable Mission Control Gesture?

1. **Essential navigation** — Core way to manage windows
2. **Space management** — Create, switch, organize desktops
3. **Window overview** — See all windows at once
4. **Intuitive gesture** — Natural swipe-up motion
5. **Productivity** — Quickly find buried windows

### Why Disable Mission Control Gesture?

- **Accidental triggers** — Gesture triggered by mistake
- **App conflicts** — Some apps use similar gestures
- **Prefer keyboard** — Use Control+↑ instead
- **Hot Corner preference** — Use corner trigger instead
- **Limited use** — Don't use multiple spaces

## Code Examples

```bash
# Read current value
defaults read com.apple.dock showMissionControlGestureEnabled

# Enable Mission Control gesture (default)
defaults write com.apple.dock showMissionControlGestureEnabled -bool true

# Disable Mission Control gesture
defaults write com.apple.dock showMissionControlGestureEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.dock showMissionControlGestureEnabled

# IMPORTANT: Restart Dock to apply changes
killall Dock
```

## Related Settings

```bash
# Mission Control gesture (this setting)
defaults write com.apple.dock showMissionControlGestureEnabled -bool true

# Launchpad gesture
defaults write com.apple.dock showLaunchpadGestureEnabled -bool true

# App Exposé gesture
defaults write com.apple.dock showAppExposeGestureEnabled -bool true

# Show Desktop gesture
defaults write com.apple.dock showDesktopGestureEnabled -bool true

# Group windows by application in Mission Control
defaults write com.apple.dock expose-group-apps -bool true

# Don't rearrange Spaces based on most recent use
defaults write com.apple.dock mru-spaces -bool false

# Mission Control animation speed
defaults write com.apple.dock expose-animation-duration -float 0.1

# Apply changes
killall Dock
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not working | Check System Settings > Trackpad > More Gestures |
| Setting not applying | Run `killall Dock` after changing |
| Gesture conflicts with app | Disable in that app's settings or here |
| Only 3 fingers work | Check gesture configuration in settings |
| Mission Control slow | Reduce expose-animation-duration |
| Spaces keep rearranging | Set mru-spaces to false |

## Compatibility

- **macOS**: 10.7+ (Lion and later)
- **Hardware**: Requires trackpad (built-in or Magic Trackpad)
- **Note**: Requires `killall Dock` to apply changes
- **Note**: Three-finger option available in Accessibility settings
- **Note**: Works with Magic Trackpad 1 and 2
- **Note**: Can use Magic Mouse with two-finger double-tap
