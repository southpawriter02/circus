# Displays Have Separate Spaces

Control whether each monitor has independent virtual desktops.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.spaces` |
| **Key** | `spans-displays` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls how Spaces (virtual desktops) work with multiple displays. When set to `false` (separate Spaces), each monitor has its own independent set of virtual desktops â€” you can have different apps in full-screen on different monitors simultaneously. When set to `true` (spanning), all monitors share a single Space, and switching Spaces changes all monitors at once.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Separate Spaces** (default) â€” Each display has independent Spaces |
| `true` | **Spanning Spaces** â€” All displays share the same Space |

## Our Configuration

**Set to**: `false` â€” Each display has its own Spaces.

### Why Separate Spaces?

1. **Productivity** â€” Use both monitors fully
2. **Full-screen** â€” Apps don't take over all monitors
3. **Independence** â€” Different workflows per screen
4. **Flexibility** â€” Switch Spaces on one monitor only
5. **Standard** â€” Apple's default recommendation

### Why Spanning Spaces?

- Prefer unified desktop experience
- Use windows across multiple monitors
- Drag windows between monitors freely
- Single mental model for Spaces
- Older workflow preference

## Code Examples

```bash
# Read current value
defaults read com.apple.spaces spans-displays

# Separate Spaces per display (recommended)
defaults write com.apple.spaces spans-displays -bool false

# Spaces span all displays
defaults write com.apple.spaces spans-displays -bool true

# Delete (reset to default)
defaults delete com.apple.spaces spans-displays
```

**Important**: This setting requires logout/login to take effect!

```bash
# After changing, logout to apply
osascript -e 'tell application "System Events" to log out'
```

## Related Settings

```bash

# Don't auto-rearrange Spaces
defaults write com.apple.dock mru-spaces -bool false

# Group windows by app in Mission Control
defaults write com.apple.dock expose-group-apps -bool true

# Switch to app's Space when clicking Dock
defaults write NSGlobalDomain AppleSpacesSwitchOnActivate -bool true
```
