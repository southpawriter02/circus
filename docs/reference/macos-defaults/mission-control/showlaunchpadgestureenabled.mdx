# Mission Control Launchpad Gesture

Control whether the trackpad pinch gesture opens Launchpad.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.dock` |
| **Key** | `showLaunchpadGestureEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the four-finger (or thumb and three fingers) pinch gesture on the trackpad opens Launchpad. Launchpad displays all your applications in an iOS-style grid, making it easy to find and launch apps. This gesture is enabled by default and provides a quick way to access your applications without using the Dock or Finder.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enable gesture** (default) â€” Pinch opens Launchpad |
| `false` | **Disable gesture** â€” Pinch does nothing |

## Our Configuration

**Set to**: `true` â€” Enable Launchpad gesture for quick app access.

### Why Enable Launchpad Gesture?

1. **Quick access** â€” Fast way to see all apps
2. **Visual browsing** â€” See app icons at a glance
3. **iOS familiarity** â€” Consistent with iPhone/iPad
4. **Search integration** â€” Type to search apps
5. **Default behavior** â€” Expected by most users

### Why Disable Launchpad Gesture?

- **Accidental triggers** â€” Gesture triggered unintentionally
- **App conflicts** â€” Design apps may use similar gestures
- **Prefer Spotlight** â€” Use Cmd+Space for app launching
- **Keyboard workflow** â€” Don't use trackpad gestures
- **Muscle memory** â€” Gesture conflicts with habits

## Code Examples

```bash
# Read current value
defaults read com.apple.dock showLaunchpadGestureEnabled

# Enable Launchpad gesture (default)
defaults write com.apple.dock showLaunchpadGestureEnabled -bool true

# Disable Launchpad gesture
defaults write com.apple.dock showLaunchpadGestureEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.dock showLaunchpadGestureEnabled

# IMPORTANT: Restart Dock to apply changes
killall Dock
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not working | Check System Settings > Trackpad > More Gestures |
| Setting not applying | Run `killall Dock` after changing |
| Gesture triggers accidentally | Disable via settings or this default |
| Launchpad slow to open | Reduce animation: springboard-show-duration |
| Apps missing from Launchpad | Drag apps to /Applications folder |
| Launchpad layout messy | Reset with ResetLaunchPad setting |

## Compatibility

- **macOS**: 10.7+ (Lion and later)
- **Hardware**: Requires trackpad (built-in or Magic Trackpad)
- **Note**: Requires `killall Dock` to apply changes
- **Note**: Gesture requires multi-touch trackpad support
- **Note**: Also works with Magic Trackpad 1 and 2
