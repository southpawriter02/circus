# AirDrop Over Ethernet

Enable AirDrop functionality over Ethernet connections.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.NetworkBrowser` |
| **Key** | `BrowseAllInterfaces` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether AirDrop can work over Ethernet connections in addition to Wi-Fi. This setting is primarily useful for older Mac Pro and Mac mini models that may not have native AirDrop support via their Wi-Fi hardware. Enabling this allows AirDrop to use the wired network connection for file transfers.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enable Ethernet AirDrop** â€” AirDrop works over wired network |
| `false` | **Wi-Fi only** (default) â€” AirDrop requires Wi-Fi/Bluetooth |

## Our Configuration

**Set to**: `true` â€” Enable for maximum flexibility.

### Why Enable?

1. **Compatibility** â€” Works with older Mac hardware
2. **Reliability** â€” Wired connection more stable
3. **Speed** â€” Gigabit Ethernet faster than Wi-Fi
4. **Flexibility** â€” Works even if Wi-Fi disabled
5. **No downside** â€” Doesn't affect normal AirDrop

### Why Keep Default (Disabled)?

- Standard AirDrop works fine
- Modern Mac with good Wi-Fi
- Prefer wireless-only operation
- Security policy restrictions

## Code Examples

```bash
# Read current value
defaults read com.apple.NetworkBrowser BrowseAllInterfaces

# Enable AirDrop over Ethernet
defaults write com.apple.NetworkBrowser BrowseAllInterfaces -bool true

# Disable (Wi-Fi only, default)
defaults write com.apple.NetworkBrowser BrowseAllInterfaces -bool false

# Delete (reset to default)
defaults delete com.apple.NetworkBrowser BrowseAllInterfaces
```

| Issue | Solution |
|-------|----------|

| Can't see other Macs | Verify both on same network subnet |

| Works wirelessly but not wired | This setting may not be needed |
| Old Mac still not working | Hardware may not support AirDrop at all |

## Compatibility

- **AirDrop**: macOS 10.7+ (Lion)
- **Note**: Primarily for older Macs without native AirDrop Wi-Fi
- **Note**: Modern Macs work without this setting
- **Note**: Bluetooth still required for device discovery
- **Note**: Both Macs need setting enabled for Ethernet transfer

## Source File

**File**: [`defaults/system/airdrop.sh`](../../../../defaults/system/airdrop.sh)

```bash
run_defaults "com.apple.NetworkBrowser" "BrowseAllInterfaces" "-bool" "true"
```
