# AirDrop Discoverable Mode

Control who can see your Mac when using AirDrop.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.sharingd` |
| **Key** | `DiscoverableMode` |
| **Type** | String |
| **Default Value** | `Contacts Only` |
| **Our Setting** | `Contacts Only` |

## Description

Controls who can discover your Mac when AirDrop is active. AirDrop uses Bluetooth to discover nearby devices and Wi-Fi for the actual file transfer. This setting determines your visibility to othersâ€”you can be invisible, visible only to people in your Contacts, or visible to everyone nearby.

## Values

| Value | Behavior |
|-------|----------|
| `Off` | **No one** â€” Your Mac is invisible to AirDrop |
| `Contacts Only` | **Contacts only** (default) â€” Only people in your Contacts can see you |
| `Everyone` | **Everyone** â€” Any nearby Apple device can see you |

## Our Configuration

**Set to**: `Contacts Only` â€” Balanced security.

### Why Contacts Only (Recommended)?

1. **Privacy** â€” Strangers can't see your Mac
2. **Security** â€” Reduces unsolicited content
3. **Convenience** â€” Friends/family can still send files
4. **Automatic** â€” Works seamlessly with your contacts
5. **iCloud integration** â€” Uses your Contacts list

### Why Off?

- Maximum privacy needed
- Don't use AirDrop
- Public environment, extra caution
- Corporate security policy

### Why Everyone?

- Receiving from someone not in Contacts
- Quick file transfer in meetings
- Temporary setting for specific need
- **Switch back after use!**

## Code Examples

```bash
# Read current value
defaults read com.apple.sharingd DiscoverableMode

# Set to Off (not discoverable)
defaults write com.apple.sharingd DiscoverableMode -string "Off"

# Set to Contacts Only (recommended)
defaults write com.apple.sharingd DiscoverableMode -string "Contacts Only"

# Set to Everyone (use temporarily)
defaults write com.apple.sharingd DiscoverableMode -string "Everyone"

# Delete (reset to default)
defaults delete com.apple.sharingd DiscoverableMode
```

## Related Settings

```bash
# AirDrop visibility (this setting)
defaults write com.apple.sharingd DiscoverableMode -string "Contacts Only"

# Enable AirDrop over Ethernet (for older Macs)
defaults write com.apple.NetworkBrowser BrowseAllInterfaces -bool true

# Firewall settings (may affect AirDrop)
# sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't see other devices | Check Wi-Fi and Bluetooth are ON |
| Other devices can't see you | Check DiscoverableMode isn't "Off" |
| Contacts Only not working | Verify both have each other in Contacts |
| AirDrop very slow | Move devices closer together |
| Receiving fails | Check available disk space |

## Compatibility

- **AirDrop**: macOS 10.7+ (Lion)
- **Note**: Requires Wi-Fi and Bluetooth hardware
- **Note**: Works between Mac, iPhone, iPad
- **Note**: Some older Macs need BrowseAllInterfaces for Ethernet
- **Note**: Setting is per-user, not system-wide
