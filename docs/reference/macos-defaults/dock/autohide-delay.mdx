# autohide-delay

Set delay before Dock appears when hidden.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.dock` |
| **Key** | `autohide-delay` |
| **Type** | Float |
| **Default Value** | `0.2` |
| **Our Setting** | `0` |

## Description

Controls the delay (in seconds) before the Dock appears when you move the cursor to the screen edge. Only applies when `autohide` is enabled.

## Values

| Value | Delay | Experience |
|-------|-------|------------|
| `0` | Instant | No wait, immediate appearance |
| `0.2` | 200ms (default) | Brief pause |
| `0.5` | 500ms | Noticeable delay |
| `1.0` | 1 second | Long delay |

## Our Configuration

**Set to**: `0` — Instant Dock appearance when needed.

## Code Examples

```bash
# Read current value
defaults read com.apple.dock autohide-delay

# Instant Dock appearance
defaults write com.apple.dock autohide-delay -float 0

# Apple default (brief delay)
defaults write com.apple.dock autohide-delay -float 0.2

# Delete (reset to default)
defaults delete com.apple.dock autohide-delay

# Restart Dock to apply
killall Dock
```

## UI Location

Not available in System Settings — terminal only.

## Related: Auto-Hide Animation Speed

Control how fast the Dock slides in/out:

```bash
# Speed up show/hide animation
defaults write com.apple.dock autohide-time-modifier -float 0.15

# Default animation speed
defaults write com.apple.dock autohide-time-modifier -float 0.5

killall Dock
```

## Complete Fast Dock Configuration

```bash
# Enable auto-hide with instant response
defaults write com.apple.dock autohide -bool true
defaults write com.apple.dock autohide-delay -float 0
defaults write com.apple.dock autohide-time-modifier -float 0.15
killall Dock
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Delay still present | Restart Dock: `killall Dock` |
| Setting resets on reboot | Ensure using `defaults write` correctly |
| Only works with autohide | This setting requires `autohide -bool true` |

## Compatibility

- **Dock**: macOS 10.2+
- **Note**: Only effective when autohide is enabled
