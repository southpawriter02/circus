# Mission Control Automatic Space Rearrangement

Control whether Spaces automatically reorder based on recent use.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.dock` |
| **Key** | `mru-spaces` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether Mission Control automatically rearranges your Spaces (virtual desktops) based on which ones you've used most recently. When enabled, frequently used Spaces move to the left. When disabled, Spaces maintain their original order, providing a consistent and predictable workspace layout.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Auto-rearrange** (default) â€” Spaces reorder by recent use |
| `false` | **Fixed order** â€” Spaces stay in original position |

## Our Configuration

**Set to**: `false` â€” Keep Spaces in fixed order for predictability.

### Why Disable MRU Spaces?

1. **Predictability** â€” Always know where each Space is
2. **Muscle memory** â€” âŒƒ1, âŒƒ2, âŒƒ3 work consistently
3. **Organization** â€” Intentional workspace layout
4. **Less confusion** â€” No surprise rearrangements
5. **Professional** â€” Stable multi-monitor setup

### Why Enable MRU Spaces?

- Prefer adaptive organization
- Don't use keyboard shortcuts for Spaces
- Want most-used Spaces easily accessible
- Default macOS behavior
- Fewer Spaces to manage (2-3)

## Code Examples

```bash
# Read current value
defaults read com.apple.dock mru-spaces

# Disable auto-rearrangement (fixed order)
defaults write com.apple.dock mru-spaces -bool false

# Enable auto-rearrangement (default)
defaults write com.apple.dock mru-spaces -bool true

# Delete (reset to default)
defaults delete com.apple.dock mru-spaces

# Apply changes
killall Dock
```
