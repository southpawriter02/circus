# Dock Show Suggested and Recent Apps

Control recent/suggested applications section in Dock.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.dock` |
| **Key** | `show-recents` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether macOS shows a "Suggested and Recent Apps" section at the right side of the Dock. This section displays apps you've used recently that aren't permanently in the Dock, plus Handoff apps from other devices. While helpful for some, it clutters the Dock for power users.

## Values

| Value | Display |
|-------|---------|
| `true` | **Show recents** (default) â€” Recent apps visible |
| `false` | **Hide recents** â€” Only pinned apps shown |

## Our Configuration

**Set to**: `false` â€” Hide recent apps.

### Why Hide Recents?

1. **Clean Dock** â€” Only apps you chose
2. **Consistent** â€” Dock doesn't change
3. **Privacy** â€” Others can't see recent apps
4. **Control** â€” You decide what's in Dock
5. **Muscle memory** â€” Icons stay in same positions

### Why Show Recents?

- Quick access to just-used apps
- Discover apps you forgot about
- Handoff from other Apple devices
- App suggestions based on usage

## Code Examples

```bash
# Read current value
defaults read com.apple.dock show-recents

# Hide recent apps (recommended for power users)
defaults write com.apple.dock show-recents -bool false

# Show recent apps (default)
defaults write com.apple.dock show-recents -bool true

# Restart Dock to apply
killall Dock
```

## Handoff Implications

With recents enabled:
- Apps from your iPhone/iPad appear here
- Click to open and continue work
- Part of Apple Continuity

With recents disabled:
- Handoff still works via other methods
- Use âŒ˜Tab or Spotlight to access
- Just not shown in Dock

## Dock Organization Philosophy

Two schools of thought:

**Minimal Dock** (recents off):
- Pin only essentials
- Use Spotlight (âŒ˜Space) for rest
- Consistent, predictable

**Dynamic Dock** (recents on):
- Pin main apps
- Let recents suggest others
- Adapts to workflow
