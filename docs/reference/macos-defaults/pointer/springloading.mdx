# Spring-Loading for Folders

Enable automatic folder opening when hovering with dragged items.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Keys** | `com.apple.springing.enabled`, `com.apple.springing.delay` |
| **Type** | Boolean, Float |
| **Default Value** | `true`, `0.5` |
| **Our Setting** | `true`, `0.5` |

## Description

Spring-loading automatically opens a folder when you hover over it while dragging a file. After a brief delay, the folder "springs" open, letting you navigate to nested folders without dropping the dragged item. Release the file at any level to drop it there.

## Values

### Enable Spring-Loading

| Value | Behavior |
|-------|----------|
| `true` | **Enabled** (default) â€” Folders open when hovering |
| `false` | **Disabled** â€” Must drop first, then move |

### Spring-Loading Delay (seconds)

| Value | Speed |
|-------|-------|
| `0.0` | Instant (no delay) |
| `0.25` | Very fast |
| `0.5` | Default |
| `1.0` | Slow |
| `2.0` | Very slow |

## Our Configuration

**Enabled**: `true` â€” Spring-loading active
**Delay**: `0.5` seconds â€” Quick but not instant

### Why Enable Spring-Loading?

1. **Deep navigation** â€” Access nested folders while dragging
2. **Efficiency** â€” No need to pre-open folders
3. **Discovery** â€” Explore folder structure with dragged item
4. **Time saving** â€” Single drag operation

### Adjusting Delay

- **Shorter delay (0.25)**: Fast workflow, experienced users
- **Default (0.5)**: Good balance
- **Longer delay (1.0+)**: Prevent accidental opening

## Code Examples

```bash
# Read current values
defaults read NSGlobalDomain com.apple.springing.enabled
defaults read NSGlobalDomain com.apple.springing.delay

# Enable spring-loading (default)
defaults write NSGlobalDomain com.apple.springing.enabled -bool true

# Set faster delay (0.25 seconds)
defaults write NSGlobalDomain com.apple.springing.delay -float 0.25

# Set default delay
defaults write NSGlobalDomain com.apple.springing.delay -float 0.5

# Disable spring-loading
defaults write NSGlobalDomain com.apple.springing.enabled -bool false

# Delete (reset to defaults)
defaults delete NSGlobalDomain com.apple.springing.enabled
defaults delete NSGlobalDomain com.apple.springing.delay
```

## How to Use

1. **Start dragging** a file
2. **Hover over** a folder (don't release)
3. **Wait** for the delay (folder highlights/pulses)
4. **Folder opens** â€” navigate inside
5. **Continue** to deeper folders as needed
6. **Release** to drop file at current location

## Spring-Loading Works On

- Finder folders
- Dock folders
- Sidebar folders
- Desktop folders
- Open/Save dialogs

## Escape to Cancel

While spring-loading:
- Press **Escape** to close opened folders
- Drag away without releasing cancels navigation

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Folders not opening | Check `com.apple.springing.enabled` |
| Opens too slowly | Decrease delay value |
| Opens accidentally | Increase delay value |
| Inconsistent behavior | Restart Finder |
