# Finder CreateDesktop

Completely hide or show the Desktop icons and files.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.finder` |
| **Key** | `CreateDesktop` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the Desktop displays any icons at all, including files, folders, drives, and aliases. When disabled, the Desktop becomes a clean wallpaper-only surface. Files still exist in `~/Desktop` but are not rendered visually. This is different from hiding specific device iconsâ€”this hides EVERYTHING.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show Desktop items** (default) â€” Normal Desktop with all icons |
| `false` | **Hide Desktop items** â€” Clean wallpaper, no icons visible |

## Our Configuration

**Set to**: `true` â€” Keep Desktop functional.

### Why Enable (Default)?

1. **File access** â€” Quick access to commonly used files
2. **Drag and drop** â€” Desktop as staging area
3. **Visual reminders** â€” See pending tasks/files
4. **Convention** â€” Expected behavior for most users

### Why Disable?

1. **Clean aesthetic** â€” Minimalist, distraction-free workspace
2. **Focus** â€” Nothing to tempt you away from work
3. **Presentations** â€” Professional look when sharing screen
4. **Philosophy** â€” Force use of organized folders instead

## Code Examples

```bash
# Read current value
defaults read com.apple.finder CreateDesktop

# Hide all Desktop icons (clean Desktop)
defaults write com.apple.finder CreateDesktop -bool false
killall Finder

# Show Desktop icons (normal behavior)
defaults write com.apple.finder CreateDesktop -bool true
killall Finder

# Delete (reset to default)
defaults delete com.apple.finder CreateDesktop
killall Finder
```

## Toggle Script

Create a quick toggle script:

```bash
#!/bin/bash
# Toggle Desktop icons visibility

current=$(defaults read com.apple.finder CreateDesktop 2>/dev/null)

if [ "$current" = "0" ] || [ "$current" = "false" ]; then
    defaults write com.apple.finder CreateDesktop -bool true
    echo "Desktop icons: VISIBLE"
else
    defaults write com.apple.finder CreateDesktop -bool false
    echo "Desktop icons: HIDDEN"
fi

killall Finder
```

## Use Cases

### Presentation Mode

```bash
# Before presentation - clean Desktop
defaults write com.apple.finder CreateDesktop -bool false
killall Finder

# After presentation - restore
defaults write com.apple.finder CreateDesktop -bool true
killall Finder
```

### Minimalist Setup

Combine with other settings for ultimate minimalism:

```bash
# Complete minimalist setup
defaults write com.apple.finder CreateDesktop -bool false
defaults write com.apple.dock autohide -bool true
defaults write com.apple.dock tilesize -int 36
killall Finder
killall Dock
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Desktop items still visible | Run `killall Finder` |
| Can't find files | They're in ~/Desktop folder |
| Setting reverts | Check no profile management overrides |
| Wallpaper not showing | Unrelated; check Desktop & Screen Saver prefs |
