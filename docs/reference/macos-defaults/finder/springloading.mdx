# Finder Spring Loading for Folders

Enable automatic folder opening when dragging items over folders.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `com.apple.springing.enabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Spring loading allows you to hover over a folder while dragging a file, and after a brief delay, the folder automatically opens (springs open). This enables deep navigation through folder hierarchies without releasing the dragged item. Continue hovering over subfolders to drill down further, then drop the file at the desired location. The delay before folders open can be customized.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enable spring loading** (default) â€” Folders open when hovering |
| `false` | **Disable spring loading** â€” Must manually open folders |

## Our Configuration

**Set to**: `true` â€” Enable spring loading for efficient file organization.

### Why Enable Spring Loading?

1. **Efficiency** â€” Navigate while dragging
2. **Deep hierarchies** â€” Access nested folders easily
3. **One-handed** â€” Complete operation without releasing mouse
4. **Natural** â€” Intuitive drag-and-drop workflow
5. **Time-saving** â€” No pre-navigation required

### Why Disable Spring Loading?

- Accidental folder openings
- Prefer deliberate navigation
- Unpredictable behavior with many folders
- Performance concerns (older Macs)
- Prefer keyboard navigation

## Spring Loading Delay

Customize how long to hover before folders open:

| Delay | Value | Description |
|-------|-------|-------------|
| Short | `0.5` | Quick response |
| Medium | `1.0` | Default, balanced |
| Long | `1.5` | More deliberate |

```bash
# Set spring loading delay (in seconds)
defaults write NSGlobalDomain com.apple.springing.delay -float 0.5

# Longer delay
defaults write NSGlobalDomain com.apple.springing.delay -float 1.5
```

## Code Examples

```bash
# Read spring loading status
defaults read NSGlobalDomain com.apple.springing.enabled

# Enable spring loading (default)
defaults write NSGlobalDomain com.apple.springing.enabled -bool true

# Disable spring loading
defaults write NSGlobalDomain com.apple.springing.enabled -bool false

# Read spring loading delay
defaults read NSGlobalDomain com.apple.springing.delay

# Set spring loading delay (0.5 seconds)
defaults write NSGlobalDomain com.apple.springing.delay -float 0.5

# Reset to defaults
defaults delete NSGlobalDomain com.apple.springing.enabled
defaults delete NSGlobalDomain com.apple.springing.delay
```

| Opens too quickly | Increase delay value |
