# AppleShowAllExtensions

Show all file extensions in Finder.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `AppleShowAllExtensions` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Finder displays file extensions for all files. By default, macOS hides extensions for "known" file types. Showing all extensions is essential for developers and improves security.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show all extensions** — .txt, .jpg, .sh, etc. visible |
| `false` | **Hide known extensions** (default) — Only shows "unusual" extensions |

## Our Configuration

**Set to**: `true` — Essential for developers and security.

## Why Show Extensions?

1. **Distinguish files**: `script` vs `script.sh` vs `script.py`
2. **Security**: Spot malicious files like `document.pdf.exe`
3. **Development**: Know exactly what file type you're working with
4. **Clarity**: No guessing based on icons

## Code Examples

```bash
# Read current value
defaults read NSGlobalDomain AppleShowAllExtensions

# Show all file extensions
defaults write NSGlobalDomain AppleShowAllExtensions -bool true

# Hide known extensions (Apple default)
defaults write NSGlobalDomain AppleShowAllExtensions -bool false

# Delete (reset to default)
defaults delete NSGlobalDomain AppleShowAllExtensions

# Restart Finder to apply
killall Finder
```

## UI Location

**Finder** → **Settings** (⌘,) → **Advanced** → **Show all filename extensions**

## Security Note

Showing extensions helps identify disguised malicious files:
- `invoice.pdf` (icon looks like PDF but might be something else)
- `invoice.pdf.app` (clearly an application, not a PDF)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Extensions still hidden | Restart Finder: `killall Finder` |
| Some files still have no extension | File genuinely has no extension |
| Per-file overrides | Uncheck "Hide extension" in File → Get Info |

## Compatibility

- **Finder**: macOS 10.4+
- **Applies to**: All Finder views
