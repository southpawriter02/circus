# ShowStatusBar

Show item count and disk space in Finder windows.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.finder` |
| **Key** | `ShowStatusBar` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays a status bar at the bottom of Finder windows showing the number of items in the current folder and available disk space.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show status bar** — Item count and disk space visible |
| `false` | **Hide status bar** (default) — No status information |

## Our Configuration

**Set to**: `true` — Useful for storage monitoring at a glance.

## Code Examples

```bash
# Read current value
defaults read com.apple.finder ShowStatusBar

# Show status bar
defaults write com.apple.finder ShowStatusBar -bool true

# Hide status bar
defaults write com.apple.finder ShowStatusBar -bool false

# Delete (reset to default)
defaults delete com.apple.finder ShowStatusBar

# Restart Finder to apply
killall Finder
```

## UI Location

**View** menu → **Show Status Bar** (⌘/)

## What It Shows

- **Item count**: "42 items" (or selection count when items selected)
- **Disk space**: "125.6 GB available"

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Status bar not showing | Restart Finder: `killall Finder` |
| Shows wrong count | Select items to see selection count |
| Disk space wrong | Check for hidden items |

## Compatibility

- **Finder**: macOS 10.5+
- **Applies to**: All Finder windows

## Source File

**File**: [`defaults/interface/finder.sh`](../../../defaults/interface/finder.sh)

```bash
run_defaults "com.apple.finder" "ShowStatusBar" "-bool" "true"
```
