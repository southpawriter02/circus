# _FXSortFoldersFirst

Keep folders at the top when sorting by name.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.finder` |
| **Key** | `_FXSortFoldersFirst` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

When sorting by name, keeps folders grouped at the top of the list, separate from files. Matches behavior of most file managers.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Folders first** â€” Folders at top, then files |
| `false` | **Mixed** (default) â€” Folders and files sorted together |

## Code Examples

```bash
# Keep folders on top
defaults write com.apple.finder _FXSortFoldersFirst -bool true

# Mix folders with files (Apple default)
defaults write com.apple.finder _FXSortFoldersFirst -bool false

killall Finder
```
