# Finder Column View Default Width

Set the default width of columns in Finder's column view.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.finder` |
| **Key** | `ColumnWidth` |
| **Type** | Integer (pixels) |
| **Default Value** | `175` |
| **Our Setting** | `200` |

## Description

Controls the default width of columns when using Finder's column view. Wider columns show more of file names without truncation, while narrower columns allow more columns to be visible on screen. This setting affects new Finder windows and folders that haven't had their column width manually adjusted.

## Values

| Value | Width | Description |
|-------|-------|-------------|
| `100` | Narrow | More columns visible, names truncated |
| `150` | Compact | Balance of width and visibility |
| `175` | Default | Standard macOS column width |
| `200` | Wide | Better name visibility |
| `250` | Very Wide | Full names, fewer columns |
| `300` | Extra Wide | Maximum readability |

## Our Configuration

**Set to**: `200` â€” Wider columns for better file name visibility.

### Why Use Wider Columns?

1. **Readability** â€” See full file names
2. **Navigation** â€” Less guessing truncated names
3. **Professional** â€” Descriptive file names visible
4. **Projects** â€” Long folder names common
5. **Fewer errors** â€” Select correct file first time

### Why Use Narrower Columns?

- See more directory depth
- Small screen real estate
- Shallow folder structures
- Short file names
- Preference for more columns

## Code Examples

```bash
# Read current value
defaults read com.apple.finder ColumnWidth

# Set wider columns (200px)
defaults write com.apple.finder ColumnWidth -int 200

# Set default columns (175px)
defaults write com.apple.finder ColumnWidth -int 175

# Set narrow columns (150px)
defaults write com.apple.finder ColumnWidth -int 150

# Delete (reset to default)
defaults delete com.apple.finder ColumnWidth

# Restart Finder to apply
killall Finder
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Width not applying | Run `killall Finder` |
| Some folders different | Per-folder settings override |
| Columns too narrow | Increase ColumnWidth value |
| Can't see all columns | Scroll right or narrow columns |
| Reset all columns | Delete `.DS_Store` files |

## Resetting Folder View Settings

To reset view settings for a folder:
```bash
# Remove .DS_Store in specific folder (resets that folder's settings)
rm ~/Documents/.DS_Store

# Remove all .DS_Store files (resets all folder settings)
find ~ -name ".DS_Store" -delete
```
