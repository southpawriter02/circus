# Finder Show External Hard Drives on Desktop

Display external hard drives as icons on the desktop.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.finder` |
| **Key** | `ShowExternalHardDrivesOnDesktop` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether external hard drives, USB drives, and other removable storage devices appear as icons on the macOS desktop. When enabled, connecting an external drive creates a visible desktop icon for quick access. When disabled, external drives are only accessible through Finder windows, keeping the desktop cleaner but requiring extra navigation to access connected storage.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show on desktop** (default) â€” External drives visible on desktop |
| `false` | **Hide from desktop** â€” Access via Finder sidebar only |

## Our Configuration

**Set to**: `true` â€” Show external drives on desktop for easy access.

### Why Show External Drives?

1. **Quick access** â€” One-click to open drive
2. **Visual feedback** â€” See when drives connect
3. **Easy ejection** â€” Drag to trash or right-click
4. **File transfers** â€” Drag files directly to drive
5. **Status awareness** â€” Know what's connected

### Why Hide External Drives?

- Clean desktop aesthetic
- Security (hide connected drives)
- Prevent accidental interactions
- Prefer Finder sidebar navigation
- Many drives connected frequently

## Code Examples

```bash
# Read current value
defaults read com.apple.finder ShowExternalHardDrivesOnDesktop

# Show external drives on desktop (default)
defaults write com.apple.finder ShowExternalHardDrivesOnDesktop -bool true

# Hide external drives from desktop
defaults write com.apple.finder ShowExternalHardDrivesOnDesktop -bool false

# Delete (reset to default)
defaults delete com.apple.finder ShowExternalHardDrivesOnDesktop

# Restart Finder to apply
killall Finder
```

## Complete Desktop Configuration

```bash
# Minimalist: Hide all drive icons
defaults write com.apple.finder ShowHardDrivesOnDesktop -bool false
defaults write com.apple.finder ShowExternalHardDrivesOnDesktop -bool false
defaults write com.apple.finder ShowRemovableMediaOnDesktop -bool false
defaults write com.apple.finder ShowMountedServersOnDesktop -bool false
killall Finder

defaults write com.apple.finder ShowHardDrivesOnDesktop -bool true
defaults write com.apple.finder ShowExternalHardDrivesOnDesktop -bool true
defaults write com.apple.finder ShowRemovableMediaOnDesktop -bool true
defaults write com.apple.finder ShowMountedServersOnDesktop -bool true

defaults write com.apple.finder ShowExternalHardDrivesOnDesktop -bool true
defaults write com.apple.finder ShowRemovableMediaOnDesktop -bool true
defaults write com.apple.finder ShowMountedServersOnDesktop -bool false
killall Finder
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Drive not appearing | Check if mounted in Disk Utility |
| Icon appears then disappears | Drive may be auto-ejecting (check Console) |
| Can't eject drive | Close apps using the drive, try `diskutil eject` |
| Wrong icon showing | macOS caches icons; restart Finder |
| Desktop icons disabled | Check System Settings â†’ Desktop & Dock |

## Security Considerations

| Scenario | Recommendation |
|----------|----------------|
| Shared workspace | Hide (false) - prevents casual browsing |
| Personal Mac | Show (true) - convenience |
| Sensitive data | Hide (false) - less obvious |
| Quick transfers | Show (true) - easier drag-drop |

## Command Line Drive Access

Even without desktop icons:

```bash
# List all mounted volumes
ls /Volumes/

# Open specific drive in Finder
open /Volumes/Backup\ Drive/

# Eject drive
diskutil eject /Volumes/Backup\ Drive

# Show disk info
diskutil info /Volumes/Backup\ Drive
```
