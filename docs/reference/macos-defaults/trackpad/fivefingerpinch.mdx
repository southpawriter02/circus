# Trackpad Five-Finger Pinch

Pinch with five fingers to open Launchpad.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `TrackpadFiveFingerPinchGesture` |
| **Type** | Integer |
| **Default Value** | `2` |
| **Our Setting** | `2` |

## Description

Controls the five-finger pinch gesture on the trackpad. Pinching five fingers together opens Launchpad (iOS-style app launcher), while spreading five fingers apart shows the desktop by moving all windows aside. This provides quick access to app launching and desktop viewing.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **Disabled** — Five-finger pinch does nothing |
| `2` | **Launchpad / Show Desktop** (default) — Pinch in for Launchpad, spread for desktop |

## Our Configuration

**Set to**: `2` — Five-finger pinch for Launchpad and Show Desktop.

### Why Enable This Gesture?

1. **Quick app access** — Launch apps without Dock or Spotlight
2. **Desktop access** — See desktop files instantly
3. **iOS familiarity** — Similar to iPad gesture
4. **Workflow** — One gesture to start apps
5. **Natural** — Intuitive pinch/spread motion

### Why Disable?

- Prefer Spotlight for launching (Cmd + Space)
- Accidental triggers with large hand movements
- Don't use Launchpad
- Prefer keyboard shortcuts
- F11 for Show Desktop is sufficient

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad TrackpadFiveFingerPinchGesture

# Enable five-finger pinch (default)
defaults write com.apple.AppleMultitouchTrackpad TrackpadFiveFingerPinchGesture -int 2

# Also set for Bluetooth trackpad
defaults write com.apple.driver.AppleBluetoothMultitouch.trackpad TrackpadFiveFingerPinchGesture -int 2

# Disable gesture
defaults write com.apple.AppleMultitouchTrackpad TrackpadFiveFingerPinchGesture -int 0
defaults write com.apple.driver.AppleBluetoothMultitouch.trackpad TrackpadFiveFingerPinchGesture -int 0

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad TrackpadFiveFingerPinchGesture
```

Log out and back in for changes to take effect.

## Keyboard Alternatives

| Shortcut | Action |
|----------|--------|
| F4 | Launchpad (some keyboards) |
| Fn + F11 | Show Desktop |
| Cmd + Space | Spotlight (alternative to Launchpad) |
| Fn + F4 | Launchpad |

## Launchpad vs Spotlight

| Feature | Launchpad | Spotlight |
|---------|-----------|-----------|
| Interface | Visual grid | Search field |
| Best for | Browsing apps | Finding specific app |
| Discovery | Good (see all apps) | Poor (must know name) |
| Speed | Medium | Fast (type and Enter) |
| Organization | Folders, pages | Recent apps shown |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not responding | Ensure all five fingers touch trackpad |
| Launchpad not opening | Check gesture is enabled in settings |
| Desktop not showing | May be disabled; check settings |
| Gesture too sensitive | Try more deliberate pinch/spread |
| Works inconsistently | Clean trackpad surface |

## Compatibility

- **macOS**: 10.7+ (Lion)
- **Hardware**: All Multi-Touch trackpads
- **Note**: Requires logout for changes to apply
- **Note**: Works with Magic Trackpad as well
- **Note**: Launchpad introduced in Lion (10.7)
