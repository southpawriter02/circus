# Trackpad Four-Finger Horizontal Swipe

Swipe between full-screen apps and desktops with four fingers.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `TrackpadFourFingerHorizSwipeGesture` |
| **Type** | Integer |
| **Default Value** | `2` |
| **Our Setting** | `2` |

## Description

Controls the four-finger horizontal swipe gesture on the trackpad. When enabled, swiping left or right with four fingers switches between full-screen applications and desktop Spaces. This is one of macOS's most powerful navigation gestures for managing multiple workspaces.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **Disabled** — Four-finger horizontal swipe does nothing |
| `1` | **Navigate** — Swipe to navigate (varies by context) |
| `2` | **Switch Spaces** (default) — Swipe between full-screen apps and Spaces |

## Our Configuration

**Set to**: `2` — Four-finger swipe switches between Spaces.

### Why Enable This Gesture?

1. **Multitasking** — Quickly switch workspaces
2. **Full-screen apps** — Navigate between maximized applications
3. **Productivity** — Separate contexts (work, personal, research)
4. **Natural** — Intuitive left/right navigation
5. **Speed** — Faster than clicking or keyboard shortcuts

### Why Disable?

- Prefer keyboard shortcuts (Control + Left/Right)
- Accidental triggers during use
- Using three-finger alternative
- Don't use multiple Spaces
- Gaming or specific app conflicts

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture

# Enable four-finger horizontal swipe for Spaces (default)
defaults write com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture -int 2

# Also set for Bluetooth trackpad
defaults write com.apple.driver.AppleBluetoothMultitouch.trackpad TrackpadFourFingerHorizSwipeGesture -int 2

# Disable gesture
defaults write com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture -int 0
defaults write com.apple.driver.AppleBluetoothMultitouch.trackpad TrackpadFourFingerHorizSwipeGesture -int 0

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture
```

Log out and back in for changes to take effect.

## Related Settings

```bash
# Four-finger horizontal swipe (this setting)
defaults write com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture -int 2

# Four-finger vertical swipe (Mission Control / App Exposé)
defaults write com.apple.AppleMultitouchTrackpad TrackpadFourFingerVertSwipeGesture -int 2

# Three-finger horizontal swipe (alternative)
defaults write com.apple.AppleMultitouchTrackpad TrackpadThreeFingerHorizSwipeGesture -int 2

# Five-finger pinch (Launchpad)
defaults write com.apple.AppleMultitouchTrackpad TrackpadFiveFingerPinchGesture -int 2
```

## Keyboard Alternatives

| Shortcut | Action |
|----------|--------|
| Control + Left Arrow | Move to Space on the left |
| Control + Right Arrow | Move to Space on the right |
| Control + Up Arrow | Mission Control |
| Control + Down Arrow | App Exposé (windows of current app) |
| Control + 1-9 | Jump to specific Space |

## Spaces Workflow Tips

### Creating Spaces:
```
+-------------------------------------------------------------+
|  Managing Spaces via Mission Control                         |
+-------------------------------------------------------------+
|                                                              |
|  1. Swipe up with four fingers (Mission Control)            |
|  2. Hover at top of screen to see Space bar                 |
|                                                              |
|  +------------------------------------------------------+   |
|  | [+] Desktop 1 | Desktop 2 | Safari FS | Desktop 3     |  |
|  +------------------------------------------------------+   |
|        ^                                                     |
|    Click + to add new Space                                 |
|                                                              |
|  3. Drag windows to different Spaces                        |
|  4. Drag window to right edge to create new Space          |
|                                                              |
+-------------------------------------------------------------+
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not working | Check if enabled in System Settings |
| Only see one Space | Create additional Spaces in Mission Control |
| Swipe goes wrong direction | Spaces are ordered left-to-right |
| Three-finger working instead | Check gesture configuration |
| Full-screen apps not in order | Reorder in Mission Control |

## Compatibility

- **macOS**: 10.7+ (Lion)
- **Hardware**: All Multi-Touch trackpads
- **Note**: Requires logout for changes to apply
- **Note**: Works with Magic Trackpad as well
- **Note**: Full-screen apps create their own Space
