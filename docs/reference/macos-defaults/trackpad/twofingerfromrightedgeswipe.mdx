# Trackpad Swipe Between Pages

Swipe with two fingers from edge to navigate between pages.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `TrackpadTwoFingerFromRightEdgeSwipeGesture` |
| **Type** | Integer |
| **Default Value** | `3` |
| **Our Setting** | `3` |

## Description

Controls the gesture for navigating between pages in apps like Safari, Preview, and other document-based applications. This setting determines whether horizontal two-finger swipes trigger back/forward navigation and how the gesture behaves. This is one of the most commonly used trackpad gestures for web browsing.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **Disabled** — No swipe navigation |
| `2` | **Swipe with two fingers** — Horizontal swipe to navigate |
| `3` | **Scroll with two or three fingers** (default) — Natural scroll-like navigation |

## Our Configuration

**Set to**: `3` — Scroll-like navigation for natural feel.

### Why Use Scroll-Like (Value 3)?

1. **Natural** — Feels like scrolling through pages
2. **Preview** — See page sliding as you swipe
3. **Control** — Can "peek" at previous page
4. **Forgiving** — Rubber-band effect if you change mind
5. **Default** — What Apple recommends

### Why Use Distinct Swipe (Value 2)?

- Clearer separation from scrolling
- Faster page transitions
- Less accidental triggers
- More deliberate action
- Prefer quick flick gesture

### Why Disable (Value 0)?

- Conflicts with other gestures
- Accidental back/forward
- Prefer keyboard shortcuts
- Use mouse for navigation
- App-specific issues

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture

# Scroll with two or three fingers (default)
defaults write com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture -int 3

# Swipe with two fingers
defaults write com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture -int 2

# Disable swipe navigation
defaults write com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture -int 0

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture
```

Changes may require logout to take effect.

## Related Settings

```bash
# Swipe between pages (this setting)
defaults write com.apple.AppleMultitouchTrackpad TrackpadTwoFingerFromRightEdgeSwipeGesture -int 3

# Swipe between full-screen apps/Spaces
defaults write com.apple.AppleMultitouchTrackpad TrackpadThreeFingerHorizSwipeGesture -int 2

# Four-finger horizontal swipe
defaults write com.apple.AppleMultitouchTrackpad TrackpadFourFingerHorizSwipeGesture -int 2

# Three-finger drag (may affect swipe gestures)
defaults write com.apple.AppleMultitouchTrackpad TrackpadThreeFingerDrag -bool true

# Natural scrolling direction
defaults write NSGlobalDomain com.apple.swipescrolldirection -bool true
```

## Gesture Comparison

| Gesture | Function |
|---------|----------|
| Two-finger horizontal scroll | Swipe between pages |
| Three-finger horizontal swipe | Switch between Spaces |
| Four-finger horizontal swipe | Switch between Spaces |
| Two-finger vertical scroll | Scroll content |

## Keyboard Alternatives

If you disable this gesture:

| Action | Keyboard Shortcut |
|--------|------------------|
| Go Back | ⌘[ or ⌘← |
| Go Forward | ⌘] or ⌘→ |
| Previous Page (PDF) | ← or Page Up |
| Next Page (PDF) | → or Page Down |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not working | Check if enabled, logout/login |
| Conflicts with scrolling | Try value 2 for distinct swipe |
| Works in Safari not Chrome | Some apps have own settings |
| Accidental triggers | Try value 2 or disable |
| Three-finger conflicts | Disable three-finger drag |
