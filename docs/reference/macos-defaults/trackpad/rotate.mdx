# Trackpad Rotation Gesture

Enable two-finger rotation gesture to rotate content.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `TrackpadRotate` |
| **Type** | Boolean/Integer |
| **Default Value** | `true` (1) |
| **Our Setting** | `true` (1) |

## Description

Controls the two-finger rotation gesture on Multi-Touch trackpads. When enabled, placing two fingers on the trackpad and rotating them (like turning a dial) rotates content in supported applications such as Photos, Preview, and Maps. This is useful for rotating images, adjusting angles, and orienting maps.

## Values

| Value | Behavior |
|-------|----------|
| `1` / `true` | **Enabled** (default) — Two-finger rotate works |
| `0` / `false` | **Disabled** — Rotation gesture ignored |

## Our Configuration

**Set to**: `true` — Enable rotation for natural content manipulation.

### Why Enable Rotation?

1. **Intuitive** — Natural gesture like turning a dial
2. **Photo editing** — Quick image straightening
3. **Maps** — Rotate map to match direction
4. **Precise** — Fine-grained angle control
5. **Efficient** — Faster than menu options

### Why Disable Rotation?

- Accidental rotation during other gestures
- Not used in your workflow
- Prefer keyboard shortcuts
- Conflicts with design app controls
- Finding it frustrating

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad TrackpadRotate

# Enable rotation gesture
defaults write com.apple.AppleMultitouchTrackpad TrackpadRotate -bool true

# Also set for Bluetooth trackpad
defaults write com.apple.driver.AppleBluetoothMultitouch.trackpad TrackpadRotate -bool true

# Disable rotation gesture
defaults write com.apple.AppleMultitouchTrackpad TrackpadRotate -bool false

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad TrackpadRotate
```

Changes may require logout to take effect.

```bash
# Natural scrolling
defaults write NSGlobalDomain com.apple.swipescrolldirection -bool true

# Three-finger drag
defaults write com.apple.AppleMultitouchTrackpad TrackpadThreeFingerDrag -bool true
```

## Keyboard Alternative

When rotation gesture is disabled or unavailable:

| Application | Keyboard Shortcut |
|-------------|-------------------|
| Photos | Cmd + R (90 CW), Cmd + L (90 CCW) |
| Preview | Cmd + R (CW), Cmd + L (CCW) |
| Many apps | Edit > Rotate menu |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Gesture not recognized | Use smoother motion, keep fingers together |
| Rotating when zooming | Keep fingers same distance apart |
| App doesn't rotate | App may not support the gesture |
| Too sensitive | Practice deliberate circular motion |
| Only works one direction | Clean trackpad, check settings |

## Gesture Comparison

| Gesture | Fingers | Motion | Action |
|---------|---------|--------|--------|
| Rotate | 2 | Circular | Rotate content |
| Pinch | 2 | Pinch/spread | Zoom |
| Scroll | 2 | Linear up/down | Scroll |
| Smart zoom | 2 | Double-tap | Zoom to fit |

## Compatibility

- **macOS**: 10.7+ (Lion)
- **Hardware**: Multi-Touch trackpad (built-in or Magic Trackpad)
- **Note**: Application must support rotation
- **Note**: Set both built-in and Bluetooth trackpad domains
