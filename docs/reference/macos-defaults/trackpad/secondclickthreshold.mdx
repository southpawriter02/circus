# Trackpad Second Click Threshold

Set the pressure required for Force Click (deep press) on Force Touch trackpads.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `SecondClickThreshold` |
| **Type** | Integer |
| **Default Value** | `1` |
| **Our Setting** | `1` |

## Description

Controls how much pressure is required to trigger a Force Click (the second, deeper click) on Force Touch trackpads. Force Click enables features like Quick Look, dictionary lookups, and variable-speed media controls. A lower threshold makes Force Click easier to trigger, while a higher threshold requires more deliberate pressure. This is separate from the regular click threshold (FirstClickThreshold).

## Values

| Value | Pressure | Description |
|-------|----------|-------------|
| `0` | **Light** | Easiest Force Click; minimal additional pressure |
| `1` | **Medium** | Balanced sensitivity (default) |
| `2` | **Firm** | Requires deliberate deep press |

## Our Configuration

**Set to**: `1` â€” Medium threshold for balanced Force Click experience.

### Why Use Medium (1)?

1. **Balanced** â€” Not too sensitive, not too hard
2. **Default** â€” Apple's chosen default
3. **Predictable** â€” Force Click when intended
4. **Accessible** â€” Works for most users

### Why Use Light (0)?

- Heavy Force Click user
- Quick Look frequently used
- Prefer minimal pressure
- Easier on fingers

### Why Use Firm (2)?

- Accidentally trigger Force Click often
- Prefer deliberate actions only
- Strong press style
- Rarely use Force Click features

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad SecondClickThreshold

# Set to light pressure
defaults write com.apple.AppleMultitouchTrackpad SecondClickThreshold -int 0

# Set to medium pressure (default)
defaults write com.apple.AppleMultitouchTrackpad SecondClickThreshold -int 1

# Set to firm pressure
defaults write com.apple.AppleMultitouchTrackpad SecondClickThreshold -int 2

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad SecondClickThreshold
```

```bash
defaults write com.apple.AppleMultitouchTrackpad Clicking -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Force Click too sensitive | Increase to 2 (Firm) |
| Force Click too hard to trigger | Decrease to 0 (Light) |
| Force Click not working | Check ActuateDetents is enabled |
| Feature not responding | Verify Force Click enabled in settings |
| Inconsistent behavior | Log out/in after defaults change |

## Compatibility

- **macOS**: 10.11+ (El Capitan) with Force Touch trackpad
- **Hardware**: MacBook 2015+, MacBook Pro 2015+, Magic Trackpad 2
- **Note**: Only applies to Force Touch trackpads
- **Note**: Traditional trackpads don't support Force Click
- **Note**: May require logout for changes to apply
