# Trackpad Actuate Detents

Enable or disable Force Click and haptic feedback on Force Touch trackpads.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AppleMultitouchTrackpad` |
| **Key** | `ActuateDetents` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Force Touch trackpads provide haptic feedback (the simulated "click" sensation) when pressing. Force Touch trackpads don't physically click—they use the Taptic Engine to simulate the feeling. When disabled, the trackpad still registers clicks but provides no tactile feedback. This setting also affects Force Click (deep press) functionality on supported trackpads.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Haptic feedback enabled** (default) — Feel click sensation |
| `false` | **Haptic feedback disabled** — Silent, no tactile click |

## Our Configuration

**Set to**: `true` — Keep haptic feedback enabled for natural feel.

### Why Enable?

1. **Natural feel** — Simulates physical clicking
2. **Feedback** — Know when clicks register
3. **Force Click** — Enables Quick Look and other features
4. **Consistency** — Expected trackpad behavior

### Why Disable?

- Completely silent operation needed
- Prefer tap-to-click exclusively
- Haptic sensation bothers you
- Taptic Engine issues

## Code Examples

```bash
# Read current value
defaults read com.apple.AppleMultitouchTrackpad ActuateDetents

# Enable haptic feedback (default)
defaults write com.apple.AppleMultitouchTrackpad ActuateDetents -bool true

# Disable haptic feedback
defaults write com.apple.AppleMultitouchTrackpad ActuateDetents -bool false

# Delete (reset to default)
defaults delete com.apple.AppleMultitouchTrackpad ActuateDetents
```

## Related Settings

```bash
# Haptic feedback (this setting)
defaults write com.apple.AppleMultitouchTrackpad ActuateDetents -bool true

# Haptic feedback strength (0=Light, 1=Medium, 2=Firm)
defaults write com.apple.AppleMultitouchTrackpad ActuationStrength -int 1

# First click pressure threshold
defaults write com.apple.AppleMultitouchTrackpad FirstClickThreshold -int 1

# Second click (Force Click) pressure threshold
defaults write com.apple.AppleMultitouchTrackpad SecondClickThreshold -int 1

# Tap to click
defaults write com.apple.AppleMultitouchTrackpad Clicking -bool true

# Three finger drag
defaults write com.apple.AppleMultitouchTrackpad TrackpadThreeFingerDrag -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No click feedback | Enable ActuateDetents; check ActuationStrength |
| Force Click not working | Enable this setting; check SecondClickThreshold |
| Click too hard/soft | Adjust FirstClickThreshold (0, 1, or 2) |
| Trackpad unresponsive | Restart Mac; check for hardware issues |
| Setting not applying | Log out and back in after change |

## Compatibility

- **macOS**: 10.11+ (El Capitan) with Force Touch trackpad
- **Hardware**: MacBook 2015+, MacBook Pro 2015+, Magic Trackpad 2
- **Note**: Only applies to Force Touch trackpads with Taptic Engine
- **Note**: Non-Force Touch trackpads ignore this setting
- **Note**: May require logout for changes to take effect
