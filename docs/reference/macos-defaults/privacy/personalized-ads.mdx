# Apple Personalized Advertising

Control Apple's ability to deliver personalized advertisements.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AdLib` |
| **Key** | `allowApplePersonalizedAdvertising` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether Apple can use your information to deliver personalized advertisements in Apple apps and services like the App Store, Apple News, and Stocks. When enabled, Apple uses data about you (demographics, device info, app usage) to show ads that might be more relevant to you. Disabling this doesn't reduce the number of adsâ€”you'll still see ads, but they won't be based on your interests.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Personalized ads** (default) â€” Ads based on your interests |
| `false` | **Generic ads** â€” Random/contextual ads only |

## Our Configuration

**Set to**: `false` â€” Disable personalized advertising for privacy.

### Why Disable?

1. **Privacy** â€” No advertising profile built
2. **No tracking** â€” Advertising Identifier not used
3. **Data minimization** â€” Less information collected
4. **No downside** â€” Ads continue, just generic
5. **Principle** â€” Opt out of unnecessary tracking

### Why Enable?

- Prefer relevant ads over random ones
- Support Apple's ad-supported services
- Don't mind targeted advertising
- Want app suggestions based on interests

## Code Examples

```bash
# Read current value
defaults read com.apple.AdLib allowApplePersonalizedAdvertising

# Disable personalized ads (recommended)
defaults write com.apple.AdLib allowApplePersonalizedAdvertising -bool false

# Enable personalized ads
defaults write com.apple.AdLib allowApplePersonalizedAdvertising -bool true

# Delete (reset to default)
defaults delete com.apple.AdLib allowApplePersonalizedAdvertising
```

## Related Settings

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Still seeing ads | Disabling doesn't remove ads, just personalizes |
| Setting not applying | Log out/in or restart; check System Settings |
| Want to reset ad profile | System Settings > Apple Advertising > Reset Identifier |
| Third-party apps tracking | Use "Ask App Not to Track" in iOS, separate setting |
| Ads everywhere | This only affects Apple's ads, not third-party |

## Compatibility

- **macOS**: 10.14+ (Mojave) for current UI location
- **Note**: Also affects iOS devices signed into same Apple ID
- **Note**: Setting syncs across devices via iCloud
- **Note**: Doesn't affect third-party app advertising
- **Note**: Apple Search Ads may still use some context
