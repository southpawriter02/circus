# askForPasswordDelay

Set the grace period before password is required after screen lock.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.screensaver` |
| **Key** | `askForPasswordDelay` |
| **Type** | Integer |
| **Default Value** | `5` |
| **Our Setting** | `0` |

## Description

Sets the grace period (in seconds) after the screen saver begins or display sleeps before a password is required. During this window, the Mac can be woken without authentication.

## Values

| Value | Delay | Security |
|-------|-------|----------|
| `0` | **Immediate** | Maximum security |
| `5` | 5 seconds (default) | Brief grace period |
| `60` | 1 minute | Low security |
| `300` | 5 minutes | Minimal security |

## Our Configuration

**Set to**: `0` — Maximum security, password required immediately.

## Code Examples

```bash
# Read current value
defaults read com.apple.screensaver askForPasswordDelay

# Require password immediately (maximum security)
defaults write com.apple.screensaver askForPasswordDelay -int 0

# 5 second grace period (Apple default)
defaults write com.apple.screensaver askForPasswordDelay -int 5

# Delete (reset to default)
defaults delete com.apple.screensaver askForPasswordDelay
```

## UI Location

**System Settings** → **Lock Screen** → **Require password after screen saver begins or display is turned off**

The dropdown offers: Immediately, After 5 seconds, After 1 minute, After 5 minutes, etc.

## Related Setting

Must be used with `askForPassword` enabled:

```bash
# Enable password requirement
defaults write com.apple.screensaver askForPassword -int 1

# Set to immediate
defaults write com.apple.screensaver askForPasswordDelay -int 0
```

## Security Recommendation

A delay of 0 is strongly recommended for:
- Laptops used in public spaces
- Shared work environments
- Any Mac with sensitive data

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Delay still active | Ensure askForPassword is also enabled |
| Setting resets | Check MDM profile not overriding |
| Password not required | Enable askForPassword first |

## Compatibility

- **Screensaver**: macOS 10.7+
- **Requires**: askForPassword to be enabled

## Source File

**File**: [`defaults/system/screensaver.sh`](../../../defaults/system/screensaver.sh)

run_defaults "com.apple.screensaver" "askForPasswordDelay" "-int" "0"
