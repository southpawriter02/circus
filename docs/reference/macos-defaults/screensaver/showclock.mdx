# showClock

Show clock on the screen saver.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.screensaver` |
| **Key** | `showClock` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays a clock overlay on the screen saver, showing the current time while the screen saver is active. Useful for quickly checking the time without unlocking.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show clock** — Time displayed on screen saver |
| `false` | **Hide clock** (default) — Screen saver only |

## Our Configuration

**Set to**: `true` — Convenient time display.

## Code Examples

```bash
# Read current value
defaults read com.apple.screensaver showClock

# Show clock on screen saver
defaults write com.apple.screensaver showClock -bool true

# Hide clock
defaults write com.apple.screensaver showClock -bool false

# Delete (reset to default)
defaults delete com.apple.screensaver showClock
```

## UI Location

**System Settings** → **Lock Screen** → **Show large clock**

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Clock not showing | Verify screen saver is running, not display off |
| Clock style wrong | Controlled by system appearance (dark/light) |
| Clock position off | Cannot be repositioned |

## Compatibility

- **Screensaver**: macOS 10.7+
- **Applies to**: Lock screen and screen saver

## Source File

**File**: [`defaults/system/screensaver.sh`](../../../defaults/system/screensaver.sh)

```bash
run_defaults "com.apple.screensaver" "showClock" "-bool" "true"
```
