# Screen Saver Clock Display

Show clock overlay on the screen saver and lock screen.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.screensaver` |
| **Key** | `showClock` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether a clock is displayed on the screen saver and lock screen. When enabled, the current time is shown as an overlay, allowing you to check the time without unlocking your Mac. This is convenient for glancing at the time during meetings or when your Mac is on a visible desk.

## Values

| Value | Display |
|-------|---------|
| `true` | **Show clock** — Time visible on lock screen |
| `false` | **Hide clock** (default) — Screen saver only |

## Our Configuration

**Set to**: `true` — Show clock on screen saver.

### Why Show Clock?

1. **Quick time check** — No unlock needed
2. **Meeting awareness** — Know when to wrap up
3. **Desk visibility** — Clock visible from distance
4. **Practical** — Useful information always available
5. **Professional** — Display serves a purpose

### Why Hide Clock?

- Cleaner aesthetic
- Screen saver art appreciation
- Presentation displays
- Digital signage without time

## Code Examples

```bash
# Read current value
defaults read com.apple.screensaver showClock

# Show clock on screen saver/lock screen
defaults write com.apple.screensaver showClock -bool true

# Hide clock (default)
defaults write com.apple.screensaver showClock -bool false

# Delete (reset to default)
defaults delete com.apple.screensaver showClock
```

## UI Location

**System Settings** → **Lock Screen** → ☑ **Show large clock**

In older macOS:
**System Preferences** → **Security & Privacy** → **General** or **Screen Saver**

## Clock Appearance

The lock screen clock:
- Large, readable font
- Shows time and date
- Follows 12/24-hour system setting
- Positioned prominently
- Semi-transparent or styled to match screen saver

## macOS Sonoma+ Lock Screen

Modern macOS includes additional lock screen customization:
- Widgets below clock
- Photo shuffle backgrounds
- Astronomy/weather widgets

The clock remains central and prominent.

## Screen Saver vs Lock Screen

| State | Clock Shown |
|-------|-------------|
| Screen saver (no password yet) | Yes (if enabled) |
| Lock screen (password required) | Yes (if enabled) |
| Login window (no user logged in) | Yes (if enabled) |

## Interaction with Screen Saver Types

Clock works with most screen savers:
- Aerial (Apple TV scenes) ✓
- Drift ✓
- Photo slideshow ✓
- Classic Mac ✓
- Custom screen savers ✓

Some third-party screen savers may have their own clock.

## Lock Screen Message + Clock

Both can be shown:
```bash
# Show clock
defaults write com.apple.screensaver showClock -bool true

# Show custom message
defaults write com.apple.loginwindow LoginwindowText -string "If found, call..."
```

## Related Settings

```bash
# Require password immediately
defaults write com.apple.screensaver askForPassword -int 1
defaults write com.apple.screensaver askForPasswordDelay -int 0

# Screen saver idle time
defaults write com.apple.screensaver idleTime -int 600

# Login window message
defaults write com.apple.loginwindow LoginwindowText -string "Your message"
```

## Always-On Display (MacBook Pro 14"/16")

On newer MacBook Pro with ProMotion:
- Display never fully off when open
- Clock visible even at minimum brightness
- Uses very little power in this state

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Clock not showing | Verify setting is true |
| Clock too small | Part of screen saver design |
| Wrong time format | Check system 12/24hr setting |

## Compatibility

- **macOS**: 10.7+ (Lion)
- **Note**: Takes effect immediately
