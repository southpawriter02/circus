# Login Window Custom Message

Display a custom message on the login and lock screens.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.loginwindow` |
| **Key** | `LoginwindowText` |
| **Type** | String |
| **Default Value** | Empty |
| **Our Setting** | Contact information |

## Description

Displays a custom text message on the login window and lock screen. This is commonly used to show contact information for returning a lost laptop, internal IT support information, or legal notices. The message appears below the login interface.

## Common Use Cases

| Use Case | Example Message |
|----------|-----------------|
| Lost laptop recovery | "If found: (555) 123-4567" |
| IT contact info | "IT Help: helpdesk@company.com" |
| Legal notice | "Property of Acme Corp" |
| User identification | "This Mac belongs to: John Doe" |
| Asset tracking | "Asset Tag: LAP-2024-0042" |

## Code Examples

```bash
# Read current value
defaults read com.apple.loginwindow LoginwindowText

# Set contact message
defaults write com.apple.loginwindow LoginwindowText -string "If found, call: (555) 123-4567"

# Set email contact
defaults write com.apple.loginwindow LoginwindowText -string "IT Support: helpdesk@company.com"

# Set property notice
defaults write com.apple.loginwindow LoginwindowText -string "Property of Acme Corp - Asset #12345"

# Set multi-line message (use literal newlines)
defaults write com.apple.loginwindow LoginwindowText -string "Property of Acme Corp
Reward for return
IT: (555) 123-4567"

# Clear message
defaults delete com.apple.loginwindow LoginwindowText
```

## Privacy Considerations

Message is visible to anyone who sees the lock screen:
- Don't include sensitive information
- Consider using general email, not personal
- Phone number alternatives: Google Voice, work number

## Multi-Line Messages

Use newlines for formatting:
```bash
defaults write com.apple.loginwindow LoginwindowText -string "Acme Corporation
Asset Tag: 12345
Lost? Contact: lost-property@acme.com"
```

## Character Limits

No hard limit, but:
- Long messages may be truncated
- Keep reasonably short for readability
- Test appearance after setting

## Enterprise Deployment

For many Macs:
- MDM profiles can set this message
- Configuration profiles (com.apple.loginwindow)
- Scripts via management tools

## Combining with Find My

For best lost laptop recovery:
1. Set login window message with contact
2. Enable Find My Mac
3. Enable Guest (for network access by finder)
4. Enable FileVault (protect data)

## Related Settings

```bash
# Disable guest account
defaults write com.apple.loginwindow GuestEnabled -bool false

# Show login as name/password
defaults write com.apple.loginwindow SHOWFULLNAME -bool true

# Disable password hints
defaults write com.apple.loginwindow RetriesUntilHint -int 0
```
