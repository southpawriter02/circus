# Photos Delete After Import

Automatically delete photos from connected devices after importing.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Photos` |
| **Key** | `IPXDefaultsDeleteAfterImport` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Photos automatically deletes photos and videos from connected cameras, SD cards, or iOS devices after successfully importing them. When enabled, source files are removed from the device immediately after import completes. This setting should be used with extreme caution as deleted photos may not be recoverable from the source device.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Delete after import** â€” Remove files from device after import |
| `false` | **Keep on device** (default) â€” Files remain on source device |

## Our Configuration

**Set to**: `false` â€” Keep photos on device as backup.

### Why Keep on Device (Recommended)?

1. **Safety** â€” Photos serve as backup until verified
2. **Recovery** â€” Can re-import if library issues occur
3. **Flexibility** â€” Delete manually when ready
4. **iCloud sync** â€” Wait for cloud backup confirmation
5. **No data loss** â€” Import interruption won't lose photos

### Why Delete After Import?

- Professional photographers with many SD cards
- Need to clear device storage immediately
- Trusted import workflow (verified over time)
- Card reader setup (not valuable device connection)

## Code Examples

```bash
# Read current value
defaults read com.apple.Photos IPXDefaultsDeleteAfterImport

# Keep photos on device (recommended)
defaults write com.apple.Photos IPXDefaultsDeleteAfterImport -bool false

# Delete after import (use with caution!)
defaults write com.apple.Photos IPXDefaultsDeleteAfterImport -bool true

# Delete (reset to default)
defaults delete com.apple.Photos IPXDefaultsDeleteAfterImport
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Photos deleted unexpectedly | Check this setting is disabled |
| Want to delete but keeps photos | Enable setting or check box during import |
| Deleted from device, not in library | Check Recently Deleted in Photos |
| Import failed, photos gone | Use data recovery software on SD card |
| Some photos deleted, others not | Import was interrupted - check both locations |

## Compatibility

- **Photos**: macOS 10.10+ (Yosemite)
- **Note**: Behavior may vary by device type
- **Note**: iOS devices move to Recently Deleted
- **Note**: SD cards delete permanently
- **Warning**: Cannot undo deletion on SD cards/cameras
