# Photos Metadata Overlay

Display photo metadata information while viewing images.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Photos` |
| **Key** | `IPXShowMetadataInfo` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Photos displays a metadata overlay showing information like date taken, location, camera settings, and file details when viewing images. This overlay provides quick context about each photo without needing to open the Info panel. Particularly useful for photographers reviewing their work or organizing photos by technical details.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show metadata** â€” Display info overlay when viewing |
| `false` | **Hide metadata** (default) â€” Clean viewing experience |

## Our Configuration

**Set to**: `true` â€” Show metadata for quick reference.

### Why Show Metadata?

1. **Photography workflow** â€” Review camera settings while culling
2. **Organization** â€” See dates/locations without extra clicks
3. **Learning** â€” Understand which settings produced good results
4. **Quick reference** â€” File size/dimensions at a glance
5. **Location context** â€” Remember where photos were taken

### Why Hide Metadata?

- Casual photo viewing (less distraction)
- Clean, immersive slideshow experience
- Screen recordings or presentations
- When metadata is not relevant

## Code Examples

```bash
# Read current value
defaults read com.apple.Photos IPXShowMetadataInfo

# Show metadata overlay (recommended)
defaults write com.apple.Photos IPXShowMetadataInfo -bool true

# Hide metadata overlay (default)
defaults write com.apple.Photos IPXShowMetadataInfo -bool false

# Delete (reset to default)
defaults delete com.apple.Photos IPXShowMetadataInfo
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No metadata showing | Enable in View > Metadata > Show Metadata |
| Missing GPS data | Photo taken with location services disabled |
| Camera info blank | Screenshot or photo from app (no EXIF) |
| Wrong date/time | Camera clock was incorrect when taken |
| Can't see all details | Use Info panel (âŒ˜I) for full information |

## Compatibility

- **Photos**: macOS 10.13+ (High Sierra)
- **Note**: Metadata shown depends on what's embedded in photo
- **Note**: Screenshots and downloads may have limited metadata
- **Note**: iCloud photos sync metadata across devices
- **Note**: RAW files typically have more detailed metadata
