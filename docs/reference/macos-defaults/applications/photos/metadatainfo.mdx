# Photos Metadata Display

Show photo metadata panel in the viewer.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Photos` |
| **Key** | `IPXShowMetadataInfo` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the Info panel (metadata sidebar) is displayed when viewing photos. The Info panel shows essential details like date, time, location, camera settings, and keywords â€” invaluable for photographers and anyone managing a photo library.

## Values

| Value | Info Panel |
|-------|------------|
| `true` | **Visible** â€” Show metadata sidebar |
| `false` | **Hidden** (default) â€” Clean viewing |

## Our Configuration

**Set to**: `true` â€” Show metadata panel.

### Why Show Metadata?

1. **Photo details** â€” Date, time, location instantly
2. **Camera settings** â€” Aperture, shutter, ISO
3. **Organization** â€” View and add keywords
4. **Editing info** â€” See if photo was edited
5. **Photographer needs** â€” Essential for photo management

### Why Hide Metadata?

- Distraction-free viewing
- Maximum screen for photo
- Casual browsing
- Small screen

## Code Examples

```bash
# Read current value
defaults read com.apple.Photos IPXShowMetadataInfo

# Show metadata panel
defaults write com.apple.Photos IPXShowMetadataInfo -bool true

# Hide metadata panel (default)
defaults write com.apple.Photos IPXShowMetadataInfo -bool false

# Delete (reset to default)
defaults delete com.apple.Photos IPXShowMetadataInfo
```

## Info Panel Sections

### Date & Time
- Capture date and time
- Click to adjust if incorrect
- Time zone information

### Location
- Place name (reverse geocoded)
- GPS coordinates
- Click to view on map
- Can manually add location

### Camera
- Device/camera name
- Lens information
- Focal length
- Exposure settings (Æ’, speed, ISO)

### Keywords
- Assigned keywords
- Click to add new keywords
- Used for searching

### People & Pets
- Detected faces
- Named individuals
- Click to add more

### Caption
- User-added description
- Transferred to EXIF if exported
- Searchable
