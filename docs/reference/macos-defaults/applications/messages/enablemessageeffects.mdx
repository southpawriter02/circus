# Messages Message Effects

Enable or disable automatic playback of bubble and screen effects.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.MobileSMS` |
| **Key** | `EnableMessageEffects` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Messages automatically plays bubble effects (slam, loud, gentle, invisible ink) and full-screen effects (balloons, confetti, lasers, fireworks, hearts, spotlight, echo) when receiving iMessages that include these effects. These effects add personality to conversations but can be distracting or inaccessible for some users.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Effects enabled** (default) â€” Animations play automatically |
| `false` | **Effects disabled** â€” Static messages, no animations |

## Our Configuration

**Set to**: `true` â€” Enable message effects for full iMessage experience.

### Why Enable Effects?

1. **Full experience** â€” See messages as sender intended
2. **Fun interactions** â€” Adds personality to conversations
3. **Social context** â€” Celebrations feel more celebratory
4. **Surprise element** â€” Invisible ink reveals are fun
5. **Apple design** â€” Part of the iMessage experience

### Why Disable Effects?

- Motion sensitivity or vestibular disorders
- Prefer minimal/professional interface
- Reduce visual distractions
- Save battery on older Macs
- Accessibility needs

## Code Examples

```bash
# Read current value
defaults read com.apple.MobileSMS EnableMessageEffects

# Enable message effects (default)
defaults write com.apple.MobileSMS EnableMessageEffects -bool true

# Disable message effects
defaults write com.apple.MobileSMS EnableMessageEffects -bool false

# Delete (reset to default)
defaults delete com.apple.MobileSMS EnableMessageEffects
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Effects not playing | Check setting is enabled |
| Reduce Motion enabled | Disable in System Settings > Accessibility |
| Effects laggy | Normal on older hardware |
| Can't send effects | Long-press/force-touch send button |
| Effect not recognized | Works with iMessage only, not SMS |

## Compatibility

- **Messages**: macOS 10.12+ (Sierra)
- **Note**: Effects only work with iMessage (blue bubbles)
- **Note**: SMS (green bubbles) does not support effects
- **Note**: Recipient must also have compatible device/settings
- **Note**: Some effects require Force Touch trackpad to send
