# Terminal Bold Fonts

Control how Terminal renders bold text.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Terminal` |
| **Key** | `UseBoldFonts` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Terminal.app renders bold text using actual bold font weight or displays it using bright/intense colors instead. When enabled, bold text appears thicker and heavier. When disabled, bold text appears in brighter versions of the standard colors, which is the traditional terminal behavior and may be preferred for certain color schemes.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Use bold fonts** (default) â€” Bold text rendered with heavier weight |
| `false` | **Use bright colors** â€” Bold text shown as bright/intense colors |

## Our Configuration

**Set to**: `true` â€” Use actual bold fonts for clearer distinction.

### Why Enable Bold Fonts?

1. **Clearer distinction** â€” Bold is obviously different from normal
2. **Modern look** â€” Takes advantage of font weights
3. **Readability** â€” Important text stands out
4. **Consistent** â€” Works well with any color scheme
5. **Semantic** â€” Bold means bold, not bright

### Why Disable Bold Fonts?

- Traditional terminal behavior
- Color scheme designed for bright-as-bold
- Font doesn't have good bold weight
- Prefer color distinction over weight
- Retro terminal aesthetic

## Code Examples

```bash
# Read current value
defaults read com.apple.Terminal UseBoldFonts

# Enable bold fonts (recommended)
defaults write com.apple.Terminal UseBoldFonts -bool true

# Disable bold fonts (use bright colors)
defaults write com.apple.Terminal UseBoldFonts -bool false

# Delete (reset to default)
defaults delete com.apple.Terminal UseBoldFonts
```

## Font Considerations

For best bold rendering:
- Use fonts with dedicated bold weights (SF Mono, Menlo, Monaco)
- Monospace fonts designed for terminals
- Avoid fonts without bold variants

| Font | Bold Support |
|------|--------------|
| SF Mono | Excellent |
| Menlo | Excellent |
| Monaco | Good |
| Courier | Fair |
| System fonts | Varies |

## Shell Prompt Impact

If your PS1/prompt uses bold:
```bash
# Bash prompt with bold
PS1='\[\033[1;32m\]\u@\h\[\033[0m\]:\[\033[1;34m\]\w\[\033[0m\]\$ '

# Bold ON:  username appears in bold green
# Bold OFF: username appears in bright green (may look washed out)
```

| Issue | Solution |

| Colors look wrong with bold | May need to adjust color scheme |
| Bold too heavy | Try different font or disable |
| Setting not applying | Close and reopen Terminal |
