# Music Sound Enhancer

Add depth and presence to audio playback with sound enhancement processing.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Music` |
| **Key** | `soundEnhancerEnabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the Music app applies audio processing to add perceived depth, width, and bass enhancement to playback. Sound Enhancer modifies the original audio signal to create a more immersive listening experience, particularly noticeable on smaller speakers. However, audiophiles and professionals often prefer to hear audio as it was originally mastered, making this setting a matter of personal preference.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sound Enhancer on** — Audio processing applied |
| `false` | **Sound Enhancer off** (default) — Original audio preserved |

## Our Configuration

**Set to**: `false` — Preserve original audio quality.

### Why Disable Sound Enhancer?

1. **Fidelity** — Hear music as the artist/producer intended
2. **Mastering** — Professional recordings are already balanced
3. **Quality headphones** — Good equipment doesn't need processing
4. **Listening fatigue** — Enhanced audio can tire ears faster
5. **Audiophile standards** — Purists prefer unprocessed audio

### Why Enable Sound Enhancer?

- Built-in Mac speakers (compensation for small drivers)
- Older or lower-quality headphones
- Prefer a more "exciting" sound signature
- Quiet background listening
- Genre preference (EDM, pop benefit from bass boost)

## Code Examples

```bash
# Read current value
defaults read com.apple.Music soundEnhancerEnabled

# Enable Sound Enhancer
defaults write com.apple.Music soundEnhancerEnabled -bool true

# Disable Sound Enhancer (recommended)
defaults write com.apple.Music soundEnhancerEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.Music soundEnhancerEnabled

# Set intensity level (0.0 to 1.0, when enabled)
defaults write com.apple.Music soundEnhancerLevel -float 0.5
```

## Audio Quality Considerations

```
+-------------------------------------------------------------+
|  Impact on Audio Quality                                     |
+-------------------------------------------------------------+
|                                                              |
|  Original file → [Sound Enhancer] → Playback                |
|                         │                                    |
|                         ▼                                    |
|                  Processing adds:                            |
|                  • Phase shifts                              |
|                  • Harmonic distortion                       |
|                  • Compression artifacts                     |
|                  • Potential clipping                        |
|                                                              |
|  For best quality:                                           |
|  ─────────────────────────────────────────────────────────  |
|  1. Use lossless audio (ALAC/FLAC)                          |
|  2. Keep Sound Enhancer OFF                                 |
|  3. Use quality headphones/speakers                         |
|  4. Use EQ only if needed                                   |
|                                                              |
|  Note: If using with Apple Music Lossless, Sound Enhancer   |
|  processing may reduce the benefit of lossless audio.       |
|                                                              |
+-------------------------------------------------------------+
```

## Related Settings

```bash
# Sound Enhancer (this setting)
defaults write com.apple.Music soundEnhancerEnabled -bool false

# Sound Enhancer intensity
defaults write com.apple.Music soundEnhancerLevel -float 0.5

# Sound Check (volume normalization)
defaults write com.apple.Music normalizeVolume -bool true

# Crossfade songs
defaults write com.apple.Music crossfadeEnabled -bool false

# Lossless audio
defaults write com.apple.Music losslessEnabled -bool true

# Dolby Atmos
defaults write com.apple.Music dolbyAtmosEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Audio sounds distorted | Disable Sound Enhancer or reduce intensity |
| Music sounds muddy | Disable - enhancement may conflict with mastering |
| Want more bass only | Use Equalizer instead (more control) |
| Enhancement too subtle | Increase intensity slider (if enabled) |
| Sounds "fake" or processed | Disable for natural sound |

## Compatibility

- **Music**: macOS 10.15+ (Catalina)
- **Note**: Affects all audio playback in Music
- **Note**: Does not affect AirPlay to other devices (processing is local)
- **Note**: Works with all audio formats
- **Legacy**: Same as iTunes Sound Enhancer
