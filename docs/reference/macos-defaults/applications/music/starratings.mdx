# Music Star Ratings

Show classic 5-star rating controls for songs.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Music` |
| **Key** | `showStarRatings` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays the classic 5-star rating system in the Music library view. Star ratings allow you to rate songs from 1 to 5 stars, which can be used to create smart playlists based on your preferences.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show star ratings** ‚Äî Display rating column and controls |
| `false` | **Hide ratings** (default) ‚Äî Modern love/dislike only |

## Our Configuration

**Set to**: `true` ‚Äî Show classic star ratings.

### Why Show Star Ratings?

1. **Granular ratings** ‚Äî 5 levels vs just love/dislike
2. **Smart playlists** ‚Äî Filter by rating (e.g., 4+ stars)
3. **Classic iTunes** ‚Äî Familiar rating system
4. **Organization** ‚Äî Mark favorites more specifically

### Modern Alternative

## Code Examples

```bash
# Read current value
defaults read com.apple.Music showStarRatings

# Show star ratings (recommended)
defaults write com.apple.Music showStarRatings -bool true

# Hide star ratings (default)
defaults write com.apple.Music showStarRatings -bool false

# Delete (reset to default)
defaults delete com.apple.Music showStarRatings
```

## Rating Songs

With star ratings enabled:
1. Hover over song in list
2. Click star position (1-5)
3. Rating is saved immediately

Or right-click song ‚Üí **Rate** ‚Üí select rating.

## Smart Playlists with Ratings

Create playlists based on ratings:

1. **File** ‚Üí **New** ‚Üí **Smart Playlist**
2. Rule: "Rating is greater than ‚≠ê‚≠ê‚≠ê‚≠ê"
3. Live-updating playlist of top tracks

Example rules:
- Rating is 5 stars (absolute favorites)
- Rating is greater than 3 stars (good songs)
- Rating is not 0 (any rated songs)

## Syncing Ratings

- Star ratings sync via iCloud Music Library
- Ratings transfer when syncing to devices
- Works alongside love/dislike

## Rating vs Love

| Feature | Star Rating | Love |
|---------|-------------|------|
| Levels | 5 (1-5 stars) | 2 (love/dislike) |
| Affects recommendations | No | Yes |
| Smart playlists | Yes | Yes |
| Visible in library | When enabled | Always |

You can use both systems simultaneously.

## Compatibility

- **macOS**: All versions with Music app
- **Legacy**: Carried over from iTunes
- **Note**: Restart Music for changes
