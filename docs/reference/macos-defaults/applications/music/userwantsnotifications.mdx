# Music Notifications

Control whether Music shows notifications when tracks change.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Music` |
| **Key** | `userWantsNotifications` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether the Music app shows macOS notifications when the currently playing song changes. Each track change triggers a banner notification showing the song name, artist, and album art.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show notifications** (default) â€” Banner on each song |
| `false` | **No notifications** â€” Silent track changes |

## Our Configuration

**Set to**: `false` â€” Reduce notification noise.

### Why Disable?

1. **Notification overload** â€” Every 3-4 minutes during playback
2. **Focus interruption** â€” Banners distract from work
3. **Already visible** â€” Music widget/player shows current song
4. **Known music** â€” You likely know what playlist you're playing

### When to Enable

- Discovering new music (radio, curated playlists)
- Shuffled playlists where you want to know song names
- Apple Music Radio listening

## Code Examples

```bash
# Read current value
defaults read com.apple.Music userWantsNotifications

# Disable song change notifications (recommended)
defaults write com.apple.Music userWantsNotifications -bool false

# Enable song change notifications
defaults write com.apple.Music userWantsNotifications -bool true

# Delete (reset to default)
defaults delete com.apple.Music userWantsNotifications

# Restart Music to apply
killall Music
```

## Alternative: Now Playing Widget

Instead of notifications, use the Now Playing widget:
1. Click Notification Center icon (top-right of menu bar)
2. Edit Widgets
3. Add "Now Playing" widget
4. Always visible without notifications

## Control Center Now Playing

Also available in Control Center:
1. Click Control Center icon in menu bar
2. See currently playing track
3. Playback controls available

## Complete Quiet Music Setup

```bash
# Minimal notifications
defaults write com.apple.Music userWantsNotifications -bool false
defaults write com.apple.Music automaticDownloadsEnabled -bool false
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Notifications still appearing | Restart Music: `killall Music` |
| Want notifications sometimes | Control via Notification Center settings |
| No Now Playing widget | Edit Widgets in Notification Center |

## Compatibility

- **System notifications**: Respects Focus/Do Not Disturb
