# Music Sound Enhancer

Apply audio processing to add depth and presence.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Music` |
| **Key** | `soundEnhancerEnabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the Music app applies digital signal processing to enhance audio playback. The Sound Enhancer adds perceived depth and widens the stereo image, making music sound more spacious and immersive through speakers or headphones. However, it modifies the original audio and may not be desired by audiophiles seeking accurate reproduction.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sound Enhancer on** â€” Audio processing applied |
| `false` | **Sound Enhancer off** (default) â€” Original audio preserved |

## Our Configuration

**Set to**: `false` â€” Disable Sound Enhancer for accurate playback.

### Why Disable Sound Enhancer?

1. **Audio fidelity** â€” Preserves original mix
2. **Audiophile preference** â€” No coloration
3. **Lossless audio** â€” Don't process hi-res files
4. **Professional use** â€” Reference listening
5. **Headphone accuracy** â€” Natural soundstage

### Why Enable Sound Enhancer?

- Built-in speakers benefit most
- Makes music feel more immersive
- Laptop speakers sound fuller
- Casual listening enhancement
- Party/ambient background music

| **Phase manipulation** | Creates spacious effect |

## Code Examples

```bash
# Read current value
defaults read com.apple.Music soundEnhancerEnabled

# Enable Sound Enhancer
defaults write com.apple.Music soundEnhancerEnabled -bool true

# Disable Sound Enhancer (default, recommended for quality)
defaults write com.apple.Music soundEnhancerEnabled -bool false

# Set enhancement intensity (0.0 to 1.0)
defaults write com.apple.Music soundEnhancerIntensity -float 0.5

# Delete (reset to default)
defaults delete com.apple.Music soundEnhancerEnabled
```

## Sound Enhancer vs Other Features

| Feature | Purpose | Modifies Audio |
|---------|---------|----------------|
| **Sound Enhancer** | Add spatial depth | Yes (DSP) |
| **Equalizer** | Frequency adjustment | Yes (EQ) |
| **Sound Check** | Volume normalization | Yes (gain) |
| **Dolby Atmos** | Spatial audio | Renders mix |
| **Lossless** | Quality format | No |
