# Zoom Mirror Video

Control whether your self-view video is mirrored.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `us.zoom.xos` |
| **Key** | `ZMMMirrorVideo` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether your self-view video (what you see of yourself) is horizontally flipped like a mirror. When enabled, your video appears as if you're looking in a mirror - raise your right hand and the image raises the hand on your right side. This feels natural but note that other participants always see the non-mirrored (true) version of you.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Mirror enabled** (default) â€” Self-view flipped like mirror |
| `false` | **Mirror disabled** â€” Self-view matches what others see |

## Our Configuration

**Set to**: `true` â€” Mirror self-view for natural interaction.

### Why Enable Mirror (Default)?

1. **Natural feel** â€” Like looking in a mirror
2. **Intuitive gestures** â€” Point right, see right
3. **Comfort** â€” Familiar self-view experience
4. **Standard behavior** â€” How most video apps work
5. **Hair/makeup check** â€” Works as expected

### Why Disable Mirror?

- Presenting slides with text
- Screen sharing with annotations
- Reading text on whiteboards
- Want WYSIWYG self-view
- Holding up documents to camera

## Code Examples

```bash
# Read current value
defaults read us.zoom.xos ZMMMirrorVideo

# Enable mirror (natural self-view, default)
defaults write us.zoom.xos ZMMMirrorVideo -bool true

# Disable mirror (WYSIWYG self-view)
defaults write us.zoom.xos ZMMMirrorVideo -bool false

# Delete (reset to default)
defaults delete us.zoom.xos ZMMMirrorVideo
```

## Virtual Background and Mirroring

| Background Type | Mirror Behavior |
|-----------------|-----------------|
| No background | Normal mirroring |
| Virtual image | Background not mirrored |
| Virtual video | Background not mirrored |
| Blur | Normal mirroring |

Text on virtual backgrounds appears correctly to others regardless of mirror setting.

## Screen Sharing with Video

When screen sharing:
- Shared screen: Not mirrored (others see correctly)
- Your video thumbnail: Follows mirror setting
- Others see shared content correctly

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Text appears backwards | Disable mirror temporarily |
| Self-view feels wrong | Enable mirror (default) |
| Setting not saving | Restart Zoom |
| Others see backwards text | Check what you're sharing (self-view is correct for others) |

## Recording Behavior

| Recording Type | What's Recorded |
|----------------|-----------------|
| Cloud recording | True (non-mirrored) video |
| Local recording | True (non-mirrored) video |
| Gallery view | True orientation |
| Speaker view | True orientation |

Recordings always show the true, non-mirrored video.

## Compatibility

- **macOS**: 10.10+ (Yosemite)
- **Zoom**: All versions
- **Note**: Change takes effect immediately
- **Note**: Per-meeting changes possible via meeting controls

## Quick Toggle During Meeting

During a meeting:
1. Click the arrow next to **Stop Video**
2. Select **Video Settings**
3. Toggle **Mirror my video**

Changes apply immediately without restarting video.
