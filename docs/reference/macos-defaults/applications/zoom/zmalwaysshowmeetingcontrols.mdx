# Zoom Always Show Meeting Controls

Keep the meeting toolbar visible at all times.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `us.zoom.xos` |
| **Key** | `ZMAlwaysShowMeetingControls` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

By default, Zoom auto-hides the meeting control toolbar after a few seconds of inactivity. This setting keeps the toolbar always visible, providing constant access to mute, video, screen share, and other controls.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Always visible** â€” Toolbar stays on screen |
| `false` | **Auto-hide** (default) â€” Toolbar fades after inactivity |

**Set to**: `true` â€” Keep controls always accessible.

### Why Always Show?

1. **Quick access** â€” No hunting for hidden controls
2. **Faster muting** â€” Click instead of move-then-click
3. **Visibility** â€” See current mute/video status
4. **Fewer mistakes** â€” Controls always where expected
5. **Accessibility** â€” Better for screen magnifier users

### Why Auto-Hide?

Some prefer auto-hide for:
- Maximum video viewing area
- Cleaner presenter view
- Less visual distraction

## Code Examples

```bash
# Always show controls (recommended)
defaults write us.zoom.xos ZMAlwaysShowMeetingControls -bool true

# Auto-hide controls (Zoom default)
defaults write us.zoom.xos ZMAlwaysShowMeetingControls -bool false

# Restart Zoom
killall zoom.us
```
