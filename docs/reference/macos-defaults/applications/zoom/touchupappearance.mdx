# Zoom Touch Up Appearance

Control the beauty filter for video calls.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `us.zoom.xos` |
| **Key** | `ZMEnableTouchUpAppearance` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Zoom applies a "Touch Up My Appearance" filter to your video feed during meetings. When enabled, this feature smooths skin texture, reduces blemishes, and applies subtle softening effects similar to beauty filters in photo apps. While it can improve appearance, it processes your video in real-time and may look artificial at higher settings.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Touch up enabled** â€” Beauty filter applied |
| `false` | **Touch up disabled** (default) â€” Natural video |

## Our Configuration

**Set to**: `false` â€” Disable touch up for natural video.

### Why Disable Touch Up?

1. **Authenticity** â€” Natural appearance in meetings
2. **Professional** â€” Appropriate for work calls
3. **Performance** â€” Less CPU/GPU processing
4. **Battery** â€” Saves power on laptops
5. **Bandwidth** â€” No processing artifacts

### Why Enable Touch Up?

- Soften appearance for video calls
- Reduce visibility of tiredness
- Consistent look across calls
- Confidence boost
- Poor lighting compensation

## What Touch Up Does

| Effect | Processing |
|--------|------------|
| **Skin smoothing** | Reduces pores and texture |
| **Blemish reduction** | Hides spots and marks |
| **Soft focus** | Slight blur effect |
| **Even skin tone** | Subtle color correction |
| **Under-eye reduction** | Lightens dark circles (at higher levels) |

## Code Examples

```bash
# Read current value
defaults read us.zoom.xos ZMEnableTouchUpAppearance

# Disable touch up (natural video)
defaults write us.zoom.xos ZMEnableTouchUpAppearance -bool false

# Enable touch up
defaults write us.zoom.xos ZMEnableTouchUpAppearance -bool true

# Delete (reset to default)
defaults delete us.zoom.xos ZMEnableTouchUpAppearance
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Touch up not visible | Check slider intensity |
| Video looks artificial | Reduce intensity |
| Performance issues | Disable or reduce |
| Battery draining fast | Disable during long calls |
