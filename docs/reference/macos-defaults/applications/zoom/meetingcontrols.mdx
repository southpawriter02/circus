# Zoom Always Show Meeting Controls

Keep meeting controls toolbar visible at all times.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `us.zoom.xos` |
| **Key** | `ZMAlwaysShowMeetingControls` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the meeting control toolbar (mute, video, share, participants, etc.) remains visible at all times during Zoom meetings. By default, Zoom auto-hides controls after a few seconds to maximize the video view.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Always visible** â€” Controls stay on screen |
| `false` | **Auto-hide** (default) â€” Controls fade after inactivity |

## Our Configuration

**Set to**: `true` â€” Always show controls.

### Why Always Show?

1. **Quick access** â€” Mute/unmute instantly
2. **No fumbling** â€” Controls always there
3. **Professionalism** â€” No delay finding buttons
4. **Screen sharing** â€” Easy to stop share
5. **Participants** â€” See attendee count

### Why Auto-Hide?

- Larger video view
- Less visual clutter
- Presentation mode
- Recording aesthetics

## Code Examples

```bash
# Read current value
defaults read us.zoom.xos ZMAlwaysShowMeetingControls

# Always show controls (recommended)
defaults write us.zoom.xos ZMAlwaysShowMeetingControls -bool true

# Auto-hide controls (default)
defaults write us.zoom.xos ZMAlwaysShowMeetingControls -bool false

# Delete (reset to default)
defaults delete us.zoom.xos ZMAlwaysShowMeetingControls
```

## Recording Considerations

During recording:
- Hidden controls = cleaner look
- Visible controls appear in recording
- Cloud recordings exclude controls
- Local recordings include everything visible

## Presentation Mode

When presenting:
- You may want controls hidden for audience
- But available for you on presenter screen
- Use separate presenter display settings
