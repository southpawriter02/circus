# Pages Show Invisible Characters

Display invisible formatting characters like spaces, tabs, and line breaks.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSWPShowInvisibles` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Pages displays invisible formatting characters such as spaces, tabs, paragraph marks, line breaks, and section breaks. Showing invisibles helps identify formatting issues, find extra spaces, and understand document structure. This is especially useful for troubleshooting layout problems or preparing documents for publishing.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show invisibles** — Display formatting characters |
| `false` | **Hide invisibles** (default) — Clean document view |

## Our Configuration

**Set to**: `false` — Hide invisibles for clean viewing.

### Why Hide Invisibles (Default)?

1. **Clean view** — See document as readers will see it
2. **Less clutter** — Symbols can distract from content
3. **Reading mode** — Better for reviewing text
4. **Print preview** — Matches printed output
5. **Default behavior** — What most users expect

### Why Show Invisibles?

- Troubleshooting formatting issues
- Finding double spaces or extra returns
- Understanding document structure
- Preparing for publishing (clean formatting)
- Teaching proper document formatting

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSWPShowInvisibles

# Hide invisibles (default)
defaults write com.apple.iWork.Pages TSWPShowInvisibles -bool false

# Show invisibles
defaults write com.apple.iWork.Pages TSWPShowInvisibles -bool true

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSWPShowInvisibles
```

```
+-------------------------------------------------------------+

+-------------------------------------------------------------+

|  Find trailing spaces:                                       |

|  Replace: "¶"                                                |
|  (Use Insert > Special Characters for ¶)                    |

```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't see spacing problems | Enable Show Invisibles |
| Too many symbols cluttering view | Disable Show Invisibles |
| Want to print without symbols | Invisibles never print |
| Symbols appearing in exported PDF | Invisibles don't export (display only) |
| Different symbols than expected | Symbols may vary by Pages version |

## Compatibility

- **Pages**: macOS 10.13+ (High Sierra)
- **Note**: Invisibles only show in edit view
- **Note**: Never printed or exported
- **Note**: Toggle applies immediately
- **Note**: Setting persists across sessions
