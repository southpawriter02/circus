# Pages Show Comments

Control whether the comments pane is displayed in Apple Pages documents.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSShowsComments` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the Comments & Changes pane is displayed in Pages documents. When enabled, comments and tracked changes appear in a sidebar, making it easy to review feedback and collaborate on documents. This is particularly useful for document review workflows, editing collaborations, and tracking revisions. Comments can be added by anyone with edit access and persist until deleted.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show comments pane** â€” Comments visible in sidebar |
| `false` | **Hide comments pane** (default) â€” Comments hidden until opened |

## Our Configuration

**Set to**: `true` â€” Show comments for collaborative editing.

### Why Show Comments?

1. **Collaboration** â€” See all feedback at a glance
2. **Review workflow** â€” Essential for document editing
3. **Track discussions** â€” Follow comment threads
4. **No missed feedback** â€” Comments always visible
5. **Team coordination** â€” Central place for input

### Why Hide Comments?

- **Focus on writing** â€” Distraction-free composing
- **More space** â€” Full width for document
- **Final review** â€” When comments are resolved
- **Personal documents** â€” No collaboration needed
- **Presentation** â€” Cleaner view when showing document

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSShowsComments

# Show comments pane
defaults write com.apple.iWork.Pages TSShowsComments -bool true

# Hide comments pane (default)
defaults write com.apple.iWork.Pages TSShowsComments -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSShowsComments
```

## Related Settings

```bash
# Show comments pane (this setting)
defaults write com.apple.iWork.Pages TSShowsComments -bool true

# Show invisibles (formatting marks)
defaults write com.apple.iWork.Pages TSWPShowInvisibles -bool false

# Show ruler
defaults write com.apple.iWork.Pages TSShowsRuler -bool true

# Show word count
defaults write com.apple.iWork.Pages TSWPShowWordCount -bool true

# Default zoom
defaults write com.apple.iWork.Pages TSDefaultZoom -int 100

# Auto-correction indicators
defaults write com.apple.iWork.Pages TSShowsAutoCorrection -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Comments not showing | View > Show Comments or click comment indicator |
| Can't add comments | Check if document is locked or read-only |
| Comments not syncing | Verify iCloud connection and sign-in |
| Old comments not visible | Scroll through comments pane |
| Can't delete comment | Only comment author or document owner can delete |
| Comments lost after export | Some formats don't preserve comments |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Pages
- **Note**: Same key exists in Numbers and Keynote
- **Note**: Comments sync via iCloud when sharing
- **Note**: Restart Pages for terminal changes to apply
- **Note**: Comments preserved in Pages and Word formats
