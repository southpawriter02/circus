# Pages Object Alignment Guides

Show alignment guides when moving objects.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSShowsObjectEdgeGuides` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Pages displays yellow alignment guides when moving or resizing objects on a page. These guides appear dynamically to show when objects are aligned with page margins, other objects' edges, or centers. This visual feedback helps create professional, well-aligned documents without manually calculating positions.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show guides** (default) â€” Alignment guides visible |
| `false` | **Hide guides** â€” No alignment assistance |

## Our Configuration

**Set to**: `true` â€” Show alignment guides for professional layouts.

### Why Enable Alignment Guides?

1. **Precision** â€” Pixel-perfect alignment
2. **Consistency** â€” Objects line up properly
3. **Speed** â€” No manual measurements
4. **Professional** â€” Clean document layouts
5. **Visual feedback** â€” See alignment in real-time

### Why Disable Alignment Guides?

- Experienced users who don't need hints
- Free-form artistic layouts
- Performance on slow systems
- Prefer coordinate-based positioning
- Distracting while working

## What Guides Show

| Guide Type | Indicates |
|------------|-----------|
| **Edge guides** | Objects' edges aligned |
| **Center guides** | Objects' centers aligned |
| **Page margin** | Object at page margin |
| **Page center** | Object at horizontal/vertical center |
| **Spacing** | Equal distance between objects |

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSShowsObjectEdgeGuides

# Show alignment guides (default, recommended)
defaults write com.apple.iWork.Pages TSShowsObjectEdgeGuides -bool true

# Hide alignment guides
defaults write com.apple.iWork.Pages TSShowsObjectEdgeGuides -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSShowsObjectEdgeGuides
```
