# Pages Show Size and Position When Moving

Display object dimensions and coordinates while dragging elements in Pages.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSShowSizeAndPositionWhenMoving` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Pages displays a tooltip showing an object's size (width Ã— height) and position (X, Y coordinates) while dragging it across the document. This real-time feedback enables precise placement of images, shapes, text boxes, and other objects without needing to manually enter dimensions. The tooltip appears near the cursor during drag operations and updates continuously as you move the object.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show size/position** (default) â€” Tooltip displays during drag |
| `false` | **Hide size/position** â€” No tooltip during drag |

## Our Configuration

**Set to**: `true` â€” Show dimensions for precise layout.

### Why Show Size/Position?

1. **Precise placement** â€” Know exact coordinates
2. **Consistent sizing** â€” Match object dimensions
3. **Professional layouts** â€” Accurate document design
4. **Efficient workflow** â€” No separate Format panel needed
5. **Real-time feedback** â€” See changes as you drag

### Why Hide Size/Position?

- **Cleaner view** â€” Less on-screen clutter
- **Casual documents** â€” Precision not needed
- **Experienced users** â€” Use Format panel instead
- **Small objects** â€” Tooltip may obscure content

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSShowSizeAndPositionWhenMoving

# Show size/position tooltip (default)
defaults write com.apple.iWork.Pages TSShowSizeAndPositionWhenMoving -bool true

# Hide size/position tooltip
defaults write com.apple.iWork.Pages TSShowSizeAndPositionWhenMoving -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSShowSizeAndPositionWhenMoving
```

## Related Settings

```bash
# Show size/position when moving (this setting)
defaults write com.apple.iWork.Pages TSShowSizeAndPositionWhenMoving -bool true

# Show object center guides
defaults write com.apple.iWork.Pages TSShowsObjectCenterGuides -bool true

# Show object edge guides
defaults write com.apple.iWork.Pages TSShowsObjectEdgeGuides -bool true

# Show ruler
defaults write com.apple.iWork.Pages TSShowsRuler -bool true

# Default zoom
defaults write com.apple.iWork.Pages TSDefaultZoom -int 100
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Tooltip not appearing | Check Settings > Rulers > Show size and position |
| Wrong units displayed | Change in Settings > Rulers > Ruler Units |
| Tooltip blocking view | Drag object away from tooltip area |
| Position seems wrong | X/Y measured from top-left of page |
| Scale not showing | Scale only appears during resize, not move |
| Setting not persisting | Quit Pages before changing via defaults |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Pages
- **Note**: Same key exists in Numbers and Keynote
- **Note**: Units follow document ruler settings
- **Note**: Restart Pages for terminal changes to apply
- **Note**: Works with all object types

## Source File

**File**: [`defaults/applications/pages.sh`](../../../../../defaults/applications/pages.sh)

```bash
# --- Show Size and Position When Moving Objects ---
# Key:          TSShowSizeAndPositionWhenMoving
# Domain:       com.apple.iWork.Pages
# Description:  Displays a tooltip showing object size and position while
#               dragging objects in the document.
# Default:      true
# Options:      true  = Show size/position info
#               false = Hide info during drag
# Set to:       true (precise positioning)
# UI Location:  Pages > Settings > Rulers
# Source:       https://support.apple.com/guide/pages/change-settings-tanf9f2cc8d6/mac
run_defaults "com.apple.iWork.Pages" "TSShowSizeAndPositionWhenMoving" "-bool" "true"
```
