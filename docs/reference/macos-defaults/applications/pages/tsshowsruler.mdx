# Pages Show Ruler

Control whether the ruler is displayed in Apple Pages documents.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSShowsRuler` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the horizontal ruler is displayed at the top of the document editing area in Apple Pages. The ruler shows measurements, tab stops, indents, and margins. It's essential for precise document formatting, alignment of text and objects, and setting up professional layouts. The ruler can be toggled on and off without affecting the document's actual formatting.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show ruler** (default) â€” Ruler visible at top of document |
| `false` | **Hide ruler** â€” More vertical editing space |

## Our Configuration

**Set to**: `true` â€” Show ruler for precise formatting.

### Why Show the Ruler?

1. **Precise alignment** â€” Set exact tab stops and indents
2. **Visual margins** â€” See document boundaries
3. **Professional layouts** â€” Create properly formatted documents
4. **Quick adjustments** â€” Drag to change formatting
5. **Industry standard** â€” Expected in word processors

### Why Hide the Ruler?

- **More screen space** â€” Maximize editing area
- **Distraction-free writing** â€” Focus on content
- **Simple documents** â€” No complex formatting needed
- **Touch interface** â€” Using trackpad gestures instead
- **Presentation mode** â€” Cleaner view when demoing

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSShowsRuler

# Show ruler (default)
defaults write com.apple.iWork.Pages TSShowsRuler -bool true

# Hide ruler
defaults write com.apple.iWork.Pages TSShowsRuler -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSShowsRuler
```
