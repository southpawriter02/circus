# Pages Show Auto-Correction

Display visual indicators when text is auto-corrected in Apple Pages.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Pages` |
| **Key** | `TSShowsAutoCorrection` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Pages displays visual feedback when auto-correction modifies your text. When enabled, a brief blue underline appears beneath auto-corrected words, alerting you to changes made automatically. This helps you catch unwanted corrections immediately and revert them if needed. The indicator disappears after a few seconds or when you continue typing.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show auto-correction indicators** (default) â€” Blue underline on corrected text |
| `false` | **Hide auto-correction indicators** â€” Corrections happen silently |

## Our Configuration

**Set to**: `true` â€” Show indicators for awareness.

### Why Show Auto-Correction Indicators?

1. **Awareness** â€” Know when text is modified
2. **Easy revert** â€” Click to undo unwanted corrections
3. **Catch errors** â€” Auto-correct isn't always right
4. **Learn patterns** â€” See what triggers corrections
5. **Quality control** â€” Verify intended text

### Why Hide Indicators?

- **Distraction-free** â€” Cleaner writing experience
- **Confidence** â€” Trust auto-correction fully
- **Fewer interruptions** â€” No visual pauses
- **Minimal UI** â€” Prefer less on-screen feedback

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Pages TSShowsAutoCorrection

# Show auto-correction indicators (default)
defaults write com.apple.iWork.Pages TSShowsAutoCorrection -bool true

# Hide auto-correction indicators
defaults write com.apple.iWork.Pages TSShowsAutoCorrection -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Pages TSShowsAutoCorrection
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Indicators not appearing | Check if auto-correction is enabled |
| Can't revert corrections | Click word while underline visible or use âŒ˜Z |

| Conflicts with custom text | Add exceptions in System Settings > Keyboard |
