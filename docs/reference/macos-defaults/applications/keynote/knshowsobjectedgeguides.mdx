# Keynote Object Edge Alignment Guides

Control whether alignment guides appear when dragging objects near the edges of other objects in Keynote.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `KNShowsObjectEdgeGuides` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls the display of visual alignment guides that appear when you drag an object so that its edge aligns with the edge of another object on your slide. These guides help you create neat, aligned layouts where objects share common boundaries, creating a professional and organized appearance.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show edge guides** (default) — Guides appear when edges align |
| `false` | **Hide edge guides** — No edge alignment assistance |

## Our Configuration

**Set to**: `true` — Show edge alignment guides for clean layouts.

### Why Enable Edge Guides?

1. **Flush layouts** — Objects share clean boundaries
2. **Column alignment** — Easy to create neat columns
3. **Consistent margins** — Maintain uniform spacing
4. **Professional look** — Aligned edges appear intentional
5. **Faster layout** — No manual positioning required

### Why Disable Edge Guides?

- **Overlapping designs** — Intentionally layered objects
- **Reduce distractions** — Fewer guides while editing
- **Artistic layouts** — Non-grid-based compositions

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote KNShowsObjectEdgeGuides

# Enable edge alignment guides (default, recommended)
defaults write com.apple.iWork.Keynote KNShowsObjectEdgeGuides -bool true

# Disable edge alignment guides
defaults write com.apple.iWork.Keynote KNShowsObjectEdgeGuides -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote KNShowsObjectEdgeGuides
```

```bash
# Show size and position when moving
defaults write com.apple.iWork.Keynote KNShowSizeAndPositionWhenMoving -bool true

# Show rulers
defaults write com.apple.iWork.Keynote TSShowsRuler -bool true

# Default zoom level
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 100
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Edge guides not appearing | Check Settings > Rulers > Show guides at object edges |
| Guides appear for some objects only | Guides only show when edges nearly align |
| Want exact edge alignment | Use Format sidebar > Arrange for precise X/Y values |
| Too many guides appearing | Zoom out or disable one guide type |
| Can't align to slide edge | Use View > Show Rulers, then drag guide from ruler |
| Setting not persisting | Restart Keynote after changing via defaults |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Keynote
- **Note**: Same setting exists in Pages and Numbers with TS prefix
- **Note**: Restart Keynote for terminal changes to apply
- **Note**: Guides only appear during active drag operations
- **Note**: Works with all object types (shapes, images, text boxes, charts)
