# Keynote Object Center Alignment Guides

Control whether alignment guides appear when dragging objects near the center of other objects in Keynote.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `KNShowsObjectCenterGuides` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls the display of visual alignment guides that appear when you drag an object near the horizontal or vertical center of another object on your slide. These guides help you precisely align objects without manual measurement, ensuring professional-looking presentations with consistent layouts.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show center guides** (default) — Guides appear when objects align |
| `false` | **Hide center guides** — No visual alignment assistance |

## Our Configuration

**Set to**: `true` — Show alignment guides for precise positioning.

### Why Enable Center Guides?

1. **Professional layouts** — Precisely aligned objects look polished
2. **Faster editing** — No manual measurement needed
3. **Consistent spacing** — Easy to center elements
4. **Visual feedback** — Immediate confirmation of alignment
5. **Slide balance** — Helps create visually balanced compositions

### Why Disable Center Guides?

- **Free-form design** — Intentionally asymmetric layouts
- **Reduce visual clutter** — Fewer on-screen elements while editing
- **Artistic placement** — Prefer organic, non-aligned positioning

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote KNShowsObjectCenterGuides

# Enable center alignment guides (default, recommended)
defaults write com.apple.iWork.Keynote KNShowsObjectCenterGuides -bool true

# Disable center alignment guides
defaults write com.apple.iWork.Keynote KNShowsObjectCenterGuides -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote KNShowsObjectCenterGuides
```

```bash
# Show size and position when moving
defaults write com.apple.iWork.Keynote KNShowSizeAndPositionWhenMoving -bool true

# Show rulers
defaults write com.apple.iWork.Keynote TSShowsRuler -bool true

# Default zoom level
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 100
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Guides not appearing | Check Settings > Rulers > Show guides at object center |
| Guides appear for some objects only | Guides only show when objects nearly align |
| Want more precise alignment | Use Format sidebar > Arrange tab for exact positions |
| Guides too sensitive | Zoom out for less frequent triggering |
| Can't align to slide center | Use View > Show Rulers, then drag from ruler |
| Setting not persisting | Restart Keynote after changing via defaults |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Keynote
- **Note**: Same setting exists in Pages (`com.apple.iWork.Pages`) and Numbers (`com.apple.iWork.Numbers`)
- **Note**: Restart Keynote for terminal changes to apply
- **Note**: Guides only appear during active drag operations
- **Note**: Works with all object types (shapes, images, text boxes, charts)
