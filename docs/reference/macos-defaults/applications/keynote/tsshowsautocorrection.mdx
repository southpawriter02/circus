# Keynote Show Auto-Correction

Display visual indicators when text is auto-corrected in Apple Keynote.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `TSShowsAutoCorrection` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Keynote displays visual feedback when auto-correction modifies your text on slides. When enabled, a brief blue underline appears beneath auto-corrected words, alerting you to changes made automatically. This is particularly important in presentations where incorrect auto-corrections could be embarrassing during delivery. The indicator disappears after a few seconds or when you continue typing.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show auto-correction indicators** (default) â€” Blue underline on corrected text |
| `false` | **Hide auto-correction indicators** â€” Corrections happen silently |

## Our Configuration

**Set to**: `true` â€” Show indicators to catch errors before presenting.

### Why Show Indicators?

1. **Catch errors** â€” Prevent embarrassing mistakes
2. **Brand names** â€” Protect proper spelling
3. **Technical terms** â€” Verify specialized vocabulary
4. **Names** â€” Ensure people's names are correct
5. **Review before presenting** â€” Quality check opportunity

### Why Hide Indicators?

- **Distraction-free** â€” Focus on content creation
- **Trust auto-correct** â€” Confident in corrections
- **Clean editing** â€” Less visual noise
- **Quick drafts** â€” Will review later anyway

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote TSShowsAutoCorrection

# Show auto-correction indicators (default)
defaults write com.apple.iWork.Keynote TSShowsAutoCorrection -bool true

# Hide auto-correction indicators
defaults write com.apple.iWork.Keynote TSShowsAutoCorrection -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote TSShowsAutoCorrection
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Indicators not appearing | Check if auto-correction is enabled in settings |
| Correction reverted wrong | Click underlined word for options |
| Keeps correcting brand names | Add to user dictionary in System Settings |
| Setting not persisting | Quit Keynote before changing via defaults |
| Indicators disappear too fast | Cannot adjust duration |
| Conflicts with custom terms | Add exceptions in System Settings > Keyboard |
