# Keynote Exit After Last Slide

Control what happens when a presentation reaches its final slide.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `KNExitAfterLastSlide` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Keynote automatically exits presentation mode after displaying the last slide, or shows a black screen and waits for manual exit. A black screen provides a clean ending that allows the presenter to conclude gracefully, answer questions, or navigate to specific slides. Automatic exit can feel abrupt and expose the editing interface to the audience.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Exit automatically** — Returns to edit view after last slide |
| `false` | **Show black screen** (default) — Wait for presenter to exit |

## Our Configuration

**Set to**: `false` — Show black screen after last slide.

### Why Show Black Screen?

1. **Professional** — Clean ending without exposing editing UI
2. **Control** — Exit when you're ready, not automatically
3. **Navigation** — Can go back to previous slides for questions
4. **Q&A friendly** — Black screen while answering questions
5. **No reveals** — Hidden slides stay hidden

### Why Auto-Exit?

- Presenting in a loop/kiosk setting
- Quick informal presentations
- Using Keynote as a simple slideshow
- Personal preference

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote KNExitAfterLastSlide

# Show black screen after last slide (recommended)
defaults write com.apple.iWork.Keynote KNExitAfterLastSlide -bool false

# Exit automatically after last slide
defaults write com.apple.iWork.Keynote KNExitAfterLastSlide -bool true

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote KNExitAfterLastSlide
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Edit view shown to audience | Set KNExitAfterLastSlide to false |
| Can't go back after last slide | Press ← (arrow key), not Esc |
| Presentation keeps looping | Disable Loop in Document Settings |
| No black screen appears | Check setting is applied correctly |
| Need to re-start presentation | Press Play (⌥⌘P) again |
