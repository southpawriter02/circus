# Keynote Show Comments

Control whether the comments pane is displayed in Apple Keynote presentations.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `TSShowsComments` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the Comments pane is displayed in Keynote presentations. When enabled, comments appear in a sidebar, making it easy to review feedback from collaborators. This is essential for presentation development workflows where multiple people provide input on slides. Comments can be attached to specific slides or objects and include threaded replies.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show comments pane** â€” Comments visible in sidebar |
| `false` | **Hide comments pane** (default) â€” Comments hidden until opened |

## Our Configuration

**Set to**: `true` â€” Show comments for collaborative editing.

### Why Show Comments?

1. **Team collaboration** â€” See all feedback at once
2. **Review workflow** â€” Essential for presentation development
3. **Track discussions** â€” Follow comment threads
4. **No missed feedback** â€” Comments always visible
5. **Efficient editing** â€” Respond without context switching

### Why Hide Comments?

- **Focus on content** â€” Distraction-free editing
- **More space** â€” Full width for slide
- **Final polish** â€” When all comments resolved
- **Individual work** â€” No collaborators involved
- **Presenting** â€” Clean view when showing slides

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote TSShowsComments

# Show comments pane
defaults write com.apple.iWork.Keynote TSShowsComments -bool true

# Hide comments pane (default)
defaults write com.apple.iWork.Keynote TSShowsComments -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote TSShowsComments
```

## Related Settings

```bash
# Show comments pane (this setting)
defaults write com.apple.iWork.Keynote TSShowsComments -bool true

# Show presenter notes
defaults write com.apple.iWork.Keynote ShowPresenterNotes -bool true

# Show rulers
defaults write com.apple.iWork.Keynote TSShowsRuler -bool true

# Default zoom level
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 100

# Auto-correction indicators
defaults write com.apple.iWork.Keynote TSShowsAutoCorrection -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Comments not showing | View > Show Comments or click comment indicator |
| Can't add comments | Check if presentation is locked or read-only |
| Comments not syncing | Verify iCloud connection and sign-in |
| Comment on wrong object | Select object before adding comment |
| Can't delete comment | Only comment author or owner can delete |
| Comments lost in export | Some formats (PPT) may not preserve comments |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Keynote
- **Note**: Same key exists in Pages and Numbers
- **Note**: Comments sync via iCloud when sharing
- **Note**: Restart Keynote for terminal changes to apply
- **Note**: Comments not visible during slideshow
