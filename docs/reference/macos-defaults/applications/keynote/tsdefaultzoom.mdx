# Keynote Default Zoom Level

Set the default zoom level for presentations in Apple Keynote.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `TSDefaultZoom` |
| **Type** | Integer |
| **Default Value** | `100` |
| **Our Setting** | `100` |

## Description

Controls the default zoom level (as a percentage) when opening or creating presentations in Keynote. This affects how large slides appear on screen during editing. A zoom level of 100% shows the slide at its actual presentation size, while higher values magnify for detailed editing and lower values show more of the canvas. This setting applies to the editing view, not the presentation mode.

## Values

| Value | Zoom Level | Use Case |
|-------|------------|----------|
| `50` | 50% | Overview of slide layouts |
| `75` | 75% | Full slide view with context |
| `100` | 100% (default) | Actual presentation size |
| `125` | 125% | Comfortable editing |
| `150` | 150% | Detail work on elements |
| `200` | 200% | Fine-tuning small elements |

## Our Configuration

**Set to**: `100` â€” Actual size for accurate preview.

### Why Use 100%?

1. **WYSIWYG** â€” See exactly what audience will see
2. **Text sizing** â€” Verify readability at real size
3. **Accurate preview** â€” Check element proportions
4. **Consistent editing** â€” Same baseline for all slides
5. **Presentation accuracy** â€” Match final output

### Why Use Other Levels?

**50-75%**: Overview of layout, slide arrangement
**125-150%**: Detail work on small elements
**200%**: Fine-tuning alignment, small text
**Fit in Window**: Automatic sizing for screen

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote TSDefaultZoom

# Set to 100% (actual size)
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 100

# Set to 125% (larger editing)
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 125

# Set to 150% (detail work)
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 150

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote TSDefaultZoom
```

## Related Settings

```bash
# Default zoom level (this setting)
defaults write com.apple.iWork.Keynote TSDefaultZoom -int 100

# Show presenter notes
defaults write com.apple.iWork.Keynote ShowPresenterNotes -bool true

# Show object center guides
defaults write com.apple.iWork.Keynote KNShowsObjectCenterGuides -bool true

# Show object edge guides
defaults write com.apple.iWork.Keynote KNShowsObjectEdgeGuides -bool true

# Exit after last slide
defaults write com.apple.iWork.Keynote KNExitAfterLastSlide -bool false

# Enable remote control
defaults write com.apple.iWork.Keynote KNRemoteEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Zoom not applying | Restart Keynote after changing via defaults |
| Existing files unaffected | Setting applies to newly opened files |
| Zoom resets when reopening | Per-file zoom saved with document |
| Can't find zoom control | Look at bottom of slide canvas area |
| Text blurry at zoom | Use standard zoom levels for clarity |
| Keyboard shortcuts not working | Ensure Keynote window is focused |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Keynote
- **Note**: Same key name used in Pages and Numbers
- **Note**: Each iWork app stores zoom separately
- **Note**: Restart Keynote for terminal changes to apply
- **Note**: Per-document zoom saved with presentation
