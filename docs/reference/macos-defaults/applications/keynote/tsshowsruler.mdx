# Keynote Show Ruler

Control whether rulers are displayed in Apple Keynote slides.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Keynote` |
| **Key** | `TSShowsRuler` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether horizontal and vertical rulers are displayed around the slide canvas in Keynote. The rulers show measurements, help with object alignment, and allow setting tab stops for text boxes. They display the slide dimensions and provide reference points for precise placement of elements. Rulers use the unit system configured in Keynote preferences (inches, centimeters, or points).

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show rulers** â€” Rulers visible around slide canvas |
| `false` | **Hide rulers** (default) â€” More canvas space |

## Our Configuration

**Set to**: `true` â€” Show rulers for precise alignment.

### Why Show Rulers?

1. **Precise placement** â€” Know exact object positions
2. **Consistent layouts** â€” Align elements accurately
3. **Tab stops** â€” Set custom text formatting
4. **Professional design** â€” Pixel-perfect slides
5. **Quick reference** â€” See dimensions at a glance

### Why Hide Rulers?

- **More space** â€” Larger canvas area
- **Cleaner view** â€” Less visual clutter
- **Simple presentations** â€” Precision not needed
- **Content focus** â€” Concentrate on messaging

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Keynote TSShowsRuler

# Show rulers
defaults write com.apple.iWork.Keynote TSShowsRuler -bool true

# Hide rulers (default)
defaults write com.apple.iWork.Keynote TSShowsRuler -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Keynote TSShowsRuler
```
