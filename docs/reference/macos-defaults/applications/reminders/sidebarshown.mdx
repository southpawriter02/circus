# Reminders Sidebar

Show or hide the sidebar with reminder lists.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.reminders` |
| **Key** | `SidebarShown` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the sidebar containing your reminder lists is visible. The sidebar shows all your lists, smart lists (Today, Scheduled, Flagged, All), and provides quick navigation.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sidebar visible** (default) â€” Full navigation |
| `false` | **Sidebar hidden** â€” Focus on single list |

## Our Configuration

**Set to**: `true` â€” Keep sidebar visible for easy navigation.

### Benefits of Visible Sidebar

1. **List switching** â€” Click to change lists
2. **Badge counts** â€” See pending items per list
3. **Smart lists** â€” Quick access to Today, Scheduled, etc.
4. **Tags** â€” Filter by tags
5. **Organization** â€” Visual overview of all reminders

### When to Hide Sidebar

- Small screen / limited space
- Focused work on single list
- Temporary for more viewing area

## Code Examples

```bash
# Show sidebar (recommended)
defaults write com.apple.reminders SidebarShown -bool true

# Hide sidebar
defaults write com.apple.reminders SidebarShown -bool false
```
