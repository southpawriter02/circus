# Mail Conversation View All Mailboxes

Include related messages from all mailboxes in conversations.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.mail` |
| **Key** | `ConversationViewAllMailboxes` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Mail includes related messages from all mailboxes when displaying email conversations. When enabled, viewing a conversation in your Inbox will also show related messages that are in Sent, Archive, or other folders. This provides complete context for email threads without manually searching other folders.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Include all mailboxes** â€” Show related messages from everywhere |
| `false` | **Current mailbox only** (default) â€” Only show messages in current folder |

## Our Configuration

**Set to**: `true` â€” Include related messages for full conversation context.

### Why Include All Mailboxes?

1. **Context** â€” See your replies alongside received messages
2. **Efficiency** â€” No need to switch between folders
3. **Clarity** â€” Understand full thread at a glance
4. **Workflow** â€” Better for reviewing long email chains
5. **Completeness** â€” Archived messages still visible in threads

### Why Limit to Current Mailbox?

- Prefer isolated folder views
- Very large mailboxes (performance)
- Specific organizational workflow
- Want to see only received messages

## Code Examples

```bash
# Read current value
defaults read com.apple.mail ConversationViewAllMailboxes

# Enable (include related messages from all mailboxes)
defaults write com.apple.mail ConversationViewAllMailboxes -bool true

# Disable (current mailbox only)
defaults write com.apple.mail ConversationViewAllMailboxes -bool false

# Delete (reset to default)
defaults delete com.apple.mail ConversationViewAllMailboxes
```

## Related Settings

```bash
# Include related messages (this setting)
defaults write com.apple.mail ConversationViewAllMailboxes -bool true

# Enable conversation view (required for above)
defaults write com.apple.mail ConversationViewEnabled -bool true

# Conversation sort order (true = newest on top)
defaults write com.apple.mail ConversationViewSortDescending -bool true

# Mark all as read when opening conversation
defaults write com.apple.mail ConversationViewMarkAllAsRead -bool false

# Highlight conversations with unread messages
defaults write com.apple.mail HighlightConversationsWithUnread -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Related messages not showing | Enable "Organize by Conversation" first |
| Too many messages in thread | Can't limit; use search to narrow |
| Deleted messages appearing | They're shown from Trash; empty Trash |
| Performance slow | Large mailboxes may be slower; consider archiving |
| Setting not working | Restart Mail after changing |

## Compatibility

- **Mail**: macOS 10.10+ (Yosemite) for full conversation features
- **Note**: Requires "Organize by Conversation" enabled
- **Note**: Works with all account types (iCloud, Gmail, Exchange)
- **Note**: May affect performance with very large mailboxes
- **Note**: Restart Mail after changing setting
