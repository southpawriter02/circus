# Mail Block Remote Content

Block automatic loading of remote images and content in emails for privacy.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.mail` |
| **Key** | `DisableURLLoading` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Mail automatically loads remote content (images, stylesheets, tracking pixels) embedded in emails. When enabled (`true`), Mail blocks all remote content by default, protecting your privacy by preventing senders from knowing when you opened their email or tracking your location via IP address. This is a critical privacy setting as tracking pixels are widely used in marketing emails.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Load remote content** (default) â€” Images and tracking pixels load automatically |
| `true` | **Block remote content** â€” Remote content blocked until you explicitly allow it |

## Our Configuration

**Set to**: `true` â€” Block remote content for maximum privacy.

### Why Block Remote Content?

1. **Privacy** â€” Prevent email open tracking
2. **Location** â€” Hide your IP address from senders
3. **Security** â€” Block potentially malicious content
4. **Bandwidth** â€” Reduce data usage
5. **Speed** â€” Emails load faster without images

### Why Allow Remote Content?

- Need to see images immediately
- Trust all email senders
- Email formatting depends on remote styles
- Convenience over privacy preference

## Code Examples

```bash
# Read current value
defaults read com.apple.mail DisableURLLoading

# Block remote content (recommended for privacy)
defaults write com.apple.mail DisableURLLoading -bool true

# Allow remote content (default)
defaults write com.apple.mail DisableURLLoading -bool false

# Delete (reset to default)
defaults delete com.apple.mail DisableURLLoading
```

## Related Settings

```bash
# Block remote content (this setting)
defaults write com.apple.mail DisableURLLoading -bool true

# Protect Mail Activity (macOS Monterey+, routes through Apple)
defaults write com.apple.mail ProtectMailActivityInICloud -bool true

# Hide IP address via Apple relay
defaults write com.apple.mail HideIPAddress -bool true

# Load remote content only from contacts
defaults write com.apple.mail LoadRemoteContentInMessages -int 1
```
