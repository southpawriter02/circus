# Mail Inline Attachment Viewing

Show email attachments as icons instead of previewing inline.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.mail` |
| **Key** | `DisableInlineAttachmentViewing` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Mail displays supported attachments (images, PDFs) inline within the message body or shows them as downloadable icon attachments. When disabled, images and documents appear embedded in the email. When enabled, all attachments appear as file icons at the bottom.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show as icons** ‚Äî Attachments appear as downloadable file icons |
| `false` | **Show inline** (default) ‚Äî Images and PDFs render within message body |

## Our Configuration

**Set to**: `true` ‚Äî Show attachments as icons for cleaner reading.

### Why Show as Icons?

1. **Faster loading** ‚Äî Don't render large images inline
2. **Better overview** ‚Äî See all attachments at a glance
3. **Privacy** ‚Äî Prevents tracking pixels from loading automatically
4. **Cleaner reading** ‚Äî Text-focused email experience
5. **Intentional viewing** ‚Äî Open attachments when you choose

### When Inline Is Better

- Casual email with photo sharing
- Marketing emails with designed layouts
- Quick visual review of images

## Code Examples

```bash
# Read current value
defaults read com.apple.mail DisableInlineAttachmentViewing

# Show attachments as icons (recommended)
defaults write com.apple.mail DisableInlineAttachmentViewing -bool true

# Show attachments inline (Apple default)
defaults write com.apple.mail DisableInlineAttachmentViewing -bool false

# Delete (reset to default)
defaults delete com.apple.mail DisableInlineAttachmentViewing

# Restart Mail to apply
killall Mail
```

## UI Location

Not available in Mail preferences ‚Äî terminal only.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Still showing inline | Restart Mail: `killall Mail` |
| Some images still inline | HTML emails may embed images directly |
| Can't preview attachments | Double-click icons to open |

## Compatibility

- **Mail**: macOS 10.10+
- **Affects**: Received messages only
- **Note**: Does not affect how recipients see your emails
