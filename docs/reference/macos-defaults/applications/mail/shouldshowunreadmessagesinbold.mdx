# Mail Show Unread Messages in Bold

Control whether unread email messages appear in bold text for visual distinction.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.mail` |
| **Key** | `ShouldShowUnreadMessagesInBold` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether unread messages in Apple Mail's message list are displayed with bold text styling. When enabled, unread emails stand out with bold sender names and subject lines, making it easy to quickly identify new messages. This visual distinction helps users prioritize and manage their inbox more effectively.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Bold unread messages** (default) — Clear visual distinction |
| `false` | **Normal text weight** — Unread messages use standard font |

## Our Configuration

**Set to**: `true` — Show unread messages in bold for quick identification.

### Why Enable Bold?

1. **Quick scanning** — Instantly identify unread messages
2. **Visual hierarchy** — Clear distinction between read/unread
3. **Prioritization** — Focus on what needs attention
4. **Accessibility** — Easier to spot new messages
5. **Standard behavior** — Expected in email clients

### Why Disable Bold?

- Prefer uniform message list appearance
- Use filters/rules for prioritization instead
- Minimal visual styling preference
- Rely solely on unread indicator dot

## Code Examples

```bash
# Read current value
defaults read com.apple.mail ShouldShowUnreadMessagesInBold

# Enable bold for unread (default)
defaults write com.apple.mail ShouldShowUnreadMessagesInBold -bool true

# Disable bold for unread
defaults write com.apple.mail ShouldShowUnreadMessagesInBold -bool false

# Delete (reset to default)
defaults delete com.apple.mail ShouldShowUnreadMessagesInBold
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Bold not appearing | Restart Mail after changing setting |
| Some messages not bold | Check if messages are actually unread |
| Bold looks wrong | Check font settings in Mail > Settings > Fonts |
| Setting not saving | Ensure Mail is quit before changing via defaults |
| Want different indicator | Blue dot cannot be changed; only bold can be toggled |

## Compatibility

- **macOS**: 10.10+ (Yosemite and later)
- **Note**: Setting applies immediately to message list
- **Note**: Works with all account types (iCloud, Gmail, Exchange)
- **Note**: Independent of conversation view settings
- **Note**: Restart Mail if changed via terminal
