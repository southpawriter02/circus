# Safari Continuous Spell Checking

Control whether Safari automatically checks spelling as you type in text fields.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Safari` |
| **Key** | `WebContinuousSpellCheckingEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Safari continuously checks spelling as you type in web form fields, text areas, and editable content. When enabled, misspelled words are underlined with a red dotted line, and you can right-click for spelling suggestions. This helps catch typos in emails, social media posts, forms, and any text input on websites.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Spell checking enabled** (default) â€” Misspellings highlighted |
| `false` | **Spell checking disabled** â€” No spelling indicators |

## Our Configuration

**Set to**: `true` â€” Enable spell checking to catch typos.

### Why Enable Spell Checking?

1. **Catch typos** â€” Immediate feedback while typing
2. **Professional communication** â€” Avoid embarrassing mistakes in emails
3. **Form accuracy** â€” Catch errors in important submissions
4. **Learning tool** â€” Helps with unfamiliar words
5. **Non-intrusive** â€” Just underlines, doesn't auto-correct

### Why Disable Spell Checking?

- Writing in a language not supported by macOS
- Entering technical terms, code, or jargon frequently
- Privacy concerns about text analysis
- Performance on very slow systems
- Personal preference for manual checking

## Code Examples

```bash
# Read current value
defaults read com.apple.Safari WebContinuousSpellCheckingEnabled

# Enable spell checking (default, recommended)
defaults write com.apple.Safari WebContinuousSpellCheckingEnabled -bool true

# Disable spell checking
defaults write com.apple.Safari WebContinuousSpellCheckingEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.Safari WebContinuousSpellCheckingEnabled
```

## Related Settings

```bash
# Continuous spell checking (this setting)
defaults write com.apple.Safari WebContinuousSpellCheckingEnabled -bool true

# Grammar checking alongside spelling
defaults write com.apple.Safari WebGrammarCheckingEnabled -bool true

# Auto-correct spelling
defaults write com.apple.Safari WebAutomaticSpellingCorrectionEnabled -bool false

# Text replacement/substitutions
defaults write com.apple.Safari WebAutomaticTextReplacementEnabled -bool true

# Smart quotes
defaults write com.apple.Safari WebAutomaticQuoteSubstitutionEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Spell check not working | Verify enabled in Edit > Spelling and Grammar |
| Wrong language detected | Set specific language in System Settings > Keyboard |
| Technical terms flagged | Right-click > Learn Spelling to add to dictionary |
| No suggestions appearing | Right-click the underlined word |
| Spell check too slow | Disable for large text areas; re-enable when done |
| Custom words not saved | Check ~/Library/Spelling/ for custom dictionaries |

## Compatibility

- **macOS**: 10.10+ (Yosemite) and later
- **Note**: Uses system spelling dictionaries and preferences
- **Note**: Safari must be quit for terminal changes to apply
- **Note**: Works with all installed system languages
- **Note**: Custom learned words stored per user
