# Safari AutoFill Passwords

Control automatic password filling in Safari.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Safari` |
| **Key** | `AutoFillPasswords` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether Safari automatically fills in saved usernames and passwords on websites. When enabled, Safari uses passwords stored in your iCloud Keychain or local keychain to automatically complete login forms. Disabling this provides additional security by requiring manual password entry or use of a dedicated password manager.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **AutoFill enabled** (default) â€” Passwords fill automatically |
| `false` | **AutoFill disabled** â€” Manual entry required |

## Our Configuration

**Set to**: `false` â€” Disable Safari password AutoFill.

### Why Disable AutoFill Passwords?

1. **Third-party password manager** â€” Use 1Password, Bitwarden, etc.
2. **Security** â€” Prevents shoulder surfing attacks
3. **Shared computers** â€” Don't expose credentials
4. **Phishing protection** â€” Manual entry helps spot fake sites
5. **Credential hygiene** â€” Forces conscious password use

### Why Enable AutoFill Passwords?

- Convenience for personal devices
- iCloud Keychain integration
- Strong password suggestions
- Cross-device sync with Apple ecosystem
- No additional software needed

## Security Implications

| Consideration | Impact |
|---------------|--------|
| **Phishing risk** | AutoFill may fill credentials on lookalike domains |
| **Shoulder surfing** | Others may see auto-completed passwords |
| **Malware** | Some malware targets browser password stores |
| **Physical access** | Anyone with device access can view filled passwords |

### Security Recommendations

```bash
# Disable all AutoFill features for maximum security
defaults write com.apple.Safari AutoFillPasswords -bool false
defaults write com.apple.Safari AutoFillFromAddressBook -bool false
defaults write com.apple.Safari AutoFillCreditCardData -bool false
defaults write com.apple.Safari AutoFillMiscellaneousForms -bool false
```

## Code Examples

```bash
# Read current value
defaults read com.apple.Safari AutoFillPasswords

# Disable password AutoFill (recommended with password manager)
defaults write com.apple.Safari AutoFillPasswords -bool false

# Enable password AutoFill (default)
defaults write com.apple.Safari AutoFillPasswords -bool true

# Delete (reset to default)
defaults delete com.apple.Safari AutoFillPasswords
```

## iCloud Keychain Interaction

| Safari AutoFill | iCloud Keychain | Behavior |
|-----------------|-----------------|----------|
| Enabled | Enabled | Full password sync and AutoFill |
| Disabled | Enabled | Passwords sync but don't AutoFill |
| Enabled | Disabled | Local-only password AutoFill |
| Disabled | Disabled | No password features |
