# Safari Suppress Search Suggestions

Hide search suggestion interface in Safari's Smart Search Field.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Safari` |
| **Key** | `SuppressSearchSuggestions` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Safari displays the search suggestions dropdown interface in the Smart Search Field. This setting works in conjunction with `UniversalSearchEnabled` â€” while that setting controls whether data is sent to Apple, this setting controls whether the suggestions UI is displayed at all. When enabled (set to true), the dropdown with search suggestions, Siri suggestions, and website recommendations is completely hidden.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Show** (default) â€” Display search suggestions dropdown |
| `true` | **Suppress** â€” Hide search suggestions interface |

## Our Configuration

**Set to**: `true` â€” Suppress suggestions for clean, focused browsing.

### Why Suppress?

1. **Focus** â€” No distracting dropdown
2. **Speed** â€” Type and go without scanning
3. **Privacy** â€” Reinforces disabled suggestions
4. **Clean UI** â€” Uncluttered address bar
5. **Intentional** â€” Search what you meant

### Why Show Suggestions?

- Discover related searches
- Quick access to websites
- Spell check queries
- Find forgotten bookmarks
- Exploratory browsing

## Code Examples

```bash
# Read current value
defaults read com.apple.Safari SuppressSearchSuggestions

# Suppress search suggestions UI (recommended)
defaults write com.apple.Safari SuppressSearchSuggestions -bool true

# Show search suggestions UI (default)
defaults write com.apple.Safari SuppressSearchSuggestions -bool false

# Delete (reset to default)
defaults delete com.apple.Safari SuppressSearchSuggestions
```

## UI Location

This setting doesn't have a direct UI toggle. It's typically set via `defaults` command or managed alongside the "Include Safari Suggestions" checkbox.

## Related Settings

```bash
# Suppress suggestion UI (this setting)
defaults write com.apple.Safari SuppressSearchSuggestions -bool true

# Disable data transmission to Apple
defaults write com.apple.Safari UniversalSearchEnabled -bool false

# Both together for complete privacy
defaults write com.apple.Safari UniversalSearchEnabled -bool false
defaults write com.apple.Safari SuppressSearchSuggestions -bool true
```

## Combined Privacy Setup

For complete search privacy:
```bash
# 1. Don't send data to Apple
defaults write com.apple.Safari UniversalSearchEnabled -bool false

# 2. Don't show suggestion UI
defaults write com.apple.Safari SuppressSearchSuggestions -bool true

# 3. Don't preload top hit
defaults write com.apple.Safari PreloadTopHit -bool false
```
