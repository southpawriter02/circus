# Safari Status Bar

Show or hide the status bar at the bottom of Safari windows.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Safari` |
| **Key** | `ShowStatusBar` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Safari displays a status bar at the bottom of browser windows. The status bar shows the destination URL when hovering over links, page loading progress, and other status information. This is valuable for security-conscious users who want to verify link destinations before clicking, and for developers debugging web applications.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show status bar** â€” URL preview visible at bottom |
| `false` | **Hide status bar** (default) â€” Cleaner interface, no URL preview |

## Our Configuration

**Set to**: `true` â€” Show status bar for link verification.

### Why Show Status Bar?

1. **Security** â€” Verify links before clicking
2. **Phishing protection** â€” Spot fake URLs
3. **Debugging** â€” See actual link targets
4. **Transparency** â€” Know where you're going
5. **Professional** â€” Standard for developers

### Why Hide Status Bar?

- Maximized content area
- Cleaner interface
- Trust link destinations
- Use Smart Search field preview instead
- Mobile-like experience

## Code Examples

```bash

# Delete (reset to default)
defaults delete com.apple.Safari ShowStatusBar
```

Restart Safari for changes to take effect.
