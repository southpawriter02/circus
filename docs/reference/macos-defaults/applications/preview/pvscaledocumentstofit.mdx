# Preview Scale Documents To Fit

Control whether documents automatically scale to fit the window.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Preview` |
| **Key** | `PVScaleDocumentsToFit` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Preview automatically scales PDF documents and images to fit within the window when opened. When enabled, the entire page or image is visible without scrolling. When disabled, documents open at their actual pixel size, which may require scrolling for large documents or high-resolution images.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Scale to fit** (default) — Document fits in window |
| `false` | **Actual size** — Show at 100% zoom, may require scrolling |

## Our Configuration

**Set to**: `true` — Scale to fit for convenient document viewing.

### Why Scale To Fit?

1. **Overview** — See entire document immediately
2. **Navigation** — Easy to scan multi-page PDFs
3. **Convenience** — No immediate scrolling needed
4. **Gestures** — Pinch-to-zoom when detail needed
5. **Efficiency** — Faster document review

### Why Use Actual Size?

- Need pixel-accurate viewing (designers, photographers)
- Reviewing image quality at native resolution
- Checking print dimensions
- Detailed inspection work
- Prefer consistent 100% baseline

## Code Examples

```bash
# Read current value
defaults read com.apple.Preview PVScaleDocumentsToFit

# Enable scale to fit (default)
defaults write com.apple.Preview PVScaleDocumentsToFit -bool true

# Disable (show at actual size)
defaults write com.apple.Preview PVScaleDocumentsToFit -bool false

# Delete (reset to default)
defaults delete com.apple.Preview PVScaleDocumentsToFit
```

```bash
# Window size behavior (1=New window, 2=Tabs, 3=Fit to window)

defaults write com.apple.Preview PVSidebarViewModeForNewDocuments -int 1

# Background color (0=Default, 1=White, 2=Black)
defaults write com.apple.Preview PVBackgroundColor -int 0
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Setting not applying | Quit and restart Preview |
| Document still scrolls | Window may be smaller than scaled document |
| Images look blurry | Scaled down images lose detail; zoom in |
| Want actual size just once | Use ⌘0 or View > Actual Size |
| PDF pages different sizes | Each page scales individually |

## Compatibility

- **Preview**: macOS 10.7+ (Lion)
- **Note**: Affects both PDFs and images
- **Note**: Setting persists across sessions
- **Note**: Individual zoom adjustments are temporary
- **Note**: Retina displays may show at 2x pixel density
