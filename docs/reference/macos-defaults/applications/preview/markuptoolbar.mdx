# Preview Markup Toolbar Auto-Show

Automatically display the Markup toolbar when opening documents.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Preview` |
| **Key** | `PVShowMarkupToolbarForNewDocuments` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Preview automatically shows the Markup toolbar when opening images or PDFs. The Markup toolbar provides quick access to annotation tools like shapes, text, signatures, and highlighting. When disabled, you access the toolbar via View menu or âŒ˜â‡§A when needed.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show toolbar automatically** â€” Markup tools always visible |
| `false` | **Hide toolbar** (default) â€” Clean view, show on demand |

## Our Configuration

**Set to**: `false` â€” Show Markup toolbar only when needed.

### Why Hide Markup Toolbar (Default)?

1. **Clean view** â€” Maximum reading space
2. **On-demand** â€” Show only when annotating
3. **Distraction-free** â€” Focus on content
4. **Quick access** â€” âŒ˜â‡§A when needed
5. **Professional** â€” Clean document viewing

### Why Show Markup Toolbar?

- Frequent annotation work
- PDF review/markup workflow
- Form filling regularly
- Image editing tasks
- One-click tool access
