# TextEdit Smart Links

Automatically detect and make URLs clickable in documents.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.TextEdit` |
| **Key** | `SmartLinks` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether TextEdit automatically detects URLs, email addresses, and web links in text and makes them clickable. When enabled, typing `https://example.com` automatically becomes a hyperlink that can be Command-clicked to open in your browser. This feature works in both plain text and rich text modes.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Smart links enabled** (default) — URLs become clickable links |
| `false` | **Smart links disabled** — URLs remain as plain text |

## Our Configuration

**Set to**: `true` — Enable smart links for convenient URL access.

### Why Enable Smart Links?

1. **Convenience** — Open URLs with a single ⌘-click
2. **Email access** — Click emails to open mail client
3. **Documentation** — Quick access to referenced URLs
4. **Notes** — Personal notes with clickable references
5. **Standard behavior** — Users expect links to be clickable

### Why Disable Smart Links?

- Writing about URLs without wanting them clickable
- Documenting code that contains URLs
- Preventing accidental link clicks
- URLs interfere with text selection
- Personal preference for plain text behavior

## Code Examples

```bash
# Read current value
defaults read com.apple.TextEdit SmartLinks

# Enable smart links (default)
defaults write com.apple.TextEdit SmartLinks -bool true

# Disable smart links
defaults write com.apple.TextEdit SmartLinks -bool false

# Delete (reset to default)
defaults delete com.apple.TextEdit SmartLinks
```

## Smart Links vs Data Detectors

```
+-------------------------------------------------------------+
|  Smart Links vs Data Detectors                               |
+-------------------------------------------------------------+
|                                                              |
|  SMART LINKS (this setting):                                 |
|  ─────────────────────────────────────────────────────────  |
|  Detects:                                                    |
|  • URLs (http://, https://, ftp://)                         |
|  • Domain names (www.example.com)                           |
|  • Email addresses (user@domain.com)                        |
|  • File paths (file:///...)                                 |
|                                                              |
|  Actions:                                                    |
|  • ⌘-click to open                                          |
|  • Copy link address                                        |
|  • Add to Reading List                                      |
|                                                              |
|  DATA DETECTORS (separate setting):                          |
|  ─────────────────────────────────────────────────────────  |
|  Detects:                                                    |
|  • Dates ("March 15, 2024")                                 |
|  • Times ("3:00 PM")                                        |

|  • Phone numbers ("(555) 123-4567")                         |
|  • Tracking numbers                                         |
|                                                              |
|  Actions:                                                    |
|  • Create calendar event                                    |
|  • Add to contacts                                          |
|  • Open in Maps                                             |
|  • Show FaceTime options                                    |
|                                                              |
|  Both can be enabled independently                          |
|                                                              |
+-------------------------------------------------------------+
```

## Related Settings

```bash
# Smart links (this setting)
defaults write com.apple.TextEdit SmartLinks -bool true

# Data detectors (dates, addresses, phone numbers)
defaults write com.apple.TextEdit DataDetectors -bool true

# Smart quotes (curly quote conversion)
defaults write com.apple.TextEdit SmartQuotes -bool false

# Smart dashes (en-dash/em-dash conversion)
defaults write com.apple.TextEdit SmartDashes -bool false

# Plain text mode
defaults write com.apple.TextEdit RichText -int 0
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Links not being detected | Check SmartLinks setting is enabled |
| Can't click links | Use ⌘-click, not regular click |
| Link opens wrong app | Check default browser/mail settings |
| Links detected incorrectly | May need space after URL |
| Want to edit link without opening | Right-click and select Edit Link |

## Compatibility

- **TextEdit**: All macOS versions
- **Note**: Works in both plain text and rich text modes
- **Note**: Link styling may differ between modes
- **Note**: Detection happens as you type
- **Note**: Existing text is re-analyzed when setting changes
