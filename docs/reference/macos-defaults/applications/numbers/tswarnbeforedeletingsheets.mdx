# Numbers Warn Before Deleting Sheets

Show confirmation dialog before permanently deleting sheets.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSWarnBeforeDeletingSheets` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Numbers displays a confirmation dialog before deleting sheets from a spreadsheet. This safety feature prevents accidental data loss from unintended deletions. When a sheet is deleted, all data, formulas, charts, and formatting on that sheet are permanently removed. This cannot be undone after the document is saved.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show warning** (default) â€” Confirmation required before deletion |
| `false` | **No warning** â€” Sheets deleted immediately on click |

## Our Configuration

**Set to**: `true` â€” Show warning before deleting sheets.

### Why Show Warning (Recommended)?

1. **Safety** â€” Prevents accidental data loss
2. **Confirmation** â€” Ensures intentional deletion
3. **Information** â€” Shows what will be deleted
4. **Second chance** â€” Opportunity to reconsider
5. **Peace of mind** â€” Know you won't lose data accidentally

### Why Disable Warning?

- Reorganizing many sheets quickly (use caution!)
- Very confident in your actions
- Working with non-critical data
- Personal preference (not recommended)

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSWarnBeforeDeletingSheets

# Show warning before deleting (recommended)
defaults write com.apple.iWork.Numbers TSWarnBeforeDeletingSheets -bool true

# Disable warning (not recommended)
defaults write com.apple.iWork.Numbers TSWarnBeforeDeletingSheets -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSWarnBeforeDeletingSheets
```
