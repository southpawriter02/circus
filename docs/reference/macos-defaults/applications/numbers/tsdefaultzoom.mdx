# Numbers Default Zoom

Set the default zoom level for new spreadsheets in Apple Numbers.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSDefaultZoom` |
| **Type** | Integer |
| **Default Value** | `125` |
| **Our Setting** | `100` |

## Description

Controls the default zoom level percentage for new spreadsheets in Apple Numbers. This setting determines how large or small the spreadsheet content appears when you create a new document or open Numbers. Higher values zoom in (larger content, less visible area), while lower values zoom out (smaller content, more visible area). The zoom level can always be adjusted manually for individual documents.

## Values

| Value | Behavior |
|-------|----------|
| `25` | **25%** — Maximum zoom out, tiny content |
| `50` | **50%** — Half size, overview mode |
| `75` | **75%** — Slightly reduced |
| `100` | **100%** — Actual size |
| `125` | **125%** (default) — Slightly enlarged |
| `150` | **150%** — Enlarged view |
| `200` | **200%** — Double size |
| `400` | **400%** — Maximum zoom in |

## Our Configuration

**Set to**: `100` — Actual size for accurate representation.

### Why Use 100%?

1. **Accuracy** — See content at actual print size
2. **Consistency** — Standard baseline view
3. **Predictability** — What you see matches output
4. **Flexibility** — Easy to zoom from neutral point

### Why Use Higher (125-150%)?

- Easier to read small text
- Working on laptop with smaller display
- Prefer larger cell content
- Vision accommodation needed

### Why Use Lower (75%)?

- Need to see more columns at once
- Large display with plenty of space
- Reviewing overall spreadsheet structure
- Working with wide data tables

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSDefaultZoom

# Set to actual size (100%)
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 100

# Set to default (125%)
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 125

# Set to enlarged (150%)
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 150

# Set to zoomed out (75%)
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 75

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSDefaultZoom
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Zoom not changing | Applies to NEW documents only |
| Old documents unchanged | Individual docs remember their zoom |
| Value rejected | Use valid integer (25-400) |
| Want per-sheet zoom | Set manually; saved per-document |
| Very small/large zoom | Use reasonable values (50-200) |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Numbers
- **Note**: Setting applies to new spreadsheets only
- **Note**: Existing documents keep their saved zoom
- **Note**: Numbers must be quit for defaults to apply
- **Note**: Same domain key as Pages (TSDefaultZoom)
