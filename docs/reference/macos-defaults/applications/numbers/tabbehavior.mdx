# Numbers Tab Key Behavior

Control whether Tab key inserts a tab or moves between cells.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSInsertTabInsteadOfMoving` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls the behavior of the Tab key when editing cells in Numbers. When disabled (default), Tab moves the selection to the next cell, similar to Excel behavior. When enabled, Tab inserts an actual tab character into the cell content, requiring a different method to navigate between cells.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Tab moves to next cell** (default) — Spreadsheet navigation |
| `true` | **Tab inserts tab character** — Text formatting in cell |

## Our Configuration

**Set to**: `false` — Tab moves between cells for spreadsheet workflow.

### Why Tab Should Move (Default)?

1. **Excel compatibility** — Familiar behavior
2. **Data entry** — Fast cell-to-cell navigation
3. **Standard spreadsheet** — Expected behavior
4. **Efficiency** — Quick data input workflow
5. **Muscle memory** — Works like other spreadsheets

### Why Tab Should Insert?

- Text-heavy spreadsheets
- Formatted cell content
- Document-like tables
- Word processor expectations
- Specific formatting needs

## Keyboard Navigation Summary

| Key | Tab Moves (false) | Tab Inserts (true) |
|-----|-------------------|-------------------|
| **Tab** | Next cell | Tab character |
| **Shift+Tab** | Previous cell | Remove indent |
| **Return** | Cell below | New line in cell |
| **⌥Tab** | N/A | Next cell |
| **⌥Return** | N/A | Exit cell |
| **Arrow keys** | Navigate cells | Move cursor in cell |

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSInsertTabInsteadOfMoving

# Tab moves to next cell (default, spreadsheet behavior)
defaults write com.apple.iWork.Numbers TSInsertTabInsteadOfMoving -bool false

# Tab inserts tab character (word processor behavior)
defaults write com.apple.iWork.Numbers TSInsertTabInsteadOfMoving -bool true

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSInsertTabInsteadOfMoving
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Tab not moving cells | Check setting is false |
| Can't insert tab in cell | Enable setting or use ⌥Tab |
| Unexpected behavior | Check if cell is in edit mode |
| Different than Excel | Verify setting matches |
