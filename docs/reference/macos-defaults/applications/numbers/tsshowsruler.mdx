# Numbers Show Ruler

Control whether the ruler is displayed in Apple Numbers spreadsheets.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSShowsRuler` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the ruler is displayed at the top of the Numbers spreadsheet editor. The ruler helps with precise positioning of objects, adjusting column widths, and aligning content. It shows measurements in your preferred unit (inches, centimeters, or points) and displays tab stops and margins for text boxes. While spreadsheets are primarily cell-based, rulers are useful when adding shapes, charts, and text elements to your sheets.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show ruler** (default) â€” Ruler visible at top of sheet |
| `false` | **Hide ruler** â€” More vertical space for content |

## Our Configuration

**Set to**: `true` â€” Keep ruler visible for precise object positioning.

### Why Show Ruler?

1. **Precise positioning** â€” Exact placement of charts and shapes
2. **Consistency** â€” Align objects across multiple sheets
3. **Print accuracy** â€” Know exact dimensions for printed output
4. **Text formatting** â€” Set tab stops in text boxes

### Why Hide Ruler?

- Pure data entry work (no objects)
- Maximize vertical space on small screens
- Prefer cleaner interface
- Rarely add non-cell elements

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSShowsRuler

# Show ruler (default)
defaults write com.apple.iWork.Numbers TSShowsRuler -bool true

# Hide ruler
defaults write com.apple.iWork.Numbers TSShowsRuler -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSShowsRuler
```

```bash
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 100

# Show comments pane
defaults write com.apple.iWork.Numbers TSShowsComments -bool false

# Show auto-correction
defaults write com.apple.iWork.Numbers TSShowsAutoCorrection -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Ruler not appearing | Check View > Show Ruler is enabled |
| Wrong measurement units | Numbers > Settings > Rulers > Ruler Units |
| Setting doesn't persist | Quit Numbers before changing via defaults |
| Ruler cuts off content | Scroll or zoom out to see full sheet |
| Tab stops not showing | Select a text box first |
