# Numbers Object Center Alignment Guides

Control whether alignment guides appear when dragging objects near the center of other objects in Numbers.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSShowsObjectCenterGuides` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls the display of visual alignment guides that appear when you drag an object near the horizontal or vertical center of another object in your Numbers spreadsheet. These guides help you precisely align charts, images, shapes, and other non-table objects, ensuring professional-looking reports and presentations created in Numbers.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show center guides** (default) — Guides appear when objects align |
| `false` | **Hide center guides** — No visual alignment assistance |

## Our Configuration

**Set to**: `true` — Show alignment guides for precise layouts.

### Why Enable Center Guides?

1. **Professional reports** — Centered charts look polished
2. **Quick alignment** — No manual measurement needed
3. **Consistent layouts** — Easy to match object positions
4. **Visual feedback** — Immediate confirmation of alignment
5. **Efficient editing** — Drag-and-snap positioning

### Why Disable Center Guides?

- **Asymmetric designs** — Intentionally off-center layouts
- **Less distraction** — Fewer visual elements while editing
- **Free positioning** — Prefer manual placement

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSShowsObjectCenterGuides

# Enable center alignment guides (default, recommended)
defaults write com.apple.iWork.Numbers TSShowsObjectCenterGuides -bool true

# Disable center alignment guides
defaults write com.apple.iWork.Numbers TSShowsObjectCenterGuides -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSShowsObjectCenterGuides
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Guides not appearing | Check Settings > Rulers > Show guides at object center |
| Guides only for some objects | Works for charts, shapes, images; not table cells |
| Want more precise alignment | Use Format sidebar > Arrange for exact X/Y values |
| Guides too sensitive | Zoom out for less frequent triggering |
| Can't align to sheet center | Drag guide from ruler for fixed reference line |
| Setting not persisting | Restart Numbers after changing via defaults |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Numbers
- **Note**: Same setting exists in Pages and Keynote
- **Note**: Restart Numbers for terminal changes to apply
- **Note**: Guides only appear during active drag operations
- **Note**: Works with charts, shapes, images, and text boxes (not table cells)
