# Numbers Show Comments

Control whether the comments pane is displayed in Apple Numbers spreadsheets.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iWork.Numbers` |
| **Key** | `TSShowsComments` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the Comments & Changes pane is displayed by default when opening spreadsheets in Apple Numbers. Comments allow collaborators to add notes, questions, and feedback to specific cells or ranges without modifying the actual data. When enabled, the comments pane shows all comments and tracked changes in a sidebar, making it easy to review and respond to feedback.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show comments pane** â€” Comments sidebar visible |
| `false` | **Hide comments pane** (default) â€” Cleaner view, access via View menu |

## Our Configuration

**Set to**: `false` â€” Hide comments pane for cleaner default view.

### Why Hide Comments Pane?

1. **More workspace** â€” Full width for spreadsheet
2. **Cleaner view** â€” Focus on data, not discussions
3. **On-demand access** â€” Show when needed via View menu
4. **Standard default** â€” Most work doesn't need comments visible

### Why Show Comments Pane?

- Active collaboration on spreadsheet
- Reviewing shared documents
- Need to respond to feedback
- Tracking changes in team projects

## Code Examples

```bash
# Read current value
defaults read com.apple.iWork.Numbers TSShowsComments

# Show comments pane by default
defaults write com.apple.iWork.Numbers TSShowsComments -bool true

# Hide comments pane (default)
defaults write com.apple.iWork.Numbers TSShowsComments -bool false

# Delete (reset to default)
defaults delete com.apple.iWork.Numbers TSShowsComments
```

## Related Settings

```bash
# Show comments pane (this setting)
defaults write com.apple.iWork.Numbers TSShowsComments -bool false

# Show ruler
defaults write com.apple.iWork.Numbers TSShowsRuler -bool true

# Show auto-correction indicators
defaults write com.apple.iWork.Numbers TSShowsAutoCorrection -bool true

# Show formula warnings
defaults write com.apple.iWork.Numbers TSShowFormulaWarnings -bool true

# Default zoom level
defaults write com.apple.iWork.Numbers TSDefaultZoom -int 100
```

## Compatibility

- **macOS**: 10.14+ (Mojave) with Numbers
- **Note**: Comments sync via iCloud for shared spreadsheets
- **Note**: Setting is app-wide, affects all documents
- **Note**: Same key name used in Pages and Keynote
- **Note**: Restart Numbers for terminal changes to take effect
