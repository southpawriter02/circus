# Chrome Native Print Dialog

Use macOS print dialog instead of Chrome's built-in preview.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.google.Chrome` |
| **Key** | `DisablePrintPreview` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Chrome uses the native macOS print dialog or its own built-in print preview. The native dialog provides better integration with macOS features like printer presets, AirPrint discovery, and system-wide print settings.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Native macOS dialog** — System print interface |
| `false` | **Chrome print preview** (default) — Built-in preview |

## Our Configuration

**Set to**: `true` — Use native macOS dialog.

### Why Native Dialog?

1. **Printer presets** — Saved configurations
2. **AirPrint** — Better discovery
3. **System integration** — Consistent with other apps
4. **PDF options** — Save as PDF, Mail PDF, etc.
5. **Print queue** — Direct access
6. **Familiarity** — Same as other macOS apps

### Why Chrome Preview?

- Google Cloud Print (discontinued)
- Specific Chrome features
- Simplified interface
- Consistent across platforms

## Code Examples

```bash
# Read current value
defaults read com.google.Chrome DisablePrintPreview

# Use native macOS print dialog
defaults write com.google.Chrome DisablePrintPreview -bool true

# Use Chrome print preview (default)
defaults write com.google.Chrome DisablePrintPreview -bool false

# Delete (reset to default)
defaults delete com.google.Chrome DisablePrintPreview
```

## UI Location

No Chrome UI setting — only via defaults command.

## Print to PDF

Both dialogs support PDF:
- Chrome preview: "Save as PDF" destination
- Native dialog: PDF dropdown in lower-left

## Expanded Print Options

For detailed options in native dialog:
```bash
# Show expanded print dialog
defaults write com.google.Chrome PMPrintingExpandedStateForPrint2 -bool true
```

## Keyboard Shortcut

| Shortcut | Action |
|----------|--------|
| ⌘P | Print (opens configured dialog) |
| ⌘⇧P | Chrome print preview (when native disabled) |

## Print Presets

With native dialog, create presets:
1. Configure desired settings
2. Click Presets dropdown
3. Select "Save Current Settings as Preset"
4. Name your preset
5. Reuse across all apps

## Related Settings

```bash
# Confirm before quitting
defaults write com.google.Chrome confirmBeforeQuitting -bool true

# Disable swipe navigation
defaults write com.google.Chrome AppleEnableSwipeNavigateWithScrolls -bool false
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Still see Chrome dialog | Check setting, restart Chrome |
| Printer not found | Check macOS Printers settings |
| Print job stuck | Open Print Queue in Dock |

## Compatibility

- **Chrome**: All versions
- **macOS**: 10.13+
- **Note**: No restart required

## Source File

**File**: [`defaults/applications/chrome.sh`](../../../../defaults/applications/chrome.sh)

```bash
run_defaults "com.google.Chrome" "DisablePrintPreview" "-bool" "true"
```
