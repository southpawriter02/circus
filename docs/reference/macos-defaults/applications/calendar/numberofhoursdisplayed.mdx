# Number of Hours Displayed

Control how many hours are visible at once in Calendar day and week views.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `number of hours displayed` |
| **Type** | Integer |
| **Default Value** | `12` |
| **Our Setting** | `12` |

## Description

Controls the number of hours visible at once in the day and week views without scrolling. A higher value shows more hours simultaneously but makes each hour slot smaller, while a lower value shows fewer hours but with larger, more readable time slots. This setting helps optimize the calendar view for your typical workday duration and screen size.

## Values

| Value | Behavior |
|-------|----------|
| `6` | Very detailed view — 6 hours visible |
| `8` | Standard workday — 8 hours visible |
| `10` | Extended workday — 10 hours visible |
| `12` | **Balanced** (default) — 12 hours visible |
| `14` | More coverage — 14 hours visible |
| `16` | Most of day — 16 hours visible |
| `18` | Nearly full day — 18 hours visible |
| `24` | Full day — All 24 hours visible |

## Our Configuration

**Set to**: `12` — Balanced view covering typical waking hours.

### Why 12 Hours (Default)?

1. **Coverage** — Shows most of typical waking/working hours
2. **Readability** — Time slots are reasonably sized
3. **Balance** — Minimal scrolling for normal schedules
4. **Standard** — Expected default behavior
5. **Versatile** — Works for most screen sizes

### Alternative Configurations

| Use Case | Value | Reason |
|----------|-------|--------|
| Small laptop | `8` | Larger, easier-to-click slots |
| Standard workday | `10` | Focus on 9-5 hours |
| Long days | `14-16` | See early and late hours |
| Shift work | `24` | Full day view |
| Quick overview | `18-24` | Maximum coverage |

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "number of hours displayed"

# Set to 12 hours (balanced - default)
defaults write com.apple.iCal "number of hours displayed" -int 12

# Set to 8 hours (focused workday)
defaults write com.apple.iCal "number of hours displayed" -int 8

# Set to 24 hours (full day view)
defaults write com.apple.iCal "number of hours displayed" -int 24

# Delete (reset to default)
defaults delete com.apple.iCal "number of hours displayed"
```

Restart Calendar for changes to take effect.

defaults write com.apple.iCal "first minute of work hours" -int 540

# Day end time (minutes from midnight)
defaults write com.apple.iCal "last minute of work hours" -int 1080

# First day of week
defaults write com.apple.iCal "first day of week" -int 1

# Scroll by weeks
defaults write com.apple.iCal "scroll by weeks in week view" -bool true
```

|  +-------+-----------------------------------------------+  |
|  | 9 AM  |                                               |  |
|  |       |   EXPANDED VIEW                               |  |

|  |       |                                               |  |
|  | 11 AM |                                               |  |
|  |       |                                               |  |
|  | ...   |                                               |  |
|  |       |                                               |  |

|  | 6 PM  |                                               |  |

|  | 7 PM  | Compressed (outside work hours)               |  |

|                                                              |
+-------------------------------------------------------------+
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Time slots too small | Reduce hours displayed value |
| Need to scroll too much | Increase hours displayed value |
| Can't see early/late events | Increase to 18-24 hours |
| Setting not applying | Restart Calendar app |
| Value not in dropdown | Only preset values (6,8,10,12,14,16,18,24) available |

## Compatibility

- **macOS**: 10.8+ (Mountain Lion and later)
- **Note**: Only affects day and week views
- **Note**: Month view is unaffected
- **Note**: Requires Calendar restart
- **Note**: Works with work hours settings
