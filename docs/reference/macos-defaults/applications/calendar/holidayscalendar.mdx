# Calendar Holidays Calendar

Control whether the regional Holidays calendar is displayed.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `add holiday calendar` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Calendar displays the regional Holidays calendar, which shows public holidays based on your system's region setting. The holidays calendar helps with planning by showing national holidays, bank holidays, and other regional observances. Events appear as all-day events and are automatically updated by Apple.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show holidays** (default) â€” Regional holidays visible in calendar |
| `false` | **Hide holidays** â€” No holiday calendar shown |

## Our Configuration

**Set to**: `true` â€” Show holidays for planning purposes.

### Why Show Holidays?

1. **Planning** â€” Know when businesses/offices are closed
2. **Scheduling** â€” Avoid booking meetings on holidays
3. **Awareness** â€” Don't miss important dates
4. **Coordination** â€” Helpful for international teams
5. **Time off** â€” Plan vacations around holidays

### Why Hide Holidays?

- Calendar looks cluttered with holiday events
- Holidays not relevant to your work
- Using a different holiday calendar (subscribed)
- Prefer minimal calendar view
- Holidays differ from your region

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "add holiday calendar"

# Show holidays calendar (default)
defaults write com.apple.iCal "add holiday calendar" -bool true

# Hide holidays calendar
defaults write com.apple.iCal "add holiday calendar" -bool false

# Delete (reset to default)
defaults delete com.apple.iCal "add holiday calendar"
```

