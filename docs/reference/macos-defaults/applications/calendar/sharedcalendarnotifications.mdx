# Shared Calendar Notifications

Control whether shared calendar invitations and updates appear in Notification Center.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `SharedCalendarNotificationsDisabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether Calendar displays notifications in Notification Center when you receive shared calendar invitations, acceptances, declines, or updates from other users. This setting is particularly relevant for collaborative environments where multiple people share calendars. Disabling these notifications can reduce interruptions but may cause you to miss important scheduling updates from colleagues.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Notifications enabled** (default) — Show shared calendar notifications |
| `true` | **Notifications disabled** — Hide shared calendar notifications |

## Our Configuration

**Set to**: `false` — Keep shared calendar notifications enabled.

### Why Keep Enabled (false)?

1. **Timely Response** — Respond to invitations promptly
2. **Stay Synchronized** — Know about schedule changes immediately
3. **Professional Courtesy** — Don't miss meeting requests
4. **Collaboration** — Stay connected with team calendars
5. **Default Behavior** — Expected standard behavior

### When to Disable

- Subscribed to many noisy shared calendars
- Need focus time without interruptions
- Prefer to check calendar inbox manually
- Calendar notifications cause distraction

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal SharedCalendarNotificationsDisabled

# Keep notifications enabled (default, recommended)
defaults write com.apple.iCal SharedCalendarNotificationsDisabled -bool false

# Disable shared calendar notifications
defaults write com.apple.iCal SharedCalendarNotificationsDisabled -bool true

# Delete (reset to default)
defaults delete com.apple.iCal SharedCalendarNotificationsDisabled
```

Restart Calendar for changes to take effect.

## Related Settings

```bash
# Shared calendar notifications (this setting)
defaults write com.apple.iCal SharedCalendarNotificationsDisabled -bool false

# Default event alert time
defaults write com.apple.iCal DefaultEventAlert -int -900

# Default all-day event alert
defaults write com.apple.iCal DefaultAllDayEventAlert -int -39600

# Time zone support
defaults write com.apple.iCal "TimeZone support enabled" -bool true

# Show week numbers
defaults write com.apple.iCal "Show Week Numbers" -bool true
```

## Notification Types vs Alert Types

```
+-------------------------------------------------------------+
|  Shared Notifications vs Event Alerts                        |
+-------------------------------------------------------------+
|                                                              |
|  SHARED CALENDAR NOTIFICATIONS (This Setting):               |
|  ─────────────────────────────────────────────────────────  |
|  • Invitations from others                                   |
|  • Updates to shared events                                  |
|  • Response notifications (accepted/declined)                |
|  • Calendar sharing invites                                  |
|                                                              |
|  EVENT ALERTS (DefaultEventAlert):                           |
|  ─────────────────────────────────────────────────────────  |
|  • Reminders before your events start                        |
|  • "Meeting starts in 15 minutes"                            |
|  • Based on time before event                                |
|                                                              |
|  These are INDEPENDENT settings!                             |
|  You can disable shared notifications while keeping          |
|  event reminder alerts enabled.                              |
|                                                              |
+-------------------------------------------------------------+
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Still getting notifications | Restart Calendar app |
| Missing invitations | Check Calendar > View > Show Inbox |
| No notifications even when enabled | Check System Settings > Notifications > Calendar |
| Notifications too noisy | Consider disabling, check inbox manually |
| Can't find setting in UI | Calendar > Settings > Alerts tab |

## Compatibility

- **macOS**: 10.8+ (Mountain Lion and later)
- **Note**: Requires Calendar restart for changes
- **Note**: Works with iCloud and Exchange calendars
- **Note**: CalDAV calendar support varies
- **Note**: System Notification settings may also affect display
