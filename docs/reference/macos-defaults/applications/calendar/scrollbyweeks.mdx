# Calendar Scroll By Weeks

Control how week view scrolling behavior works in Calendar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `scroll by weeks in week view` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether scrolling in Calendar's week view moves by individual days or by entire weeks. When enabled, each scroll action jumps a full 7 days, making it faster to navigate through the calendar. When disabled (default), scrolling moves by single days, providing finer control but requiring more scrolling to move through weeks.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Scroll by week** — Each scroll jumps 7 days |
| `false` | **Scroll by day** (default) — Each scroll moves one day |

## Our Configuration

**Set to**: `true` — Scroll by week for easier navigation.

### Why Scroll by Week?

1. **Efficiency** — Faster navigation through calendar
2. **Consistency** — Week boundaries stay aligned
3. **Planning** — See complete weeks at a glance
4. **Workflow** — Match how most people think about time
5. **Less effort** — Fewer scroll/swipe actions needed

### Why Scroll by Day?

- Need fine-grained day-by-day navigation
- Reviewing events across week boundaries
- Personal preference for gradual scrolling
- Working with day-specific workflows

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "scroll by weeks in week view"

# Enable scroll by week (recommended)
defaults write com.apple.iCal "scroll by weeks in week view" -bool true

# Disable (scroll by day, default)
defaults write com.apple.iCal "scroll by weeks in week view" -bool false

# Delete (reset to default)
defaults delete com.apple.iCal "scroll by weeks in week view"
```

## Related Settings

```bash
# Scroll by weeks (this setting)
defaults write com.apple.iCal "scroll by weeks in week view" -bool true

# First day of week (0=Sun, 1=Mon, etc.)
defaults write com.apple.iCal "first day of week" -int 1

# Day start time (minutes from midnight)
defaults write com.apple.iCal "first minute of work hours" -int 540

# Day end time (minutes from midnight)
defaults write com.apple.iCal "last minute of work hours" -int 1080

# Number of hours displayed at once
defaults write com.apple.iCal "number of hours displayed" -int 12

# Show week numbers
defaults write com.apple.iCal "Show Week Numbers" -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Setting not applying | Restart Calendar app |
| Trackpad scrolling too fast | Adjust trackpad sensitivity in System Settings |
| Want different behavior for day view | This only affects week view |
| Arrow keys still move by day | Keyboard nav unchanged; setting affects scroll/swipe |
| Month view unaffected | This setting is week view specific |

## Compatibility

- **Calendar**: macOS 10.8+ (Mountain Lion)
- **Note**: Setting only affects week view scrolling
- **Note**: Keyboard navigation (arrow keys) not affected
- **Note**: Month and day views have separate behavior
- **Note**: Syncs via iCloud preferences
