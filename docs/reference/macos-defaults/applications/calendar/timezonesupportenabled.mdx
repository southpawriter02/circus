# Calendar Time Zone Support

Enable time zone support for scheduling events across different time zones.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `TimeZone support enabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether Calendar allows events to be associated with specific time zones. When enabled, you can set events to occur in different time zones, and Calendar will automatically adjust display times based on your current location. This is essential for remote work, international scheduling, and travel planning. Without time zone support, all events are displayed in the system's local time zone.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Time zone support enabled** â€” Events can have specific time zones |
| `false` | **Time zone support disabled** (default) â€” All events use local time |

## Our Configuration

**Set to**: `true` â€” Enable time zone support for accurate scheduling.

### Why Enable Time Zone Support?

1. **Remote work** â€” Schedule with team members globally
2. **Travel planning** â€” Events adjust when you change zones
3. **Meeting accuracy** â€” Never confuse time zones again
4. **Flight bookings** â€” Enter times in departure/arrival zones
5. **International business** â€” Essential for global operations

### Why Disable Time Zone Support?

- **Local-only use** â€” Never schedule across time zones
- **Simplicity** â€” Fewer options when creating events
- **Avoid confusion** â€” Some users find it confusing
- **Single location** â€” Always work from same time zone

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "TimeZone support enabled"

# Enable time zone support (recommended for remote work)
defaults write com.apple.iCal "TimeZone support enabled" -bool true

# Disable time zone support (default)
defaults write com.apple.iCal "TimeZone support enabled" -bool false

# Delete (reset to default)
defaults delete com.apple.iCal "TimeZone support enabled"
```

## Related Settings

```bash
# Time zone support (this setting)
defaults write com.apple.iCal "TimeZone support enabled" -bool true

# Show week numbers
defaults write com.apple.iCal "Show Week Numbers" -bool true

# First day of week
defaults write com.apple.iCal "first day of week" -int 1

# Default event alert
defaults write com.apple.iCal DefaultEventAlert -int -900

# Time to Leave alerts
defaults write com.apple.iCal TimeToLeaveEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Time zone menu not showing | Enable in Calendar > Settings > Advanced |
| Events showing wrong time | Check event's assigned time zone |
| Time zone selector grayed out | Restart Calendar after enabling |
| Invited events wrong time | Check sender's time zone setting |
| All-day events spanning days | All-day events can behave differently across zones |
| DST changes confusing | Calendar should handle DST automatically |

## Compatibility

- **macOS**: 10.9+ (Mavericks) with Calendar
- **Note**: Works with iCloud, Google, and Exchange calendars
- **Note**: Syncs setting with iOS Calendar via iCloud
- **Note**: Some calendar services handle time zones differently
- **Note**: Restart Calendar for changes to take effect

## Source File

**File**: [`defaults/applications/calendar.sh`](../../../../../defaults/applications/calendar.sh)

```bash
# --- Time Zone Support ---
# Key:          "TimeZone support enabled"
# Domain:       com.apple.iCal
# Description:  Enables time zone support, allowing events to be set to specific
#               time zones. Useful for scheduling across time zones.
# Default:      false
# Options:      true  = Enable time zone support
#               false = Disable time zone support
# Set to:       true (important for remote work)
# UI Location:  Calendar > Settings > Advanced > Turn on time zone support
# Source:       https://support.apple.com/guide/calendar/change-advanced-preferences-icl1027/mac
run_defaults "com.apple.iCal" "TimeZone support enabled" "-bool" "true"
```
