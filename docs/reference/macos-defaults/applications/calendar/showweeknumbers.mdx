# Calendar Week Numbers

Display ISO week numbers in calendar views.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `Show Week Numbers` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Displays week numbers (1-52/53) in the calendar sidebar and month view. Week numbers follow ISO 8601 standard, making it easy to refer to specific weeks in scheduling conversations, project planning, and international business communications.

## Values

| Value | Display |
|-------|---------|
| `true` | **Show week numbers** — Numbers visible |
| `false` | **Hide week numbers** (default) — Cleaner view |

## Our Configuration

**Set to**: `true` — Show week numbers.

### Why Show Week Numbers?

1. **Project planning** — "Deadline is week 12"
2. **International business** — Standard reference
3. **Sprints** — Agile teams use week numbers
4. **Quick reference** — Know where you are in the year
5. **ISO 8601** — International standard

### Why Hide Week Numbers?

- Cleaner visual appearance
- Personal calendars don't need them
- Takes up sidebar space

## ISO 8601 Week Numbering

| Rule | Description |
|------|-------------|
| Week 1 | First week with Thursday in new year |
| Weeks start | Monday |
| Weeks per year | 52 or 53 |
| Notation | "W01" through "W52/53" |

Example:
- 2024-12-30 (Monday) = Week 1 of 2025
- 2025-01-01 (Wednesday) = Week 1 of 2025

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "Show Week Numbers"

# Show week numbers (recommended)
defaults write com.apple.iCal "Show Week Numbers" -bool true

# Hide week numbers (default)
defaults write com.apple.iCal "Show Week Numbers" -bool false

# Delete (reset to default)
defaults delete com.apple.iCal "Show Week Numbers"
```

## UI Location

**Calendar** → **Settings** (⌘,) → **General** → ☑ **Show week numbers**

## Using Week Numbers in Communication

Professional scheduling:
```
✓ "Let's schedule the review for W12"
✓ "Deliverable due end of week 45"
✓ "I'm on vacation W30-W31"

✗ "Let's meet the week of March 17th" (less precise)
```

## Cross-Year Considerations

Week numbering at year boundaries:
```
2024:
  Week 52: Dec 23-29
  Week 1:  Dec 30 - Jan 5 (belongs to 2025!)

2025:
  Week 1:  Dec 30 (2024) - Jan 5 (2025)
  Week 2:  Jan 6-12
```

Some days in December belong to Week 1 of next year.

## Related Settings

```bash
# First day of week (should be Monday for ISO weeks)
defaults write com.apple.iCal "first day of week" -int 1

# Show events in year view
defaults write com.apple.iCal "Show year view" -bool true
```

## Week Number in Notification Center

The Date widget in Notification Center can also show week numbers:
- Click date in menu bar
- Calendar widget appears
- May show week numbers based on Calendar setting

## Calculating Week Numbers

Command line check:
```bash
# Current week number
date +%V

# Week number for specific date
date -j -f "%Y-%m-%d" "2025-03-15" +%V
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Numbers not showing | Restart Calendar |
| Wrong week number | Check first day of week setting |
| Numbers differ from colleagues | Verify ISO 8601 compliance |
