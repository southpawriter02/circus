# Calendar Birthdays Display

Show birthdays from Contacts in Calendar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `display birthdays calendar` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Calendar displays a special "Birthdays" calendar that automatically shows birthday events for all contacts in your Contacts app that have birthday information. This creates automatic all-day events on each contact's birthday without needing to manually create calendar entries.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show Birthdays** (default) â€” Birthday events appear in calendar |
| `false` | **Hide Birthdays** â€” No birthday events shown |

## Our Configuration

**Set to**: `true` â€” Show birthday calendar for contact awareness.

### Why Show Birthdays?

1. **Never forget** â€” Automatic reminders
2. **Contact awareness** â€” See upcoming birthdays
3. **No manual entry** â€” Auto-populated from Contacts
4. **iCloud sync** â€” Works across devices
5. **Age display** â€” Shows how old person is turning

### Why Hide Birthdays?

- Calendar too cluttered
- Use separate birthday app
- Privacy concerns
- Don't track birthdays
- Work calendar only

## Birthday Event Details

Birthday events include:
- Contact's name
- Age they're turning (if birth year known)
- All-day event format
- Link back to contact card
- Repeats annually

## Code Examples

```bash
# Read current value (note: key has spaces)
defaults read com.apple.iCal "display birthdays calendar"

# Show birthdays calendar (default)
defaults write com.apple.iCal "display birthdays calendar" -bool true

# Hide birthdays calendar
defaults write com.apple.iCal "display birthdays calendar" -bool false

# Delete (reset to default)
defaults delete com.apple.iCal "display birthdays calendar"
```

## Setting Birthday Alerts

```bash
# Default alert for all-day events (affects birthdays)
# Value in seconds before event (negative = seconds before midnight)
# -39600 = 9:00 AM on day of event

defaults write com.apple.iCal DefaultAllDayEventAlert -int -39600
```

Common values:
| Value | Alert Time |
|-------|------------|
| `-39600` | 9:00 AM day of |
| `-64800` | Day before at 6:00 PM |
| `-86400` | Day before at midnight |
| `0` | No alert |

```bash
defaults write com.apple.iCal "first day of week" -int 1
```

## Contacts Birthday Field

To add birthdays to contacts:
1. Open **Contacts**
2. Edit contact
3. Add **Birthday** field
4. Enter date (with or without year)

With year: Shows "Turning X" in Calendar
Without year: Shows "Birthday" without age

## iCloud Sync Behavior

| Contacts Location | Birthdays Appear |
|-------------------|------------------|
| iCloud | Yes, synced everywhere |
| On My Mac | Yes, local only |
| Google Contacts | Yes, if synced |
| Exchange | Yes, if birthday field synced |

## Privacy Considerations

The Birthdays calendar:
- Only visible on your devices
- Not shared with shared calendars
- Read-only (can't edit in Calendar)
- Updates automatically from Contacts

- **Note**: Takes effect immediately
