# Default All-Day Event Alert

Set the default alert time for all-day events in Calendar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `DefaultAllDayEventAlert` |
| **Type** | Integer |
| **Default Value** | `-39600` (9:00 AM day of event) |
| **Our Setting** | `-39600` |

## Description

Controls the default reminder time for all-day events like birthdays, holidays, and full-day appointments. Unlike timed events that use minutes-before values, all-day events use seconds from midnight on the event day. The default value of `-39600` represents 9:00 AM on the day of the event (9 hours × 3600 seconds = 32400, but negative indicates same day). This setting provides a morning reminder for events that don't have a specific start time.

## Values

The value is specified in seconds, where negative values indicate time on the day of the event:

| Value | Alert Time | Description |
|-------|------------|-------------|
| `0` | **None** | No default alert |
| `-32400` | 9:00 AM | Morning reminder (9 hours from midnight) |
| `-39600` | 9:00 AM (default) | Morning reminder |
| `-25200` | 5:00 PM previous day | Day-before evening |
| `-86400` | Day before (9 AM) | 24 hours before event day |
| `-172800` | 2 days before | Early warning |

|  ─────────────────────────────────────────────────────────  |

|                                                              |

## Our Configuration

**Set to**: `-39600` — Alert at 9:00 AM on the day of the event.

### Why 9:00 AM Alert?

1. **Morning Planning** — Start day knowing about all-day events
2. **Not Too Early** — Respects typical wake-up times
3. **Actionable** — Time to prepare if needed
4. **Default Behavior** — Expected standard setting
5. **Professional** — Reminder before work day gets busy

### Alternative Configurations

| Use Case | Value | Description |
|----------|-------|-------------|
| Early bird | `-32400` | 9 AM alert |
| Day before | `-86400` | Evening before event |
| No reminders | `0` | Disable default alerts |
| Afternoon | `-21600` | Noon reminder |

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal DefaultAllDayEventAlert

# Set to 9:00 AM on day of event (default)
defaults write com.apple.iCal DefaultAllDayEventAlert -int -39600

# Set to day before at 9:00 AM
defaults write com.apple.iCal DefaultAllDayEventAlert -int -118800

# Disable default all-day alerts
defaults write com.apple.iCal DefaultAllDayEventAlert -int 0

# Delete (reset to default)
defaults delete com.apple.iCal DefaultAllDayEventAlert
```

Restart Calendar for changes to take effect.

## Related Settings

```bash
# All-day event alerts (this setting)
defaults write com.apple.iCal DefaultAllDayEventAlert -int -39600

# Timed event alerts
defaults write com.apple.iCal DefaultEventAlert -int -900

# Show birthdays calendar (uses all-day alerts)
defaults write com.apple.iCal "display birthdays calendar" -bool true

# Shared calendar notifications
defaults write com.apple.iCal SharedCalendarNotificationsDisabled -bool false
```

## Event Types Using All-Day Alerts

| Event Type | Uses This Setting | Notes |
|------------|-------------------|-------|
| All-day events you create | Yes | Default alert applies |
| Birthdays from Contacts | Yes | Calendar's built-in birthdays |
| Holidays | Varies | Some come with predefined alerts |
| Anniversaries | Yes | Full-day events |
| Vacation days | Yes | Multi-day events |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Alert not appearing | Check System Notifications for Calendar |
| Wrong time | Verify value in seconds calculation |
| Alerts too early/late | Adjust value using formula above |
| Existing events unaffected | Setting only affects new events |
| Birthdays not alerting | Check "display birthdays calendar" setting |

## Compatibility

- **macOS**: 10.8+ (Mountain Lion and later)
- **Note**: Only affects new all-day events
- **Note**: Existing events retain their alert settings
- **Note**: Individual events can override this default
- **Note**: Requires Calendar restart for changes
