# Calendar Week Start Day

Set which day of the week calendars start on.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iCal` |
| **Key** | `first day of week` |
| **Type** | Integer |
| **Default Value** | `0` (Sunday) |
| **Our Setting** | `1` (Monday) |

## Description

Controls which day of the week appears as the first column in Calendar's week and month views. This is a cultural preference — some regions start weeks on Sunday, others on Monday. Setting this correctly makes calendar navigation feel natural.

## Values

| Value | Day | Common Regions |
|-------|-----|----------------|
| `0` | **Sunday** (default) | US, Canada, Japan |
| `1` | **Monday** | Europe, UK, Australia |
| `2` | Tuesday | (rare) |
| `3` | Wednesday | (rare) |
| `4` | Thursday | (rare) |
| `5` | Friday | (rare) |
| `6` | **Saturday** | Some Middle Eastern countries |

## Our Configuration

**Set to**: `1` — Monday start.

### Why Monday Start?

1. **ISO 8601** — International standard starts Monday
2. **European convention** — Most of Europe uses Monday
3. **Workweek logic** — Work Mon-Fri, weekend Sat-Sun together
4. **Week numbers** — ISO weeks start Monday
5. **Global business** — Many international standards

### Why Sunday Start?

- US cultural convention
- Some religious calendars
- Personal preference

## Code Examples

```bash
# Read current value
defaults read com.apple.iCal "first day of week"

# Set to Monday (recommended)
defaults write com.apple.iCal "first day of week" -int 1

# Set to Sunday (US default)
defaults write com.apple.iCal "first day of week" -int 0

# Set to Saturday
defaults write com.apple.iCal "first day of week" -int 6

# Delete (reset to default)
defaults delete com.apple.iCal "first day of week"
```

## UI Location

**Calendar** → **Settings** (⌘,) → **General** → **Start week on**

Dropdown menu with all days available.

## Week Numbers

Combined with week start setting:
```bash
# Show week numbers
defaults write com.apple.iCal "Show Week Numbers" -bool true
```

ISO week numbers assume Monday start for consistency.

## Effects on Other Apps

This setting primarily affects Apple Calendar. Other apps may:
- Follow system locale settings
- Have their own week start settings
- Not be affected

## System Locale vs App Setting

macOS language/region settings suggest a week start, but Calendar's setting overrides:

```
System: Region → United States → Suggests Sunday
Calendar: Setting → Monday → Uses Monday
```

App setting takes precedence.

## iCloud Sync

This preference syncs via iCloud to:
- Other Macs (same Apple ID)
- iOS devices
- iPadOS devices

All your calendars stay consistent.

## Troubleshooting

| Wrong day showing | Check value is correct (0-6) |
| Synced devices differ | Check iCloud Calendar sync |
