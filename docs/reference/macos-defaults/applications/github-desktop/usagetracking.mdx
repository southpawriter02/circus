# GitHub Desktop Usage Tracking

Opt out of anonymous usage data collection.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.github.GitHubClient` |
| **Key** | `optOutOfUsageTracking` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether GitHub Desktop collects anonymous usage statistics and sends them to GitHub. This telemetry data helps GitHub improve the application by understanding which features are used and how. Opting out prevents any usage data from being collected or transmitted, which may be preferred for privacy reasons or in corporate environments.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Opt out** â€” No usage data collected |
| `false` | **Opt in** (default) â€” Anonymous telemetry sent |

## Our Configuration

**Set to**: `true` â€” Opt out of usage tracking for privacy.

### Why Opt Out?

1. **Privacy** â€” No usage data leaves your machine
2. **Corporate policy** â€” May be required in some workplaces
3. **Bandwidth** â€” No background data transmission
4. **Minimal footprint** â€” Leaner application behavior
5. **Personal preference** â€” Don't contribute telemetry

### Why Stay Opted In?

- Help improve GitHub Desktop
- Support open source development
- Data is anonymized
- Minimal performance impact
- Default behavior

## What Data Is Collected (When Opted In)

|-----------|-----------|---------|

| Feature usage | âœ“ | Product decisions |
| Error frequency | âœ“ | Bug prioritization |

| Repository names | âœ— | Never collected |
| Code content | âœ— | Never collected |
| Commit messages | âœ— | Never collected |
| Personal info | âœ— | Never collected |

## Security Implications

| Aspect | With Tracking | Without Tracking |
|--------|---------------|------------------|
| **Data transmitted** | Anonymous metrics | None |
| **Network requests** | Periodic telemetry | Git operations only |
| **Local storage** | May cache metrics | No metric storage |
| **Privacy** | GitHub receives usage | Complete privacy |

## Code Examples

```bash
# Read current value
defaults read com.github.GitHubClient optOutOfUsageTracking

# Opt out of usage tracking (privacy)
defaults write com.github.GitHubClient optOutOfUsageTracking -bool true

# Opt in to usage tracking (default)
defaults write com.github.GitHubClient optOutOfUsageTracking -bool false

# Delete (reset to default - opted in)
defaults delete com.github.GitHubClient optOutOfUsageTracking
```
