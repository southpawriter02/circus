# Contacts Sort Order

Control how contacts are sorted (by first or last name).

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AddressBook` |
| **Key** | `ABNameSortingFormat` |
| **Type** | String |
| **Default Value** | `sortingFirstName sortingLastName` |
| **Our Setting** | `sortingLastName sortingFirstName` |

## Description

Controls how contacts are ordered in the contact list. Sorting by last name is standard in professional contexts and makes alphabetical lookup easier for formal names.

## Values

| Value | Sort Order | Example |
|-------|------------|---------|
| `sortingFirstName sortingLastName` | **First name** (default) | Alice, Bob, Carol... |
| `sortingLastName sortingFirstName` | **Last name** | Adams, Brown, Clark... |

## Our Configuration

**Set to**: `sortingLastName sortingFirstName` — Professional alphabetical order.

### Why Sort by Last Name?

1. **Professional standard** — Address book convention
2. **Easier lookup** — Find "Smith" quickly
3. **Family grouping** — Family members appear together
4. **Formal lists** — Matches business directory format

### When First Name Sorting Works

- Personal contacts only
- Informal address books
- When you know people by first name

## Code Examples

```bash
# Read current value
defaults read com.apple.AddressBook ABNameSortingFormat

# Sort by last name (professional)
defaults write com.apple.AddressBook ABNameSortingFormat -string "sortingLastName sortingFirstName"

# Sort by first name (Apple default)
defaults write com.apple.AddressBook ABNameSortingFormat -string "sortingFirstName sortingLastName"

# Delete (reset to default)
defaults delete com.apple.AddressBook ABNameSortingFormat
```

## UI Location

**Contacts** → **Settings** (⌘,) → **General** → **Sort By**

Options:
- First Name
- Last Name

## Related Setting: Display Order

Sort order and display order are independent:

```bash
# Display as "John Smith" (first name first)
defaults write com.apple.AddressBook ABNameDisplay -int 0

# Display as "Smith, John" (last name first)
defaults write com.apple.AddressBook ABNameDisplay -int 1
```

You can sort by last name but still display "John Smith".

## Complete Professional Setup

```bash
# Business-friendly contacts
defaults write com.apple.AddressBook ABNameSortingFormat -string "sortingLastName sortingFirstName"
defaults write com.apple.AddressBook ABNameDisplay -int 0
defaults write com.apple.AddressBook ABShowNickname -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Contacts not resorting | Restart Contacts app |
| Sort order different on iPhone | Check iOS Settings → Contacts separately |
| Names displaying wrong | This is sort order, not display order (see above) |

## Compatibility

- **Syncs**: Setting syncs via iCloud to iOS/iPadOS
