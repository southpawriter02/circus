# Contacts Show Nickname

Display contact nicknames alongside real names.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AddressBook` |
| **Key** | `ABShowNickname` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether nicknames are displayed for contacts that have them. When enabled, nicknames appear alongside or instead of the contact's real name, making it easier to recognize people by their preferred names.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Show nicknames** — Display nickname when available |
| `false` | **Real names only** (default) — Always show formal name |

## Our Configuration

**Set to**: `true` — Show nicknames for personalized display.

### Why Show Nicknames?

1. **Recognition** — Easier to identify people by casual names
2. **Personalization** — Respects preferred names
3. **Friendliness** — More casual contact list
4. **Memory** — Associate names with how you know them

## Code Examples

```bash
# Read current value
defaults read com.apple.AddressBook ABShowNickname

# Show nicknames (recommended)
defaults write com.apple.AddressBook ABShowNickname -bool true

# Hide nicknames (formal names only)
defaults write com.apple.AddressBook ABShowNickname -bool false

# Delete (reset to default)
defaults delete com.apple.AddressBook ABShowNickname
```

## UI Location

**Contacts** → **Settings** (⌘,) → **General** → ☑ **Show nickname**

## Related Setting: Prefer Nickname

Replace first name with nickname entirely:

```bash
# Use nickname instead of first name
defaults write com.apple.AddressBook ABPrefersNickname -bool true
```

| Setting | Effect |
|---------|--------|
| `ABShowNickname` | Shows nickname alongside name |
| `ABPrefersNickname` | Uses nickname as primary name |

## Adding Nicknames

To add a nickname to a contact:
1. Open contact in Contacts app
2. Click **Edit**
3. Scroll to **Nickname** field (or add it via the + button)
4. Enter the nickname

## Compatibility

- **macOS**: All versions
- **iOS**: Same setting syncs via iCloud
- **Note**: Changes apply immediately
