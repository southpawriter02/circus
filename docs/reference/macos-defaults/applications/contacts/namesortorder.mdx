# Contacts Name Sort Order

Control how contacts are sorted in the list.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AddressBook` |
| **Key** | `ABNameSortingFormat` |
| **Type** | String |
| **Default Value** | `sortingFirstName sortingLastName` |
| **Our Setting** | `sortingLastName sortingFirstName` |

## Description

Controls whether contacts are sorted by first name or last name in the Contacts list. Sorting by last name is common in professional settings, while first name sorting feels more personal.

## Values

| Value | Sort Order |
|-------|------------|
| `sortingFirstName sortingLastName` | First name (John Smith before Jane Doe) |
| `sortingLastName sortingFirstName` | Last name (Doe, Jane before Smith, John) |

## Our Configuration

**Set to**: `sortingLastName sortingFirstName` — Sort by last name.

### Why Sort by Last Name?

1. **Professional** — Standard in business contexts
2. **Alphabetical** — Traditional phone book ordering
3. **Searchable** — Easier to find by surname
4. **Consistency** — Matches most directory systems

## Code Examples

```bash
# Read current value
defaults read com.apple.AddressBook ABNameSortingFormat

# Sort by last name (recommended)
defaults write com.apple.AddressBook ABNameSortingFormat -string "sortingLastName sortingFirstName"

# Sort by first name
defaults write com.apple.AddressBook ABNameSortingFormat -string "sortingFirstName sortingLastName"

# Delete (reset to default)
defaults delete com.apple.AddressBook ABNameSortingFormat
```

## UI Location

**Contacts** → **Settings** (⌘,) → **General** → **Sort By**

## Related Setting: Display Order

How names are displayed (separate from sorting):

```bash
# Display first name first (John Smith)
defaults write com.apple.AddressBook ABNameDisplay -int 0

# Display last name first (Smith, John)
defaults write com.apple.AddressBook ABNameDisplay -int 1
```

## Compatibility

- **macOS**: All versions
- **Syncs**: Via iCloud Contacts
- **Note**: Changes apply immediately
