# Contacts Include Notes in vCard Export

Control whether notes are included when exporting contact vCards.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.AddressBook` |
| **Key** | `ABIncludeNotesInVCard` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether the Notes field content is included when exporting contacts as vCard (.vcf) files. Notes can contain sensitive personal information, meeting details, or private context about relationships. Disabling this option provides a privacy-conscious default when sharing contact information, ensuring personal notes aren't inadvertently shared.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Include notes** (default) â€” Notes exported with vCard |
| `false` | **Exclude notes** â€” vCard contains only standard fields |

## Our Configuration

**Set to**: `false` â€” Exclude notes from vCard exports for privacy.

### Why Exclude Notes?

1. **Privacy** â€” Notes often contain personal observations
2. **Security** â€” May include access codes or sensitive info
3. **Professionalism** â€” Candid notes not meant to be shared
4. **GDPR/compliance** â€” Personal data shouldn't be shared carelessly
5. **Default safety** â€” Better to exclude than accidentally share

### Why Include Notes?

- Transferring contacts to your own devices
- Full backup of contact information
- Notes contain only neutral information
- Sharing with trusted family members

## Code Examples

```bash
# Read current value
defaults read com.apple.AddressBook ABIncludeNotesInVCard

# Exclude notes from vCard (recommended for sharing)
defaults write com.apple.AddressBook ABIncludeNotesInVCard -bool false

# Include notes in vCard (default)
defaults write com.apple.AddressBook ABIncludeNotesInVCard -bool true

# Delete (reset to default)
defaults delete com.apple.AddressBook ABIncludeNotesInVCard
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Notes still exported | Verify setting with `defaults read` |
| Need notes for backup | Temporarily enable, export, then disable |
| Notes missing on import | Original export excluded notes |
| vCard too large | Disable photos export |
| Can't find notes after import | Notes may have been excluded at source |

## Compatibility

- **Contacts**: macOS 10.8+ (Mountain Lion)
- **Note**: Setting applies to all export methods
- **Note**: Does not affect iCloud sync
- **Note**: vCard 3.0 format supports notes
- **Note**: Some apps may not import notes field
