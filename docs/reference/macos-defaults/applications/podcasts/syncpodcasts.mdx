# Podcasts Sync Across Devices

Sync podcast subscriptions and progress via iCloud.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `syncPodcasts` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether podcast subscriptions, listening progress, and episode status sync across all devices signed into the same Apple ID via iCloud. When enabled, you can start listening on your iPhone during a commute and continue exactly where you left off on your Mac at home. Subscriptions, played/unplayed status, and playback position all stay in sync.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Enabled** (default) â€” Sync subscriptions and progress across devices |
| `false` | **Disabled** â€” Keep podcasts local to this device only |

## Our Configuration

**Set to**: `true` â€” Enable sync for seamless multi-device experience.

### Why Enable Sync?

1. **Continuity** â€” Resume anywhere you left off
2. **Consistency** â€” Same subscriptions everywhere
3. **Convenience** â€” Subscribe once, available everywhere
4. **Progress** â€” Never lose your place
5. **Simplicity** â€” One library across devices

### Why Disable Sync?

- Different podcast preferences per device
- Privacy concerns about iCloud data
- Limited iCloud storage
- Shared Apple ID situations
- Work/personal device separation

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts syncPodcasts

# Enable podcast sync (recommended)
defaults write com.apple.podcasts syncPodcasts -bool true

# Disable podcast sync
defaults write com.apple.podcasts syncPodcasts -bool false

# Delete (reset to default)
defaults delete com.apple.podcasts syncPodcasts
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Podcasts not syncing | Check iCloud settings, same Apple ID |
| Progress not updating | Force quit and reopen Podcasts |
| Subscriptions missing | Sign out/in to iCloud |
| Sync is slow | Check internet connection |
| Conflicts between devices | Let one device finish syncing first |
| Old episodes reappearing | Sync may restore previously deleted |
