# Podcasts Auto Download

Control automatic downloading of new podcast episodes.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `downloadNewEpisodes` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Apple Podcasts automatically downloads new episodes when they become available. When enabled, subscribed podcasts download in the background for offline listening. Individual shows can override this default setting.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Auto-download enabled** (default) — New episodes download automatically |
| `false` | **Streaming only** — Episodes stream on-demand, manual download |

## Our Configuration

**Set to**: `true` — Automatically download new episodes.

### Why Enable Auto-Download?

1. **Offline listening** — Listen without internet
2. **No buffering** — Instant playback
3. **Commute-ready** — Episodes ready for subway/airplane
4. **Background sync** — Downloads while you work
5. **Reliable playback** — No network interruptions

### Why Disable Auto-Download?

- Limited storage space
- Metered data connection
- Only listen to select episodes
- Prefer streaming
- Manage downloads manually

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts downloadNewEpisodes

# Enable auto-download (default)
defaults write com.apple.podcasts downloadNewEpisodes -bool true

# Disable auto-download (stream only)
defaults write com.apple.podcasts downloadNewEpisodes -bool false

# Delete (reset to default)
defaults delete com.apple.podcasts downloadNewEpisodes
```

Restart Podcasts for changes to take effect.

## Related Settings

```bash
# Auto-download episodes (this setting)
defaults write com.apple.podcasts downloadNewEpisodes -bool true

# Remove played episodes automatically
defaults write com.apple.podcasts removePlayedEpisodes -bool true

# Episode limit per show
defaults write com.apple.podcasts episodeLimit -int 10

# Skip forward time (seconds)
defaults write com.apple.podcasts skipForwardTime -int 30

# Skip back time (seconds)
defaults write com.apple.podcasts skipBackTime -int 15

# Continuous playback
defaults write com.apple.podcasts continuousPlaybackEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Episodes not downloading | Check internet, restart Podcasts |
| Downloads paused | Check storage space |
| Slow downloads | Check network speed, try Wi-Fi |
| Setting not applying | Restart Podcasts app |
| Per-show override not working | Check show-specific settings |

## Sync Behavior

When syncing is enabled:
- Download settings sync across devices
- Per-show overrides sync
- Downloaded files do NOT sync (too large)
- Each device downloads independently

## Compatibility

- **macOS**: 10.15+ (Catalina)
- **Note**: Podcasts app required (included with macOS)
- **Note**: Restart app for changes to take effect
- **Note**: Per-show settings override global default
