# Podcasts Playback Speed

Control the default playback speed for podcast episodes.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `PlaybackSpeed` |
| **Type** | Float |
| **Default Value** | `1.0` |
| **Our Setting** | `1.0` |

## Description

Controls the default playback speed for podcast episodes in the Apple Podcasts app. Adjusting playback speed allows you to listen to podcasts faster to consume more content, or slower for better comprehension. The setting affects all podcasts by default, though individual shows can have their own speed settings. Common speeds include 1.5x for experienced listeners who want to save time, or 0.75x for complex content.

## Values

| Value | Behavior |
|-------|----------|
| `0.5` | **Half speed** — Very slow, useful for language learning |
| `0.75` | **Three-quarters** — Slower for complex content |
| `1.0` | **Normal speed** (default) — Original recording pace |
| `1.25` | **Slightly faster** — Mild time savings |
| `1.5` | **One and a half** — Popular fast listening speed |
| `1.75` | **Fast** — Significant time savings |
| `2.0` | **Double speed** — Maximum speed for experienced listeners |

## Our Configuration

**Set to**: `1.0` — Normal speed for authentic listening experience.

### Why Use Normal Speed?

1. **Authentic** — Hear content as creators intended
2. **Comprehension** — Maximum understanding of complex topics
3. **Enjoyment** — Natural pacing for entertainment content
4. **Flexibility** — Adjust per-episode as needed

### Why Increase Speed?

- Limited listening time
- Familiar with fast playback
- Conversational content with natural pauses
- Want to consume more content

### Why Decrease Speed?

- Learning new language
- Complex technical content
- Hearing impairment
- Prefer slower pace

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts PlaybackSpeed

# Set to normal speed (1.0)
defaults write com.apple.podcasts PlaybackSpeed -float 1.0

# Set to 1.5x speed
defaults write com.apple.podcasts PlaybackSpeed -float 1.5

# Set to double speed (2.0)
defaults write com.apple.podcasts PlaybackSpeed -float 2.0

# Set to slower speed (0.75)
defaults write com.apple.podcasts PlaybackSpeed -float 0.75

# Delete (reset to default)
defaults delete com.apple.podcasts PlaybackSpeed
```

## Related Settings

```bash
# Playback speed (this setting)
defaults write com.apple.podcasts PlaybackSpeed -float 1.0

# Trim silence (removes pauses)
defaults write com.apple.podcasts TrimSilence -bool false

# Skip forward interval (seconds)
defaults write com.apple.podcasts skipForwardTime -int 30

# Skip back interval (seconds)
defaults write com.apple.podcasts skipBackTime -int 15

# Continuous playback (auto-play next)
defaults write com.apple.podcasts continuousPlaybackEnabled -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Speed not changing | Restart Podcasts app; check per-show override |
| Audio sounds distorted | Lower speed; some voices don't compress well |
| Speed resets | Per-show settings may override default |
| Can't find speed control | Check Now Playing screen bottom area |
| Voices sound chipmunk-like | Apple Podcasts uses pitch correction; this is a bug |

## Compatibility

- **macOS**: 10.15+ (Catalina - when Podcasts became standalone app)
- **Note**: Speed changes don't affect downloaded file, only playback
- **Note**: Per-show settings override this default
- **Note**: iOS Podcasts app syncs speed preferences via iCloud
- **Note**: Some third-party podcast apps have different speed ranges
