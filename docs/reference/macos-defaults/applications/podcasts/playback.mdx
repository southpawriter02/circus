# Podcasts Playback Settings

Configure skip times, continuous playback, and headphone controls.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Keys** | `skipForwardTime`, `skipBackTime`, `continuousPlaybackEnabled` |
| **Type** | Integer (times), Boolean (continuous) |
| **Default Values** | Forward: 30s, Back: 15s, Continuous: true |
| **Our Setting** | Forward: 30s, Back: 15s, Continuous: true |

## Description

Controls playback behavior in Apple Podcasts including how many seconds to skip forward/backward and whether to automatically play the next episode when one finishes. These settings apply globally but can be overridden with keyboard shortcuts.

## Values

### Skip Forward Time

| Value | Skip Duration |
|-------|---------------|
| `10` | 10 seconds |
| `15` | 15 seconds |
| `30` | 30 seconds (default) |
| `45` | 45 seconds |
| `60` | 60 seconds |

### Skip Back Time

| Value | Skip Duration |
|-------|---------------|
| `10` | 10 seconds |
| `15` | 15 seconds (default) |
| `30` | 30 seconds |
| `45` | 45 seconds |
| `60` | 60 seconds |

### Continuous Playback

| Value | Behavior |
|-------|----------|
| `true` | **Auto-play next** (default) — Next episode plays automatically |
| `false` | **Stop after episode** — Playback stops when episode ends |

## Code Examples

```bash
# Read current values
defaults read com.apple.podcasts skipForwardTime
defaults read com.apple.podcasts skipBackTime
defaults read com.apple.podcasts continuousPlaybackEnabled

# Set skip forward time (30 seconds)
defaults write com.apple.podcasts skipForwardTime -int 30

# Set skip back time (15 seconds)
defaults write com.apple.podcasts skipBackTime -int 15

# Enable continuous playback
defaults write com.apple.podcasts continuousPlaybackEnabled -bool true

# Disable continuous playback
defaults write com.apple.podcasts continuousPlaybackEnabled -bool false

# Delete (reset all to defaults)
defaults delete com.apple.podcasts skipForwardTime
defaults delete com.apple.podcasts skipBackTime
defaults delete com.apple.podcasts continuousPlaybackEnabled
```

Restart Podcasts for changes to take effect.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Right Arrow | Skip forward |
| Left Arrow | Skip back |
| Space | Play/Pause |
| Cmd + Right | Next episode |
| Cmd + Left | Previous episode |
| Cmd + Up | Volume up |
| Cmd + Down | Volume down |

## Headphone Controls

```
+-------------------------------------------------------------+
|  Headphone Button Behavior                                   |
+-------------------------------------------------------------+
|                                                              |
|  AirPods / Wired Headphones:                                |
|                                                              |
|  Single Press:    Play / Pause                              |
|  Double Press:    Skip Forward (default)                    |
|                   OR Next Episode (if enabled)              |
|  Triple Press:    Skip Back                                 |
|                                                              |
|  Setting: headphoneControlsNextEpisode                      |
|  +------------------------------------------------------+   |
|  |  [ ] Double-press skips to next episode              |   |
|  +------------------------------------------------------+   |
|                                                              |
+-------------------------------------------------------------+
```

```bash
# Headphone double-press behavior
# false = skip forward, true = next episode
defaults write com.apple.podcasts headphoneControlsNextEpisode -bool false
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Skip buttons not working | Restart Podcasts, check settings |
| Wrong skip duration | Verify defaults with `defaults read` |
| Continuous playback not working | Check Up Next queue isn't empty |
| Headphones skip wrong | Check headphoneControlsNextEpisode setting |
| Settings reset | Preferences may have corrupted; delete plist |

## Compatibility

- **macOS**: 10.15+ (Catalina)
- **Note**: Restart Podcasts for changes to take effect
- **Note**: Skip times must be one of: 10, 15, 30, 45, 60
- **Note**: Headphone controls depend on headphone type
