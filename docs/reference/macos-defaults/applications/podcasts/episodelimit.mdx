# Podcasts Episode Limit

Set maximum episodes to keep per podcast subscription.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `episodeLimit` |
| **Type** | Integer |
| **Default Value** | `0` |
| **Our Setting** | `10` |

## Description

Controls the maximum number of episodes to retain for each podcast subscription. When the limit is reached, older episodes are automatically removed to make room for new ones. This helps manage storage space, especially on devices with limited capacity. A value of `0` means no limit â€” all episodes are kept indefinitely.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **No limit** (default) â€” Keep all episodes |
| `1` | Most recent episode only |
| `3` | Last 3 episodes |
| `5` | Last 5 episodes |
| `10` | Last 10 episodes |

## Our Configuration

**Set to**: `10` â€” Keep last 10 episodes for balance of storage and availability.

### Why 10 Episodes?

1. **Storage** â€” Predictable, bounded storage use
2. **Catch-up** â€” Two weeks of backlog for most shows
3. **Flexibility** â€” Miss a few days without losing content
4. **Automatic** â€” No manual cleanup needed
5. **Reasonable** â€” Older episodes rarely listened to

### Why No Limit?

- Archive shows for reference
- Irregular listening schedule
- Plenty of storage available
- Want complete collections
- Serial podcasts (need all episodes)

### Why Fewer Episodes?

- Very limited storage
- Listen immediately
- Many subscriptions
- Stream rather than download

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts episodeLimit

# Keep last 10 episodes
defaults write com.apple.podcasts episodeLimit -int 10

# Keep last 5 episodes
defaults write com.apple.podcasts episodeLimit -int 5

# Keep only most recent episode
defaults write com.apple.podcasts episodeLimit -int 1

# No limit (keep all)
defaults write com.apple.podcasts episodeLimit -int 0

# Delete (reset to default)
defaults delete com.apple.podcasts episodeLimit
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Episodes not being removed | Check per-show overrides |
| Wrong number of episodes | Restart Podcasts app |
| Storage still growing | Check "Remove Played Downloads" |
| Episodes disappearing too fast | Increase limit |
| Need old episode | Re-download from show page |

## Compatibility

- **macOS**: 10.15+ (Catalina, when Podcasts became separate app)
- **Note**: Setting syncs via iCloud (affects all devices)
- **Note**: Per-show overrides take precedence
- **Note**: Restart may be required for changes
