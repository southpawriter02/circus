# Podcasts Notifications Enabled

Control whether the Podcasts app shows notifications for new episodes.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `NotificationsEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Apple Podcasts sends notifications when new episodes are available for subscribed podcasts. When enabled, you receive alerts through the macOS notification system when your favorite shows release new content. This helps you stay current with podcasts without manually checking the app.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Notifications enabled** (default) â€” Alerts for new episodes |
| `false` | **Notifications disabled** â€” Silent updates |

## Our Configuration

**Set to**: `true` â€” Enable notifications to stay informed about new episodes.

### Why Enable Notifications?

1. **Stay current** â€” Know when favorites release new episodes
2. **Timely listening** â€” Don't miss time-sensitive content
3. **Discovery** â€” Remember subscriptions you might forget
4. **Convenience** â€” No need to check app manually
5. **Quick access** â€” Click notification to start listening

### Why Disable Notifications?

- Too many subscriptions, overwhelming notifications
- Prefer to browse at your own pace
- Notification fatigue
- Focus/DND workflow
- Check podcasts on own schedule

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts NotificationsEnabled

# Enable notifications (default)
defaults write com.apple.podcasts NotificationsEnabled -bool true

# Disable notifications
defaults write com.apple.podcasts NotificationsEnabled -bool false

# Delete (reset to default)
defaults delete com.apple.podcasts NotificationsEnabled
```

## Related Settings

```bash
# Notifications enabled (this setting)
defaults write com.apple.podcasts NotificationsEnabled -bool true

# Auto-download new episodes
defaults write com.apple.podcasts downloadNewEpisodes -bool true

# Sync podcasts via iCloud
defaults write com.apple.podcasts syncPodcasts -bool true

# Episode limit per podcast
defaults write com.apple.podcasts episodeLimit -int 10

# Remove played episodes
defaults write com.apple.podcasts removePlayedEpisodes -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No notifications appearing | Check System Settings > Notifications > Podcasts |
| Too many notifications | Disable per-show in show settings |
| Sound with notifications | Configure in System Settings > Notifications |
| Notifications delayed | Check Focus mode settings |
| Setting has no effect | Restart Podcasts app after changing |
