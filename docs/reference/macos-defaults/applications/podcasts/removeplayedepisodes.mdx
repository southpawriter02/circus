# Podcasts Remove Played Episodes

Automatically delete downloaded episodes after playing.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.podcasts` |
| **Key** | `removePlayedEpisodes` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether the Podcasts app automatically removes downloaded episode files after they've been played. This helps manage storage space by removing episodes you've already listened to. The episode remains in your library (not deleted from subscription), but the downloaded audio file is removed from local storage.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Auto-remove** (default) â€” Delete download after playing |
| `false` | **Keep all** â€” Downloaded episodes remain until manually deleted |

## Our Configuration

**Set to**: `true` â€” Automatically remove played episodes to save storage.

### Why Auto-Remove?

1. **Storage** â€” Prevents podcast buildup consuming disk space
2. **Automatic** â€” No manual cleanup required
3. **Re-downloadable** â€” Can always re-download if needed
4. **Efficient** â€” Most episodes only listened to once
5. **Default behavior** â€” What most users expect

### Why Keep All?

- Frequently re-listen to episodes
- Reference podcast for work/study
- Offline access important
- Large storage available
- Archiving specific shows

## Code Examples

```bash
# Read current value
defaults read com.apple.podcasts removePlayedEpisodes

# Enable auto-remove (default, recommended)
defaults write com.apple.podcasts removePlayedEpisodes -bool true

# Disable auto-remove (keep all downloads)
defaults write com.apple.podcasts removePlayedEpisodes -bool false

# Delete (reset to default)
defaults delete com.apple.podcasts removePlayedEpisodes
```

## Related Settings

```bash
# Remove played episodes (this setting)
defaults write com.apple.podcasts removePlayedEpisodes -bool true

# Episode limit per podcast (0 = no limit)
defaults write com.apple.podcasts episodeLimit -int 10

# Auto-download new episodes
defaults write com.apple.podcasts downloadNewEpisodes -bool true

# Sync podcasts across devices
defaults write com.apple.podcasts syncPodcasts -bool true

# Sync listening history
defaults write com.apple.podcasts syncListeningHistory -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Episodes not being removed | Ensure fully played (not partially) |
| Want to keep specific episode | Star/favorite it, or disable per-show |
| Accidentally removed episode | Re-download from episode list |
| Storage still high | Check per-show settings; review limits |
| Synced device keeping episodes | Check setting on each device |
