# Books Page Turn Animation

Enable or disable the animated page-turning effect when reading.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKAnimatePageTurn` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Books displays an animated page-turning effect when navigating between pages. When enabled, pages appear to curl and flip like a physical book. Disabling this animation results in instant page transitions, which some users prefer for faster reading or reduced visual distraction.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Animation enabled** (default) — Pages curl and flip like a real book |
| `false` | **Animation disabled** — Instant page transitions |

## Our Configuration

**Set to**: `true` — Enable page turn animation for natural reading feel.

### Why Enable Animation?

1. **Natural feel** — Mimics physical book experience
2. **Visual feedback** — Confirms page turn action
3. **Reading rhythm** — Creates natural reading pace
4. **Orientation** — Helps track position in book
5. **Enjoyment** — Many readers find it satisfying

### Why Disable Animation?

- Prefer instant transitions
- Speed reading or skimming
- Motion sensitivity
- Faster navigation when searching
- Minimal interface preference

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKAnimatePageTurn

# Enable page turn animation (default)
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true

# Disable page turn animation
defaults write com.apple.iBooksX BKAnimatePageTurn -bool false

# Delete (reset to default)
defaults delete com.apple.iBooksX BKAnimatePageTurn
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Animation not showing | Check setting is enabled |
| Animation is choppy | Normal on older Macs; try disabling |
| Want faster reading | Disable animation for instant transitions |
| Setting not applying | Restart Books app |
| Different on iPad/iPhone | iOS has separate setting |

## Compatibility

- **Books**: macOS 10.14+ (Mojave)
- **Note**: Animation only applies to EPUB format books
- **Note**: PDFs use different navigation (no page curl)
- **Note**: Audiobooks not affected (no pages)
- **Note**: Setting syncs via iCloud with Books settings
