# Books Reader Theme

Configure the color theme for reading books in Apple Books.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKReaderTheme` |
| **Type** | Integer |
| **Default Value** | `0` |
| **Our Setting** | `0` |

## Description

Controls the background and text color theme used when reading books in Apple Books. The reader theme affects the overall appearance of the reading interface, including background color, text color, and UI elements. Different themes can reduce eye strain depending on ambient lighting conditions and personal preference. The theme applies to all books in your library.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **White** (default) â€” Light background with dark text |
| `1` | **Sepia** â€” Warm, yellowish background with dark text |
| `2` | **Gray** â€” Medium gray background with light text |
| `3` | **Night** â€” Dark background with light text |

## Our Configuration

**Set to**: `0` â€” White theme for clear, high-contrast reading.

### Why Use White Theme?

1. **Maximum readability** â€” Highest contrast for text clarity
2. **Standard experience** â€” Matches printed books
3. **Daylight reading** â€” Best for bright environments
4. **Color accuracy** â€” Book covers and images display correctly

### Why Use Sepia?

- Extended reading sessions (reduces eye fatigue)
- Prefer warm, paper-like appearance
- Reading in moderate lighting
- Sensitive to bright white screens

### Why Use Gray?

- Moderate lighting environments
- Transitioning between day and night
- Prefer lower overall brightness

### Why Use Night?

- Reading in bed or dark rooms
- Nighttime reading
- Minimize screen brightness
- OLED display battery savings

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKReaderTheme

# Set to White theme (default)
defaults write com.apple.iBooksX BKReaderTheme -int 0

# Set to Sepia theme
defaults write com.apple.iBooksX BKReaderTheme -int 1

# Set to Gray theme
defaults write com.apple.iBooksX BKReaderTheme -int 2

# Set to Night theme
defaults write com.apple.iBooksX BKReaderTheme -int 3

# Delete (reset to default)
defaults delete com.apple.iBooksX BKReaderTheme
```

## Related Settings

```bash
# Reader theme (this setting)
defaults write com.apple.iBooksX BKReaderTheme -int 0

# Auto night theme (auto-switch based on lighting/time)
defaults write com.apple.iBooksX BKAutoNightThemeEnabled -bool true

# Page turn animation
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true

# Scroll direction (horizontal vs vertical)
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Show status bar while reading
defaults write com.apple.iBooksX BKShowStatusBarWhileReading -bool false

# Last used font family
defaults write com.apple.iBooksX BKLastUsedFontFamilyName -string "Georgia"

# Last used font size
defaults write com.apple.iBooksX BKLastUsedFontSize -int 18
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Theme not changing | Close and reopen the book; restart Books app |
| Theme resets on reopen | Check if Auto Night Theme is overriding; disable if unwanted |
| Text hard to read | Try different theme; adjust font size with Aa button |
| Colors look wrong | Check display color profile; disable Night Shift if active |
| Theme different on other device | Enable iCloud sync for Books; use same Apple ID |

## Compatibility

- **macOS**: 10.14+ (Mojave and later)
- **Note**: Books was renamed from iBooks in macOS Mojave
- **Note**: Theme preference syncs via iCloud if enabled
- **Note**: Some PDFs may not respect theme settings
- **Note**: Fixed-layout books (comics, picture books) ignore theme

## Source File

**File**: [`defaults/applications/books.sh`](../../../../../defaults/applications/books.sh)

```bash
# BKReaderTheme controls the reading color theme
# Values: 0 = White, 1 = Sepia, 2 = Gray, 3 = Night
# The setting is stored per-user and syncs via iCloud
```
