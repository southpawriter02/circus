# Books Show Status Bar While Reading

Control whether the macOS menu bar remains visible while reading books in fullscreen mode.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKShowStatusBarWhileReading` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `false` |

## Description

Controls whether the macOS menu bar (status bar) remains visible when reading a book in Apple Books. When disabled, the menu bar hides during reading to provide a more immersive, distraction-free experience. When enabled, the menu bar stays visible, providing quick access to the time, battery status, Wi-Fi, and other menu bar items without moving the cursor to the top of the screen.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Hide status bar** (default) — Immersive reading mode |
| `true` | **Show status bar** — Menu bar always visible while reading |

## Our Configuration

**Set to**: `false` — Hide status bar for immersive reading.

### Why Hide Status Bar (false)?

1. **Immersive reading** — Focus entirely on the book
2. **Maximum space** — More text visible on screen
3. **Distraction-free** — No clock or notifications
4. **Book-like** — Closer to physical book experience
5. **Default behavior** — Standard Books experience

### Why Show Status Bar (true)?

- **Time awareness** — Need to see the clock
- **Battery monitoring** — Check power while reading
- **Quick access** — Frequent menu bar use
- **Consistency** — Prefer consistent macOS UI
- **Work reading** — Stay connected to system

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKShowStatusBarWhileReading

# Hide status bar while reading (default, recommended)
defaults write com.apple.iBooksX BKShowStatusBarWhileReading -bool false

# Show status bar while reading
defaults write com.apple.iBooksX BKShowStatusBarWhileReading -bool true

# Delete (reset to default)
defaults delete com.apple.iBooksX BKShowStatusBarWhileReading
```

## Related Settings

```bash

defaults write com.apple.iBooksX BKShowStatusBarWhileReading -bool false

# Auto night theme
defaults write com.apple.iBooksX BKAutoNightThemeEnabled -bool true

# Page turn animation
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true

# Scroll direction (horizontal/vertical)
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Sync reading position
defaults write com.apple.iBooksX BKSyncReadingPosition -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Status bar not hiding | Ensure you're in fullscreen or reading mode |
| Can't access menu bar | Move cursor to top of screen to reveal |
| Setting not applying | Restart Books app after changing via defaults |
| Want status bar only sometimes | Move cursor to top to temporarily reveal |
| Clock needed while reading | Enable status bar or use external clock |
| Menu bar stuck visible | Check macOS System Settings > Desktop & Dock |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Books
- **Note**: Only affects reading mode, not library view
- **Note**: Restart Books for terminal changes to apply
- **Note**: Menu bar can always be revealed by moving cursor to top
- **Note**: Works in both windowed and fullscreen modes
