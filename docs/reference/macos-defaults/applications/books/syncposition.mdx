# Books Sync Reading Position

Continue reading on any device from where you left off.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKSyncReadingPosition` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Syncs your current reading position across all Apple devices signed into the same Apple ID. Start reading on your Mac and continue on iPhone or iPad exactly where you left off.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sync position** (default) — Resume anywhere |
| `false` | **Local only** — Each device tracks separately |

## Our Configuration

**Set to**: `true` — Seamless cross-device reading.

### Why Sync Position?

1. **Continuity** — Read on any device
2. **Convenience** — No manual bookmarking
3. **Flexibility** — Mac at home, iPhone on commute
4. **Automatic** — Just works

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKSyncReadingPosition

# Sync reading position (default)
defaults write com.apple.iBooksX BKSyncReadingPosition -bool true

# Don't sync (local position only)
defaults write com.apple.iBooksX BKSyncReadingPosition -bool false

# Delete (reset to default)
defaults delete com.apple.iBooksX BKSyncReadingPosition
```

## UI Location

**Books** → **Settings** (⌘,) → **General**

## What Syncs

With sync enabled, these sync via iCloud:
- Current page/position
- Bookmarks
- Highlights
- Notes/annotations
- Collections

## Related Sync Settings

```bash
# Sync bookmarks and highlights
defaults write com.apple.iBooksX BKSyncBookmarksAndHighlights -bool true

# Sync collections (folders)
defaults write com.apple.iBooksX BKSyncCollections -bool true
```

## iCloud Requirements

For sync to work:
1. Sign into same Apple ID
2. Enable iCloud Books
3. Internet connection

## Conflict Resolution

If reading on multiple devices simultaneously:
- Most recent position wins
- Occasional sync conflicts possible
- Manual bookmarks are preserved

## Compatibility

- **macOS**: 10.14+
- **iOS**: Requires iCloud
- **Requires**: Same Apple ID
