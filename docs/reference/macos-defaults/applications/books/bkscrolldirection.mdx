# Books Scroll Direction

Control horizontal or vertical scrolling when reading in Books.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKScrollDirection` |
| **Type** | Integer |
| **Default Value** | `1` |
| **Our Setting** | `1` |

## Description

Controls the scrolling direction for reading content in the Books app. Horizontal scrolling (the default) mimics the experience of turning pages in a physical book, moving left to right through pages. Vertical scrolling provides a continuous scrolling experience similar to web pages or documents, which some readers prefer for longer reading sessions.

## Values

| Value | Behavior |
|-------|----------|
| `0` | **Vertical scrolling** — Scroll up/down through content continuously |
| `1` | **Horizontal scrolling** (default) — Turn pages left/right like a book |

## Our Configuration

**Set to**: `1` — Horizontal scrolling for a book-like reading experience.

### Why Horizontal Scrolling (Default)?

1. **Natural feel** — Mimics turning pages in a physical book
2. **Page awareness** — Clear boundaries help track progress
3. **Animation support** — Works with page turn animations
4. **Standard experience** — Most readers expect this behavior
5. **Better for most content** — Optimal for fiction and formatted books

### Why Use Vertical Scrolling?

- Reading technical documentation
- Long reference materials
- Personal preference for continuous flow
- Faster scanning through content
- Web-style reading habit

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKScrollDirection

# Set horizontal scrolling (default, book-like)
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Set vertical scrolling (continuous)
defaults write com.apple.iBooksX BKScrollDirection -int 0

# Delete (reset to default)
defaults delete com.apple.iBooksX BKScrollDirection
```

## Related Settings

```bash
# Scroll direction (this setting)
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Page turn animation
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true

# Two-page view
defaults write com.apple.iBooksX BKTwoPageMode -bool false

# Auto night theme
defaults write com.apple.iBooksX BKAutoNightThemeEnabled -bool true

# Show status bar while reading
defaults write com.apple.iBooksX BKShowStatusBarWhileReading -bool false
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Scroll direction not changing | Restart Books app |
| Can't find View menu option | Must be reading a book (not in library) |
| Scroll feels inverted | Check System Settings trackpad natural scrolling |
| Page animations not working | Horizontal mode needed for page animations |
| Content layout looks wrong | Try switching direction for that book type |

## Compatibility

- **Books**: macOS 10.14+ (Mojave)
- **Note**: Setting applies to all EPUB books
- **Note**: PDFs have their own scroll behavior
- **Note**: Comics may override with their own navigation
- **Note**: Setting syncs via iCloud with Books settings
