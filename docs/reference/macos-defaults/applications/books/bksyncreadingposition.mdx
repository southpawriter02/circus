# Books Sync Reading Position

Control whether your current reading position syncs across all your Apple devices via iCloud.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKSyncReadingPosition` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Apple Books syncs your current page or reading position across all devices signed into the same Apple ID via iCloud. When enabled, you can start reading on your Mac and seamlessly continue from the exact same spot on your iPhone, iPad, or another Mac. This creates a truly unified reading experience across the Apple ecosystem.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sync position** (default) — Reading position syncs via iCloud |
| `false` | **Don't sync** — Each device tracks position independently |

## Our Configuration

**Set to**: `true` — Sync reading position across all devices.

### Why Enable Position Sync?

1. **Seamless reading** — Continue anywhere you stopped
2. **Multi-device use** — Read on Mac, continue on iPhone
3. **No manual tracking** — iCloud remembers for you
4. **Commute reading** — Switch devices effortlessly
5. **Progress tracking** — Always know where you are

### Why Disable Position Sync?

- **Privacy concerns** — Multiple people using same Apple ID
- **Offline usage** — No iCloud connectivity
- **Different reading** — Want different positions per device
- **Troubleshooting** — Sync causing issues

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKSyncReadingPosition

# Enable reading position sync (default, recommended)
defaults write com.apple.iBooksX BKSyncReadingPosition -bool true

# Disable reading position sync
defaults write com.apple.iBooksX BKSyncReadingPosition -bool false

# Delete (reset to default)
defaults delete com.apple.iBooksX BKSyncReadingPosition
```


## Troubleshooting

| Issue | Solution |
|-------|----------|
| Position not syncing | Check iCloud connection and sign-in |
| Wrong position on new device | Force quit Books, reopen; wait for sync |
| Position keeps reverting | Check for conflicting edits on another device |
| Sync delayed | iCloud syncs periodically, not instantly |
| Position off by a page | Different font sizes affect pagination |
| PDFs not syncing | Some PDFs may not support position sync |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Books
- **iOS**: Works with iPhone and iPad Books app
- **Note**: Requires iCloud account and internet connection
- **Note**: Restart Books for terminal changes to apply
- **Note**: Books must be purchased or uploaded to iCloud Books
- **Note**: Side-loaded EPUBs may have limited sync support
