# Books Highlight Color Index

Set the default highlight color for annotations in Apple Books.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKHighlightColorIndex` |
| **Type** | Integer |
| **Default Value** | `0` |
| **Our Setting** | `0` |

## Description

Controls the default color used when highlighting text in Apple Books. When you select text and choose to highlight it, this color is applied automatically. You can change the color for individual highlights, but this setting determines which color is selected by default. Different colors can be used to categorize highlights (e.g., yellow for important, pink for questions, blue for definitions).

## Values

| Value | Color | Description |
|-------|-------|-------------|
| `0` | **Yellow** | Default, classic highlighter color |
| `1` | **Green** | Secondary highlight color |
| `2` | **Blue** | Cool-toned highlight |
| `3` | **Pink** | Warm accent color |
| `4` | **Purple** | Distinctive accent color |

## Our Configuration

**Set to**: `0` â€” Yellow as the traditional default highlight color.

### Why Use Yellow (0)?

1. **Traditional** â€” Classic highlighter color
2. **Visible** â€” High contrast on white pages
3. **Neutral** â€” Works with all reading themes
4. **Familiar** â€” Matches physical highlighter experience

### Why Use Other Colors?

- **Green (1)**: Prefer cooler highlight tone
- **Blue (2)**: Match a color-coding system
- **Pink (3)**: Want warmer accent
- **Purple (4)**: Distinctive personal style

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKHighlightColorIndex

# Set to yellow (default)
defaults write com.apple.iBooksX BKHighlightColorIndex -int 0

# Set to green
defaults write com.apple.iBooksX BKHighlightColorIndex -int 1

# Set to blue
defaults write com.apple.iBooksX BKHighlightColorIndex -int 2

# Set to pink
defaults write com.apple.iBooksX BKHighlightColorIndex -int 3

# Set to purple
defaults write com.apple.iBooksX BKHighlightColorIndex -int 4

# Delete (reset to default)
defaults delete com.apple.iBooksX BKHighlightColorIndex
```

## Related Settings

```bash
# Highlight color (this setting)
defaults write com.apple.iBooksX BKHighlightColorIndex -int 0

# Sync highlights via iCloud
defaults write com.apple.iBooksX BKSyncBookmarksAndHighlights -bool true

# Font size for reading
defaults write com.apple.iBooksX BKLastUsedFontSize -int 4

# Font family
defaults write com.apple.iBooksX BKLastUsedFontFamilyName -string "Georgia"

# Reader theme
defaults write com.apple.iBooksX BKReaderTheme -int 0
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Wrong color applied | Select text, change color in popup; becomes new default |
| Highlights not syncing | Check BKSyncBookmarksAndHighlights is enabled |
| Can't highlight text | Some books may restrict highlighting (DRM) |
| Color looks different | May vary by reading theme (Sepia, Night, etc.) |
| Setting not persisting | Quit Books before changing via defaults |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Books app
- **Note**: Highlights sync across devices via iCloud
- **Note**: Color appearance varies by reading theme
- **Note**: Some protected books may not allow highlights
- **Note**: Works with ePub and iBooks format
