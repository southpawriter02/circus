# Books Last Used Font Size

Set the default font size for reading books in Apple Books.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKLastUsedFontSize` |
| **Type** | Integer |
| **Default Value** | `4` |
| **Our Setting** | `4` |

## Description

Controls the default font size used when reading books in Apple Books. This setting remembers your preferred text size and applies it to new books. Individual books may remember their own font size settings, but this sets the baseline for the reading experience. Larger sizes improve readability but show less content per page; smaller sizes fit more text but may strain eyes.

## Values

| Value | Size | Description |
|-------|------|-------------|
| `0` | **XS** | Extra Small (~10pt) |
| `1` | **S** | Small (~12pt) |
| `2` | **SM** | Small-Medium (~14pt) |
| `3` | **M** | Medium (~16pt) |
| `4` | **ML** | Medium-Large (~18pt) â€” Default |
| `5` | **L** | Large (~20pt) |
| `6` | **XL** | Extra Large (~24pt) |
| `7` | **XXL** | Extra Extra Large (~28pt+) |

## Our Configuration

**Set to**: `4` â€” Medium-Large for comfortable default reading.

### Why Use Medium-Large (4)?

1. **Comfortable** â€” Easy on eyes for extended reading
2. **Balanced** â€” Good text-per-page ratio
3. **Accessible** â€” Works for most vision needs
4. **Default** â€” Matches Apple's chosen default

### Why Use Smaller?

- Read more content per page
- Reference books with lots of text
- Comfortable with small text
- Larger displays where small text is still readable

### Why Use Larger?

- Vision difficulties
- Reading in low light
- Reduce eye strain
- Leisurely reading pace

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKLastUsedFontSize

# Set to small (size 1)
defaults write com.apple.iBooksX BKLastUsedFontSize -int 1

# Set to medium-large (default, size 4)
defaults write com.apple.iBooksX BKLastUsedFontSize -int 4

# Set to extra large (size 6)
defaults write com.apple.iBooksX BKLastUsedFontSize -int 6

# Delete (reset to default)
defaults delete com.apple.iBooksX BKLastUsedFontSize
```

## Related Settings

```bash
# Font size (this setting)
defaults write com.apple.iBooksX BKLastUsedFontSize -int 4

# Font family used
defaults write com.apple.iBooksX BKLastUsedFontFamilyName -string "Georgia"

# Reader theme (White/Sepia/Gray/Night)
defaults write com.apple.iBooksX BKReaderTheme -int 0

# Scroll direction
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Page turn animation
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Font size not persisting | Quit Books before changing via defaults |
| Different size per book | Individual books remember their settings |
| Can't change font size | Some PDFs/fixed-layout books don't allow |
| Text too small/large | Use âŒ˜+ or âŒ˜- to adjust while reading |
| Setting has no visible effect | Try different value (0-7 range) |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Books app
- **Note**: Only affects reflowable books (not fixed-layout)
- **Note**: PDFs maintain their original formatting
- **Note**: Individual books may override this default
- **Note**: Syncs with iOS Books app via iCloud
