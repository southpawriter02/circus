# Books Sync Bookmarks and Highlights

Sync your bookmarks, highlights, and annotations across all Apple devices.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKSyncBookmarksAndHighlights` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether Books syncs your bookmarks, highlights, and text annotations across all devices signed into the same Apple ID via iCloud. When enabled, any highlight you make on your Mac appears on your iPhone and iPad, and vice versa. This creates a seamless reading experience across all your Apple devices.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Sync enabled** (default) â€” Annotations sync across all devices |
| `false` | **Sync disabled** â€” Annotations stay local to each device |

## Our Configuration

**Set to**: `true` â€” Enable sync for seamless multi-device reading.

### Why Enable Sync (Recommended)?

1. **Continuity** â€” Start highlighting on Mac, continue on iPhone
2. **Backup** â€” Annotations stored in iCloud (recoverable)
3. **Study** â€” Review all highlights from any device
4. **Convenience** â€” No duplicate effort across devices
5. **Export** â€” All annotations available for export from any device

### Why Disable Sync?

- Privacy concerns about iCloud storage
- Using different Apple IDs on devices
- Want separate annotation sets per device
- Limited iCloud storage
- Shared device with other users

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKSyncBookmarksAndHighlights

# Enable annotation sync (recommended)
defaults write com.apple.iBooksX BKSyncBookmarksAndHighlights -bool true

# Disable annotation sync
defaults write com.apple.iBooksX BKSyncBookmarksAndHighlights -bool false

# Delete (reset to default)
defaults delete com.apple.iBooksX BKSyncBookmarksAndHighlights
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Annotations not syncing | Check iCloud settings, same Apple ID |
| Missing highlights on device | Wait for sync; check internet connection |
| Duplicate annotations | May occur with network issues; usually resolves |
| Can't export annotations | Use Notes panel copy/paste or third-party tools |
| Sync taking too long | Large libraries take time; be patient |
