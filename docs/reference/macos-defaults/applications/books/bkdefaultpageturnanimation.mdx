# Books Default Page Turn Animation Style

Set the default animation style for page turns in Apple Books.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.iBooksX` |
| **Key** | `BKDefaultPageTurnAnimation` |
| **Type** | Integer |
| **Default Value** | `1` |
| **Our Setting** | `1` |

## Description

Controls the default animation style used when turning pages in Apple Books. This setting determines how page transitions appear when reading ebooks. The curl animation provides a realistic book-like experience, while other options offer faster or simpler transitions. This is separate from `BKAnimatePageTurn` which enables/disables animations entirely.

## Values

| Value | Animation Style | Description |
|-------|----------------|-------------|
| `0` | **None** | Instant page change, no animation |
| `1` | **Curl** (default) | Realistic page curl effect |
| `2` | **Slide** | Pages slide horizontally |

## Our Configuration

**Set to**: `1` â€” Curl animation for realistic page turns.

### Why Use Curl Animation?

1. **Immersive** â€” Feels like reading a physical book
2. **Visual feedback** â€” Clear indication of page turning
3. **Apple default** â€” Polished, expected behavior
4. **Relaxing** â€” Smooth animation for leisure reading
5. **Natural** â€” Mimics real-world interaction

### Why Use Other Animations?

**Slide (2)**:
- Faster page transitions
- Modern, app-like feel
- Good for skimming content

**None (0)**:
- Maximum reading speed
- Better for reference materials
- Reduced visual distraction
- Lower battery usage

## Code Examples

```bash
# Read current value
defaults read com.apple.iBooksX BKDefaultPageTurnAnimation

# Set to no animation (instant)
defaults write com.apple.iBooksX BKDefaultPageTurnAnimation -int 0

# Set to curl animation (default)
defaults write com.apple.iBooksX BKDefaultPageTurnAnimation -int 1

# Set to slide animation
defaults write com.apple.iBooksX BKDefaultPageTurnAnimation -int 2

# Delete (reset to default)
defaults delete com.apple.iBooksX BKDefaultPageTurnAnimation
```

## Related Settings

```bash
# Page turn animation style (this setting)
defaults write com.apple.iBooksX BKDefaultPageTurnAnimation -int 1

# Enable/disable page turn animation entirely
defaults write com.apple.iBooksX BKAnimatePageTurn -bool true

# Scroll direction (horizontal vs vertical)
defaults write com.apple.iBooksX BKScrollDirection -int 1

# Reader theme
defaults write com.apple.iBooksX BKReaderTheme -int 0

# Font size
defaults write com.apple.iBooksX BKLastUsedFontSize -int 3
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Animation not changing | Quit and reopen Books app |
| Animation feels slow | Switch to Slide or None |
| Animation feels jarring | Use Curl for smoother effect |
| Battery draining quickly | Use None to reduce GPU usage |
| PDFs not animating | PDF page turns may have separate behavior |
| Setting not persisting | Check file permissions in ~/Library/Preferences |

## Compatibility

- **macOS**: 10.14+ (Mojave) with Books app
- **Note**: This setting applies to ebooks (EPUB) primarily
- **Note**: PDFs may have different animation behavior
- **Note**: Audiobooks don't use page turn animations
- **Note**: Restart Books for terminal changes to apply
