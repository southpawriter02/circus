# Notes Group Notes By Date

Control whether notes are grouped by date in the note list.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.Notes` |
| **Key** | `GroupNotesByDate` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `true` |

## Description

Controls whether notes in the note list are grouped under date headers like "Today", "Yesterday", "Previous 7 Days", "Previous 30 Days", and older periods. When enabled, notes are organized chronologically with visual separators. When disabled, notes appear as a flat list without date groupings.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Group by date** (default) â€” Notes organized under date headers |
| `false` | **Flat list** â€” All notes in a single ungrouped list |

## Our Configuration

**Set to**: `true` â€” Group notes by date for better organization.

### Why Group By Date?

1. **Recency** â€” Quickly find recently edited notes
2. **Organization** â€” Visual structure aids navigation
3. **Context** â€” Know when you last worked on a note
4. **Productivity** â€” Less scrolling to find recent items
5. **Default behavior** â€” Matches user expectations

### Why Use Flat List?

- Prefer continuous scrolling without interruptions
- Many notes edited recently (groups get large)
- Use search primarily to find notes
- Prefer alphabetical or manual sorting
- Fewer total notes don't need grouping

## Code Examples

```bash
# Read current value
defaults read com.apple.Notes GroupNotesByDate

# Enable date grouping (default)
defaults write com.apple.Notes GroupNotesByDate -bool true

# Disable date grouping (flat list)
defaults write com.apple.Notes GroupNotesByDate -bool false

# Delete (reset to default)
defaults delete com.apple.Notes GroupNotesByDate
```

## Related Settings

```bash
# Group notes by date (this setting)
defaults write com.apple.Notes GroupNotesByDate -bool true

# Sort notes order (1=Date Edited, 2=Date Created, 3=Title)
defaults write com.apple.Notes NotesSortOrder -int 1

# Folders sort order (1=Manual, 2=Name A-Z, 3=Name Z-A)
defaults write com.apple.Notes FoldersSortOrder -int 1

# Default font size
defaults write com.apple.Notes DefaultFontSize -int 16

# New note body style (0=Title, 1=Heading, 2=Subheading, 3=Body)
defaults write com.apple.Notes NewNoteBodyStyle -int 0
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Setting not applying | Quit and relaunch Notes |
| Dates seem wrong | Check system date/time settings |
| Too many notes in "Today" | Notes uses edit date, not create date |
| Grouping not visible | May be disabled when sorting by Title |
| Groups collapsed | Click group header to expand |
