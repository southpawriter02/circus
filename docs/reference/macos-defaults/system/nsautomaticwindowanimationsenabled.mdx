# Disable Window Opening Animations

Control whether windows animate when opening documents.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `NSAutomaticWindowAnimationsEnabled` |
| **Type** | Boolean |
| **Default Value** | `true` |
| **Our Setting** | `false` |

## Description

Controls whether macOS displays animations when opening documents from Finder. When enabled, windows "zoom out" from the file icon to their full size with a brief animation. Disabling this makes windows appear instantly, creating a snappier feel especially when opening many documents throughout the day.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Animations enabled** (default) â€” Windows zoom open from file icon |
| `false` | **Animations disabled** â€” Windows appear instantly |

## Our Configuration

**Set to**: `false` â€” Disable window animations for instant response.

### Why Disable Animations?

1. **Speed** â€” Windows appear instantly
2. **Productivity** â€” No waiting for animations
3. **Responsiveness** â€” System feels snappier
4. **Focus** â€” Less visual distraction
5. **Efficiency** â€” Smoother workflow

### Why Keep Animations?

- Enjoy the visual polish
- Easier to track window positions
- Prefer macOS default behavior
- Visual continuity from icon to window
- New to Mac (helps understand UI flow)

## Code Examples

```bash
# Read current value
defaults read NSGlobalDomain NSAutomaticWindowAnimationsEnabled

# Disable window animations (recommended)
defaults write NSGlobalDomain NSAutomaticWindowAnimationsEnabled -bool false

# Enable window animations (default)
defaults write NSGlobalDomain NSAutomaticWindowAnimationsEnabled -bool true

# Delete (reset to default)
defaults delete NSGlobalDomain NSAutomaticWindowAnimationsEnabled
```

## Related Settings

```bash
# Disable window animations (this setting)
defaults write NSGlobalDomain NSAutomaticWindowAnimationsEnabled -bool false

# Faster Dock animations
defaults write com.apple.dock autohide-time-modifier -float 0.2

# Faster Launchpad animations
defaults write com.apple.dock springboard-show-duration -float 0.1
defaults write com.apple.dock springboard-hide-duration -float 0.1

# Faster Mission Control animations
defaults write com.apple.dock expose-animation-duration -float 0.1

# Faster window resize (per-app, example: Terminal)
defaults write com.apple.Terminal NSWindowResizeTime -float 0.001

# System-wide reduce motion (less precise)
defaults write com.apple.universalaccess reduceMotion -bool true
```
