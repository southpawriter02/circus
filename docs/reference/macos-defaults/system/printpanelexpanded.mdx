# Print Dialog Expanded State

Automatically expand the print dialog to show all options.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `NSGlobalDomain` |
| **Key** | `PMPrintingExpandedStateForPrint2` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Controls whether the print dialog opens in expanded mode, showing all printing options like paper size, orientation, scaling, and printer-specific settings. By default, macOS shows a simplified print dialog that hides advanced options behind a "Show Details" button. Enabling this setting shows all options immediately.

## Values

| Value | Behavior |
|-------|----------|
| `false` | **Collapsed** (default) â€” Simple dialog, click to expand |
| `true` | **Expanded** â€” All options visible immediately |

## Our Configuration

**Set to**: `true` â€” Show expanded print dialog for full control.

### Why Enable Expanded Dialog?

1. **All options visible** â€” No clicking to expand
2. **Faster workflow** â€” Access settings immediately
3. **Preview** â€” See page preview
4. **Professional** â€” Control paper size, scaling
5. **Fewer reprints** â€” Set options correctly first time

### Why Keep Collapsed (Default)?

- Simple printing needs
- Always use default settings
- Faster for basic prints
- Less overwhelming interface
- Prefer minimal dialogs

## Code Examples

```bash
# Read current value
defaults read NSGlobalDomain PMPrintingExpandedStateForPrint2

# Enable expanded print dialog
defaults write NSGlobalDomain PMPrintingExpandedStateForPrint2 -bool true

# Also set the older key (for compatibility)
defaults write NSGlobalDomain PMPrintingExpandedStateForPrint -bool true

# Disable (collapsed, default)
defaults write NSGlobalDomain PMPrintingExpandedStateForPrint2 -bool false

# Delete (reset to default)
defaults delete NSGlobalDomain PMPrintingExpandedStateForPrint2
```

Changes take effect immediately for new print dialogs.

## UI Location

There is no system setting for this. Each print dialog has:
- **Show Details** button to expand
- **Hide Details** button to collapse

The setting makes "Show Details" the default state.

## Print Presets

Save frequently used settings:
1. Configure all options in expanded dialog
2. Click **Presets** dropdown
3. Select **Save Current Settings as Preset...**
4. Name your preset
5. Available in all applications

## App-Specific Behavior

Some apps have custom print dialogs:
- **Adobe apps** â€” Own print interface
- **Chrome** â€” Custom print preview
- **Office apps** â€” Extended options

These may not respect the system setting.

## Legacy Key

macOS uses two preference keys:

| Key | macOS Version |
|-----|---------------|
| `PMPrintingExpandedStateForPrint` | 10.6-10.8 |
| `PMPrintingExpandedStateForPrint2` | 10.9+ |

Set both for maximum compatibility:
```bash
defaults write NSGlobalDomain PMPrintingExpandedStateForPrint -bool true
defaults write NSGlobalDomain PMPrintingExpandedStateForPrint2 -bool true
```

## Compatibility

- **macOS**: 10.6+ (Snow Leopard)
- **Note**: Takes effect immediately for new print dialogs
- **Note**: Some applications have custom print dialogs
- **Note**: Printer-specific options depend on driver
