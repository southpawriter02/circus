# Spotlight Suggestions

Control whether Spotlight sends searches to Apple.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.lookup.shared` |
| **Key** | `LookupSuggestionsDisabled` |
| **Type** | Boolean |
| **Default Value** | `false` |
| **Our Setting** | `true` |

## Description

Spotlight can send your search queries to Apple to provide suggestions from the internet, including Wikipedia articles, news, iTunes content, and web results. Disabling this keeps all searches local for privacy.

## Values

| Value | Behavior |
|-------|----------|
| `true` | **Suggestions disabled** â€” Local search only |
| `false` | **Suggestions enabled** (default) â€” Queries sent to Apple |

## Our Configuration

**Set to**: `true` â€” Disable internet suggestions for privacy.

### What Gets Sent to Apple

When Spotlight Suggestions are enabled:
- Your search query (as you type)
- Approximate location (for local results)
- Device information
- Language and region

### What You Lose

When disabled, you won't see:
- Wikipedia summaries
- Web search suggestions
- iTunes/App Store results
- News results
- Maps suggestions

### What You Keep

Local Spotlight still searches:
- Applications
- Documents
- Folders
- Emails
- Calendar events
- Contacts
- System Preferences

## Code Examples

```bash
# Read current value
defaults read com.apple.lookup.shared LookupSuggestionsDisabled

# Disable Spotlight suggestions (privacy)
defaults write com.apple.lookup.shared LookupSuggestionsDisabled -bool true

# Enable Spotlight suggestions (Apple default)
defaults write com.apple.lookup.shared LookupSuggestionsDisabled -bool false

# Delete (reset to default)
defaults delete com.apple.lookup.shared LookupSuggestionsDisabled
```

## Spotlight Reindexing

If Spotlight behaves unexpectedly:

```bash
# Reindex entire drive
sudo mdutil -E /

# Check indexing status
mdutil -s /

# Disable/enable indexing
sudo mdutil -i off /
sudo mdutil -i on /
```

| Issue | Solution |
|-------|----------|
| Suggestions still appearing | Log out and back in |
| Search results slow | Run `sudo mdutil -E /` to reindex |
