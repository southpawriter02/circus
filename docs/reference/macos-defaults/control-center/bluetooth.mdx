# Bluetooth (Control Center)

Show Bluetooth icon in the menu bar.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.controlcenter` |
| **Key** | `Bluetooth` |
| **Type** | Integer |
| **Default Value** | `18` |
| **Our Setting** | `2` |

## Description

Controls whether the Bluetooth icon appears in the menu bar. The icon provides quick access to Bluetooth devices and settings.

## Values

| Value | Behavior |
|-------|----------|
| `2` | **Always show** — Icon always in menu bar |
| `8` | **Show when active** — Only when Bluetooth is on |
| `18` | **Control Center only** (default) — In Control Center, not menu bar |
| `24` | **Don't show** — Hidden completely |

## Our Configuration

**Set to**: `2` — Always show in menu bar for quick access.

## Code Examples

```bash
# Read current value
defaults read com.apple.controlcenter Bluetooth

# Always show Bluetooth in menu bar
defaults write com.apple.controlcenter Bluetooth -int 2

# Show only in Control Center (Apple default)
defaults write com.apple.controlcenter Bluetooth -int 18

# Delete (reset to default)
defaults delete com.apple.controlcenter Bluetooth

# Restart Control Center to apply
killall ControlCenter
```

## UI Location

**System Settings** → **Control Center** → **Bluetooth** → **Show in Menu Bar**

## Why Always Show?

- Quick device switching (headphones, etc.)
- See Bluetooth status at a glance
- Fast access to turn Bluetooth on/off
- Easy device connection management

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Icon not appearing | Restart ControlCenter |
| Value unclear | Use `2` for always, `18` for Control Center only |
| Changes not sticking | Log out and back in |

## Compatibility

- **Control Center**: macOS 11.0+
- **Applies to**: Menu bar icon
