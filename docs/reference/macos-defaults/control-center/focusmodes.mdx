# Control Center Focus Modes

Configure Focus/Do Not Disturb visibility in Control Center.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.controlcenter` |
| **Key** | `FocusModes` |
| **Type** | Integer |
| **Default Value** | `18` |
| **Our Setting** | `18` |

## Description

Controls where Focus (formerly Do Not Disturb) appearsâ€”in the menu bar, Control Center, or hidden. Focus modes silence notifications and communications based on your activity (Work, Personal, Sleep, etc.).

## Values

| Value | Visibility |
|-------|------------|
| `0` | Hidden/Disabled |
| `2` | Always in menu bar |
| `8` | Show in menu bar when active |
| `18` | **Control Center only** (default) |
| `24` | Don't show anywhere |

## Our Configuration

**Set to**: `18` â€” Control Center only.

### Why Control Center Only?

1. **Clean menu bar** â€” One less icon
2. **Not constant** â€” Focus changes are intentional
3. **Easy access** â€” Still quick via Control Center
4. **Less distraction** â€” Ironic to have Focus icon distracting

### When to Show in Menu Bar

Set to `2` or `8` if you:
- Toggle Focus frequently
- Need to see Focus status at a glance
- Share your screen and want others to see Focus

## Code Examples

```bash
# Read current value
defaults read com.apple.controlcenter FocusModes

# Show only in Control Center
defaults write com.apple.controlcenter FocusModes -int 18

# Always show in menu bar
defaults write com.apple.controlcenter FocusModes -int 2

# Show when Focus is active
defaults write com.apple.controlcenter FocusModes -int 8

# Apply changes
killall ControlCenter
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Focus icon not showing | Set value to 2 or 8 |
| Focus not silencing | Check Focus settings in System Settings |
| Calls still coming through | Check "Allow Calls From" setting |

## Focus vs Do Not Disturb

- **Do Not Disturb** (pre-Monterey): Simple on/off
- **Focus** (Monterey+): Multiple modes with customization

Focus replaced DND with more granular control.

## Compatibility

- **macOS**: 12.0 (Monterey)+ for Focus
- **Earlier macOS**: Do Not Disturb in notification settings
- **Syncs**: Focus syncs across Apple devices
