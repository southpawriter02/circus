# Control Center AirDrop Visibility

Configure where AirDrop controls appear.

## Overview

| Property | Value |
|----------|-------|
| **Domain** | `com.apple.controlcenter` |
| **Key** | `AirDrop` |
| **Type** | Integer |
| **Default Value** | `18` |
| **Our Setting** | `18` |

## Description

Controls where the AirDrop toggle appears â€” in the menu bar, Control Center only, or hidden entirely. AirDrop allows wireless file sharing between Apple devices and needs to be accessible for quick configuration of receiving settings.

## Visibility Values

| Value | Location | Icon Visibility |
|-------|----------|-----------------|
| `0` | Disabled | Not shown anywhere |
| `2` | Menu bar | Always visible in menu bar |
| `8` | Menu bar (when active) | Shows only during transfers |
| `18` | Control Center only | Default, access via Control Center |
| `24` | Hidden | Not shown (still functional) |

## Our Configuration

**Set to**: `18` â€” Control Center only.

### Why Control Center Only?

1. **Clean menu bar** â€” Less clutter
2. **Infrequent use** â€” Not needed constantly
3. **Easy access** â€” One click to Control Center
4. **Security** â€” Less obvious to others
5. **Default** â€” Apple's recommendation

### When to Show in Menu Bar

- Frequently share files via AirDrop
- Quick access to receiving settings
- Team environments with regular sharing

## Code Examples

```bash
# Read current value
defaults read com.apple.controlcenter AirDrop

# Control Center only (default, recommended)
defaults write com.apple.controlcenter AirDrop -int 18

# Always show in menu bar
defaults write com.apple.controlcenter AirDrop -int 2

# Show only when active
defaults write com.apple.controlcenter AirDrop -int 8

# Hide completely
defaults write com.apple.controlcenter AirDrop -int 24

# Delete (reset to default)
defaults delete com.apple.controlcenter AirDrop
```

## AirDrop Receiving Settings

Regardless of visibility, AirDrop has three modes:

| Mode | Receives From |
|------|---------------|
| Receiving Off | Nobody |
| Contacts Only | People in your Contacts |
| Everyone | Any nearby Apple device |

âš ï¸ **Security**: Use "Contacts Only" or "Off" in public places.

## AirDrop Requirements

For AirDrop to work:
- Wi-Fi enabled (doesn't need network)
- Bluetooth enabled
- Devices within ~30 feet
- Both devices unlocked

## AirDrop Troubleshooting

| Issue | Solution |
|-------|----------|
| Not seeing devices | Check Bluetooth and Wi-Fi on |
| Can't receive | Check receiving mode |
| Firewall blocking | Allow incoming in Firewall options |
| Old device | Check device compatibility |
